Para contar bem o que aconteceu, você precisa dizer **quando**. Esta lição te dá os marcadores que se combinam com o passado recente para ancorar as ações no tempo.

## Os marcadores básicos

| Euskera | Português |
|---|---|
| *gaur* | hoje |
| *atzo* | ontem |
| *bihar* | amanhã |
| *etzi* | depois de amanhã |
| *orain* | agora |
| *oraintxe* | agora mesmo |
| *lehen* | antes |
| *gero* | logo, depois |

> Se você reparou, *atzo* (ontem) e *bihar* (amanhã) são duas palavras tão curtas que se aprendem de uma vez. *Etzi* (depois de amanhã) é uma palavra que o português não tem numa única peça — um pequeno presente do euskera.

## Combinando com momentos do dia

Os marcadores se combinam com *goizean*, *arratsaldean*, *gauean* e companhia da unidade anterior. A fórmula é: **marcador + parte do dia**.

| Euskera | Português |
|---|---|
| *gaur goizean* | esta manhã / hoje de manhã |
| *gaur eguerdian* | hoje ao meio-dia |
| *gaur arratsaldean* | hoje à tarde |
| *gaur gauean* | hoje à noite |
| *atzo goizean* | ontem de manhã |
| *atzo arratsaldean* | ontem à tarde |
| *atzo gauean* | ontem à noite |
| *bihar goizean* | amanhã de manhã |

> **Atenção:** *gaur gauean* é "hoje à noite" (a noite que você tem pela frente ou a que está vivendo); *atzo gauean* é "ontem à noite". O euskera distingue muito claramente.

## Frequência e vezes

| Euskera | Português |
|---|---|
| *behin* | uma vez |
| *bi aldiz* | duas vezes |
| *hiru aldiz* | três vezes |
| *askotan* | com frequência, muitas vezes |
| *batzuetan* | às vezes |
| *inoiz ez* | nunca |
| *gaur goizetik* | desde esta manhã |
| *aste honetan* | esta semana |
| *hilabete honetan* | este mês |

## Frases com passado recente e marcadores

> *Gaur goizean lanera joan naiz autoz.* — Esta manhã fui ao trabalho de carro.
>
> *Gaur arratsaldean Mikelekin egon naiz.* — Hoje à tarde estive com Mikel.
>
> *Gaur oso berandu esnatu naiz.* — Hoje acordei muito tarde.
>
> *Gaur eguerdian etxean bazkaldu dut.* — Hoje ao meio-dia almocei em casa.
>
> *Aste honetan bi aldiz egon naiz zinean.* — Esta semana estive duas vezes no cinema.

## A ordem da frase

Em euskera a ordem típica de uma frase com marcador temporal é:

> **QUANDO + ONDE + O QUÊ + VERBO**

> **Gaur goizean** + **euskaltegian** + **euskara** + **ikasi dut**. — Esta manhã estudei euskera no euskaltegi.

Não é a única possibilidade — o euskera permite mover os elementos para enfatizar — mas se você seguir esta ordem não erra.

## Mini-conversa

> *— Eta gaur zer egin duzu, Maialen?*
>
> *— **Gaur goizean** lanera joan naiz oinez. **Eguerdian** lagunekin bazkaldu dut tabernan. **Arratsaldean** etxera bueltatu naiz eta liburua irakurri dut. **Gauean** ez dut afaldu, asko jan baitut bazkarian.*

> **Nota:** *gaur* abrange o dia atual inteiro. Se você quiser precisar, acrescente o momento (*gaur goizean*, *gaur arratsaldean*). Em conversas comuns, *Zer egin duzu gaur?* se entende como "o que você fez hoje em geral?".
