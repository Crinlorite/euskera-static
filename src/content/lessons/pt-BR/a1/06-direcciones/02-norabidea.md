Quando alguém te pergunta como chegar a um lugar — ou quando você se perde e pergunta — você precisa de um punhado de verbos no imperativo e de advérbios de direção. Nesta lição reunimos o essencial.

## Imperativos básicos

Em euskera, o imperativo afirmativo dos verbos de movimento é bem direto:

| Forma | Verbo base | Sentido |
|---|---|---|
| **Jo** | jo | bata / dirija (jo ezkerrera = vire à esquerda) |
| **Segi** / **Jarraitu** | segitu / jarraitu | siga, continue |
| **Hartu** | hartu | pegue (hartu kale hau = pegue esta rua) |
| **Igo** | igo | suba |
| **Jaitsi** | jaitsi | desça |
| **Sartu** | sartu | entre |
| **Irten** | irten | saia |
| **Zeharkatu** | zeharkatu | atravesse |
| **Iritsi** / **heldu** | iritsi / heldu | chegue (… arte = "até…") |

> *Jo* literalmente significa "bater", mas é usado metaforicamente como "dirigir-se a": **jo ezkerrera** = "siga para a esquerda".

## Advérbios e referências direcionais

| Euskera | Português |
|---|---|
| *ezkerra* | a esquerda |
| *eskuina* | a direita |
| *aurrera* | em frente |
| *atzera* | para trás |
| *zuzen* | reto, em linha reta |
| *gora* | para cima |
| *behera* | para baixo |

> Repare: para indicar **direção em direção a** usa-se o sufixo `-ra` (que você verá a fundo na lição 03): *ezkerre-ra* (à esquerda), *aurre-ra* (em frente), *atze-ra* (para trás). O mesmo para *gora*, *behera*.

## Lugares de referência típicos

| Euskera | Português |
|---|---|
| *semaforoa* | semáforo |
| *bidegurutzea* | cruzamento, interseção |
| *zebrabidea* | faixa de pedestres |
| *biribilgunea* / *errotonda* | rotatória |
| *aldapa* | ladeira |
| *eskailerak* | escadas |
| *zubia* | ponte |
| *errepidea* | estrada |
| *kalea* | rua |

## Frases-tipo

- **Jo ezkerrera bidegurutzean.** — Vire à esquerda no cruzamento.
- **Segi aurrera semaforora arte.** — Siga reto até o semáforo.
- **Zeharkatu zebrabidea.** — Atravesse a faixa de pedestres.
- **Igo eskailerak eta sartu eraikinean.** — Suba as escadas e entre no prédio.
- **Jaitsi aldapa, eta gero jo eskuinera.** — Desça a ladeira e depois vire à direita.
- **Egin atzera; gehiegi joan zara.** — Volte atrás; você passou do ponto.

## Pedir e dar informações

> — *Barkatu, **non dago** udaletxea?* — Com licença, onde fica a prefeitura?
>
> — ***Segi aurrera** semaforora arte. Han **jo eskuinera** eta **zeharkatu** plaza. Udaletxea elizaren ondoan dago.* — Siga reto até o semáforo. Ali vire à direita e atravesse a praça. A prefeitura fica ao lado da igreja.
>
> — *Eta gertu **dago**?* — E fica perto?
>
> — *Bai, oso gertu. Bost minutu oinez.* — Sim, bem perto. Cinco minutos a pé.

> **Dica:** muitas indicações seguem um padrão fixo: imperativo (*jo*, *segi*, *zeharkatu*) + lugar com sufixo (*ezkerrera*, *semaforora*, *zebrabidea*). Se você memorizar o padrão, pode encadear instruções longas combinando peças.
