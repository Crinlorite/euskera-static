Esta lição integra tudo o que vimos no tema 9: os cômodos, os móveis, as tarefas e com quem você mora. O objetivo é que você consiga **descrever a sua casa** em frases simples, como se estivessem perguntando para você na aula ou em uma conversa informal.

## As perguntas básicas

| Pergunta | Português |
|---|---|
| *Non dago zure etxea?* | Onde fica a sua casa? |
| *Nolakoa da?* | Como é? |
| *Zenbat gela ditu?* | Quantos cômodos tem? |
| *Norekin bizi zara?* | Com quem você mora? |
| *Zenbatgarren solairuan bizi zara?* | Em que andar você mora? |
| *Etxeak badu igogailua / balkoia / terraza?* | A casa tem elevador / sacada / terraço? |

## Adjetivos úteis para descrever a casa

| Euskera | Português |
|---|---|
| *handia* | grande |
| *txikia* | pequena |
| *argitsua* | luminosa |
| *iluna* | escura |
| *berria* | nova |
| *zaharra* | velha |
| *lasaia* | tranquila |
| *zaratatsua* | barulhenta |
| *erosoa* | confortável |
| *garestia* | cara |
| *merkea* | barata |

> Lembre-se do padrão **NOLAKOA da?** já visto em outras unidades: a resposta leva o adjetivo + `-a` final + verbo *izan*. Ex.: *Nire etxea **handia da**.* — Minha casa é grande.

## Um parágrafo modelo

Aqui está um texto simples descrevendo uma casa. Leia-o em voz alta e preste atenção nos padrões.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## Sua vez: descreva a sua casa

Monte o seu próprio parágrafo respondendo nesta ordem:

1. **Non bizi zara?** (Onde você mora?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (Como é?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (Quantos cômodos?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (O que tem?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

## Vocabulário de encerramento

| Euskera | Português |
|---|---|
| *etxebizitza* | moradia |
| *etxejabea* | proprietário |
| *alokairua* | aluguel |
| *errenta* | renda / aluguel mensal |
| *auzoa* | bairro |
| *erdigunea* | centro (da cidade) |
| *aldiriak* | periferia / arredores |

> **Dica:** quando for falar da sua casa na aula, não tente traduzir cada detalhe. Fique com frases curtas e bem feitas. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* é uma descrição melhor do que uma frase longa cheia de erros.
