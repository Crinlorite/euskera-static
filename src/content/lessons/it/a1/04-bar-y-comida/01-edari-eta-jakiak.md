Buona parte della vita sociale in Euskal Herria passa per la **taberna** (il bar). Conoscere le bevande e i cibi di base in euskera ti aprirà porte immediate: una compagnia di amici, una visita a un paese nuovo, un *poteo*. Cominciamo dal vocabolario.

## Il caffè e le sue varianti

Il caffè è probabilmente ciò che ordinerai di più. Ha diverse forme:

| Euskera | Italiano |
|---|---|
| *kafea* | il caffè (generico) |
| *kafe hutsa* | caffè nero (letterale: "caffè puro/vuoto") |
| *kafesnea* | caffellatte (da *kafe* + *esne*) |
| *kafe ebakia* | caffè macchiato (*ebaki* = "tagliato") |
| *kafe deskafeinatua* | caffè decaffeinato |
| *infusioa* | infuso |
| *tea* | tè |

> *Kafesne* letteralmente significa "caffè-latte". L'euskera forma molte parole composte così, senza trattino e con la parola principale alla fine.

## Altre bevande

| Euskera | Italiano |
|---|---|
| *ura* | l'acqua |
| *zumoa* | succo |
| *esnea* | il latte |
| *ardoa* | vino |
| *ardo gorria* | vino rosso |
| *ardo zuria* | vino bianco |
| *garagardoa* | birra |
| *zuritoa* | caña (birra piccola, tipica) |
| *sagardoa* | sidro |
| *txakolina* | txakoli (vino bianco basco) |
| *patxarana* | pacharán |

> Tre parole curiose: **sagardo** (sidro) = *sagar* (mela) + *ardo* (vino); **garagardo** (birra) = *garagar* (orzo) + *ardo*; **txakolina** è un vino bianco giovane e leggero, marchio di fabbrica nei Paesi Baschi.

## Cibi di base al bar

Per accompagnare ciò che bevi:

| Euskera | Italiano |
|---|---|
| *ogia* | il pane |
| *gazta* | formaggio |
| *urdaiazpikoa* | prosciutto |
| *tortilla* | frittata (di patate) |
| *patata frijituak* | patatine fritte |
| *gozokia* | dolce (generico) |
| *txokolatea* | cioccolato |

## L'articolo `-a`

Quasi tutte queste parole portano **`-a`** alla fine quando compaiono "neutre" in una frase: è l'articolo "il / la".

- *Ardoa gustatzen zait.* — Mi piace il vino.
- *Ura nahi dut.* — Voglio dell'acqua.

Quando chiedi "uno", la `-a` sparisce ed entra **bat** ("uno/una"):

- *Kafe **bat**, mesedez.* — Un caffè, per favore.
- *Garagardo **bat**.* — Una birra.

> Nella lezione 3 vedrai questo in dettaglio. Per ora memorizza le forme con `-a` per riconoscerle nel menù del bar.
