Questa lezione integra tutto ciò che è stato trattato nel tema 9: le stanze, i mobili, le faccende e con chi vivi. L'obiettivo è che tu sappia **descrivere la tua casa** in frasi semplici, come se ti stessero facendo domande in classe o in una conversazione informale.

## Le domande di base

| Domanda | Italiano |
|---|---|
| *Non dago zure etxea?* | Dov'è la tua casa? |
| *Nolakoa da?* | Com'è? |
| *Zenbat gela ditu?* | Quante stanze ha? |
| *Norekin bizi zara?* | Con chi vivi? |
| *Zenbatgarren solairuan bizi zara?* | A che piano vivi? |
| *Etxeak badu igogailua / balkoia / terraza?* | La casa ha ascensore / balcone / terrazza? |

## Aggettivi utili per descrivere la casa

| Euskera | Italiano |
|---|---|
| *handia* | grande |
| *txikia* | piccola |
| *argitsua* | luminosa |
| *iluna* | buia |
| *berria* | nuova |
| *zaharra* | vecchia |
| *lasaia* | tranquilla |
| *zaratatsua* | rumorosa |
| *erosoa* | comoda |
| *garestia* | cara |
| *merkea* | economica |

> Ricorda lo schema **NOLAKOA da?** già visto in altre unità: la risposta porta l'aggettivo + `-a` finale + verbo *izan*. Es.: *Nire etxea **handia da**.* — La mia casa è grande.

## Un paragrafo modello

Ecco un testo semplice che descrive una casa. Leggilo ad alta voce e fai attenzione agli schemi.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## Tocca a te: descrivi la tua casa

Costruisci il tuo paragrafo rispondendo in quest'ordine:

1. **Non bizi zara?** (Dove vivi?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (Com'è?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (Quante stanze?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (Cosa c'è?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

## Vocabolario di chiusura

| Euskera | Italiano |
|---|---|
| *etxebizitza* | abitazione |
| *etxejabea* | proprietario |
| *alokairua* | affitto |
| *errenta* | canone / affitto mensile |
| *auzoa* | quartiere |
| *erdigunea* | centro (della città) |
| *aldiriak* | periferia / dintorni |

> **Consiglio:** quando parli della tua casa in classe, non cercare di tradurre ogni dettaglio. Limitati a frasi brevi e ben fatte. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* è una descrizione migliore di una frase lunga piena di errori.
