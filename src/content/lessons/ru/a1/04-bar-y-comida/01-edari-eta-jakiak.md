Значительная часть социальной жизни в Euskal Herria проходит через **taberna** (бар). Знание основных напитков и блюд на эускера сразу откроет вам двери: компания друзей, поездка в новую деревню, *poteo*. Начнём со словарного запаса.

## Кофе и его разновидности

Кофе — это, вероятно, то, что вы будете заказывать чаще всего. У него есть несколько форм:

| Эускера | Русский |
|---|---|
| *kafea* | кофе (общее) |
| *kafe hutsa* | чёрный кофе (букв.: "чистый/пустой кофе") |
| *kafesnea* | кофе с молоком (от *kafe* + *esne*) |
| *kafe ebakia* | кофе с каплей молока (*ebaki* = "разрезанный") |
| *kafe deskafeinatua* | кофе без кофеина |
| *infusioa* | травяной настой |
| *tea* | чай |

> *Kafesne* буквально означает "кофе-молоко". В эускера многие сложные слова образуются так, без дефиса и с главным словом в конце.

## Другие напитки

| Эускера | Русский |
|---|---|
| *ura* | вода |
| *zumoa* | сок |
| *esnea* | молоко |
| *ardoa* | вино |
| *ardo gorria* | красное вино |
| *ardo zuria* | белое вино |
| *garagardoa* | пиво |
| *zuritoa* | кружечка (маленькое пиво, типичное) |
| *sagardoa* | сидр |
| *txakolina* | чаколи (баскское белое вино) |
| *patxarana* | пачаран |

> Три любопытных слова: **sagardo** (сидр) = *sagar* (яблоко) + *ardo* (вино); **garagardo** (пиво) = *garagar* (ячмень) + *ardo*; **txakolina** — молодое лёгкое белое вино, фирменный знак Страны Басков.

## Основные блюда в баре

Чтобы закусить то, что вы пьёте:

| Эускера | Русский |
|---|---|
| *ogia* | хлеб |
| *gazta* | сыр |
| *urdaiazpikoa* | хамон |
| *tortilla* | тортилья (картофельная) |
| *patata frijituak* | картофель фри |
| *gozokia* | сладость (общее) |
| *txokolatea* | шоколад |

## Артикль `-a`

Почти все эти слова получают **`-a`** в конце, когда появляются "нейтрально" в предложении: это артикль "el / la".

- *Ardoa gustatzen zait.* — Мне нравится вино.
- *Ura nahi dut.* — Я хочу воды.

Когда вы заказываете "один", `-a` исчезает и появляется **bat** ("один/одна"):

- *Kafe **bat**, mesedez.* — Один кофе, пожалуйста.
- *Garagardo **bat**.* — Одно пиво.

> В уроке 3 вы рассмотрите это подробно. Пока запомните формы с `-a`, чтобы узнавать их в меню бара.
