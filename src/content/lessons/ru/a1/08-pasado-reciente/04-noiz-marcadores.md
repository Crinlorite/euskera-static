Чтобы хорошо рассказать о том, что произошло, нужно сказать **когда**. Этот урок даёт тебе маркеры, которые сочетаются с недавним прошедшим, чтобы закрепить действия во времени.

## Базовые маркеры

| Эускера | Русский |
|---|---|
| *gaur* | сегодня |
| *atzo* | вчера |
| *bihar* | завтра |
| *etzi* | послезавтра |
| *orain* | сейчас |
| *oraintxe* | прямо сейчас |
| *lehen* | раньше |
| *gero* | потом, после |

> Если ты заметил, *atzo* (вчера) и *bihar* (завтра) — это два настолько коротких слова, что запоминаются сразу. *Etzi* (послезавтра) — это слово, которого в русском нет в одном куске... но в эускере оно есть — маленький подарок.

## Сочетание с моментами дня

Маркеры сочетаются с *goizean*, *arratsaldean*, *gauean* и компанией из предыдущего раздела. Формула такая: **маркер + часть дня**.

| Эускера | Русский |
|---|---|
| *gaur goizean* | сегодня утром / этим утром |
| *gaur eguerdian* | сегодня в полдень |
| *gaur arratsaldean* | сегодня во второй половине дня |
| *gaur gauean* | сегодня ночью |
| *atzo goizean* | вчера утром |
| *atzo arratsaldean* | вчера во второй половине дня |
| *atzo gauean* | вчера ночью |
| *bihar goizean* | завтра утром |

> **Внимание:** *gaur gauean* — это "сегодня ночью" (ночь, которая впереди, или та, которую ты проживаешь); *atzo gauean* — это "вчера ночью / минувшей ночью". Эускера различает это очень чётко.

## Частота и количество раз

| Эускера | Русский |
|---|---|
| *behin* | один раз |
| *bi aldiz* | два раза |
| *hiru aldiz* | три раза |
| *askotan* | часто, много раз |
| *batzuetan* | иногда |
| *inoiz ez* | никогда |
| *gaur goizetik* | с сегодняшнего утра |
| *aste honetan* | на этой неделе |
| *hilabete honetan* | в этом месяце |

## Фразы с недавним прошедшим и маркерами

> *Gaur goizean lanera joan naiz autoz.* — Сегодня утром я поехал на работу на машине.
>
> *Gaur arratsaldean Mikelekin egon naiz.* — Сегодня во второй половине дня я был с Mikel.
>
> *Gaur oso berandu esnatu naiz.* — Сегодня я проснулся очень поздно.
>
> *Gaur eguerdian etxean bazkaldu dut.* — Сегодня в полдень я пообедал дома.
>
> *Aste honetan bi aldiz egon naiz zinean.* — На этой неделе я был в кино два раза.

## Порядок слов в предложении

В эускере типичный порядок предложения с временным маркером такой:

> **КОГДА + ГДЕ + ЧТО + ГЛАГОЛ**

> **Gaur goizean** + **euskaltegian** + **euskara** + **ikasi dut**. — Сегодня утром я учил эускеру в эускальтеги.

Это не единственный вариант — эускера позволяет переставлять элементы для акцента — но если ты следуешь этому порядку, не ошибёшься.

## Мини-диалог

> *— Eta gaur zer egin duzu, Maialen?*
>
> *— **Gaur goizean** lanera joan naiz oinez. **Eguerdian** lagunekin bazkaldu dut tabernan. **Arratsaldean** etxera bueltatu naiz eta liburua irakurri dut. **Gauean** ez dut afaldu, asko jan baitut bazkarian.*

> **Примечание:** *gaur* охватывает весь текущий день. Если хочешь уточнить, добавь момент (*gaur goizean*, *gaur arratsaldean*). В обычных разговорах *Zer egin duzu gaur?* понимается как "что ты сегодня делал в целом?".
