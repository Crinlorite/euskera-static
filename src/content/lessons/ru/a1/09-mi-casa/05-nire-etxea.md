Этот урок объединяет всё из темы 9: комнаты, мебель, домашние дела и то, с кем ты живёшь. Цель — чтобы ты умел **описать свой дом** простыми фразами, как если бы тебя спрашивали на занятии или в неформальном разговоре.

## Основные вопросы

| Вопрос | Русский |
|---|---|
| *Non dago zure etxea?* | Где находится твой дом? |
| *Nolakoa da?* | Какой он? |
| *Zenbat gela ditu?* | Сколько в нём комнат? |
| *Norekin bizi zara?* | С кем ты живёшь? |
| *Zenbatgarren solairuan bizi zara?* | На каком этаже ты живёшь? |
| *Etxeak badu igogailua / balkoia / terraza?* | В доме есть лифт / балкон / терраса? |

## Полезные прилагательные для описания дома

| Euskera | Русский |
|---|---|
| *handia* | большой |
| *txikia* | маленький |
| *argitsua* | светлый |
| *iluna* | тёмный |
| *berria* | новый |
| *zaharra* | старый |
| *lasaia* | спокойный |
| *zaratatsua* | шумный |
| *erosoa* | удобный |
| *garestia* | дорогой |
| *merkea* | дешёвый |

> Помни модель **NOLAKOA da?**, уже встречавшуюся в других разделах: в ответе идёт прилагательное + конечная `-a` + глагол *izan*. Например: *Nire etxea **handia da**.* — Мой дом большой.

## Образцовый абзац

Вот простой текст с описанием дома. Прочитай его вслух и обрати внимание на модели.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## Твоя очередь: опиши свой дом

Построй собственный абзац, отвечая в таком порядке:

1. **Non bizi zara?** (Где ты живёшь?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (Какой дом?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (Сколько комнат?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (Что там есть?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

> **Совет:** когда говоришь о своём доме на занятии, не пытайся перевести каждую деталь. Ограничься короткими и правильно построенными фразами. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* — это лучшее описание, чем длинная фраза, полная ошибок.
