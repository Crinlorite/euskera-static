Есть два одинаково правильных способа спросить имя:

- **Zein da zure izena?** — «Как твоё имя?»
- **Zure izena zein da?** — порядок слов может меняться; оба варианта естественны.

## Структура

- **Zein** = который / какой (в вопросах с выбором)
- **izena** = имя (конечная `-a` — это артикль «определённость»)
- **zure** = твой
- **da** = есть (является)

## Как ответить

У тебя есть два варианта, одинаково правильных:

1. **Nire izena Egoitz da.** — «Меня зовут Egoitz».
2. **Egoitz naiz.** — «Я Egoitz». (модель из предыдущего урока)

> Базовые притяжательные местоимения — *nire* (мой), *zure* (твой), *bere* (его/её). Ты будешь встречать их ещё много раз дальше — начинай знакомиться с ними уже сейчас.
