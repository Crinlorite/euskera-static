Unha boa parte da vida social en Euskal Herria pasa pola **taberna** (o bar). Coñecer as bebidas e comidas básicas en euskera abrirache portas inmediatas: unha cuadrilla de amigos, unha visita a un pobo novo, un *poteo*. Comezamos polo vocabulario.

## O café e as súas variantes

O café é probablemente o que máis vas pedir. Ten varias formas:

| Euskera | Galego |
|---|---|
| *kafea* | o café (xenérico) |
| *kafe hutsa* | café só (literal: "café puro/baleiro") |
| *kafesnea* | café con leite (de *kafe* + *esne*) |
| *kafe ebakia* | café cortado (*ebaki* = "cortado") |
| *kafe deskafeinatua* | café descafeinado |
| *infusioa* | infusión |
| *tea* | té |

> *Kafesne* literalmente significa "café-leite". O euskera forma moitas palabras compostas así, sen guión e coa palabra principal ao final.

## Outras bebidas

| Euskera | Galego |
|---|---|
| *ura* | a auga |
| *zumoa* | zume |
| *esnea* | o leite |
| *ardoa* | viño |
| *ardo gorria* | viño tinto |
| *ardo zuria* | viño branco |
| *garagardoa* | cervexa |
| *zuritoa* | cunca (cervexa pequena, típica) |
| *sagardoa* | sidra |
| *txakolina* | txakoli (viño branco vasco) |
| *patxarana* | pacharán |

> Tres palabras curiosas: **sagardo** (sidra) = *sagar* (mazá) + *ardo* (viño); **garagardo** (cervexa) = *garagar* (cebada) + *ardo*; **txakolina** é un viño branco novo e lixeiro, marca da casa en Euskadi.

## Comidas básicas no bar

Para acompañar o que bebes:

| Euskera | Galego |
|---|---|
| *ogia* | o pan |
| *gazta* | queixo |
| *urdaiazpikoa* | xamón |
| *tortilla* | tortilla (de pataca) |
| *patata frijituak* | patacas fritas |
| *gozokia* | doce (xenérico) |
| *txokolatea* | chocolate |

## O artigo `-a`

Case todas estas palabras levan **`-a`** ao final cando aparecen "neutras" nunha frase: é o artigo "o / a".

- *Ardoa gustatzen zait.* — Gústame o viño.
- *Ura nahi dut.* — Quero auga.

Cando pides "un", desaparece o `-a` e entra **bat** ("un/unha"):

- *Kafe **bat**, mesedez.* — Un café, por favor.
- *Garagardo **bat**.* — Unha cervexa.

> Na lección 3 verás isto en detalle. Polo de agora memoriza as formas con `-a` para recoñecelas na carta do bar.
