Para contar ben o que pasou, precisas dicir **cando**. Esta lección dáche os marcadores que se combinan co pasado recente para ancorar as accións no tempo.

## Os marcadores básicos

| Euskera | Galego |
|---|---|
| *gaur* | hoxe |
| *atzo* | onte |
| *bihar* | mañá |
| *etzi* | pasadomañá |
| *orain* | agora |
| *oraintxe* | agora mesmo |
| *lehen* | antes |
| *gero* | logo, despois |

> Se te fixaches, *atzo* (onte) e *bihar* (mañá) son dúas palabras tan curtas que se aprenden dun golpe. *Etzi* (pasadomañá) é unha palabra que o galego non ten nunha soa peza — un pequeno agasallo do euskera.

## Combinando con momentos do día

Os marcadores combínanse con *goizean*, *arratsaldean*, *gauean* e compaña da unidade anterior. A fórmula é: **marcador + parte do día**.

| Euskera | Galego |
|---|---|
| *gaur goizean* | esta mañá / hoxe pola mañá |
| *gaur eguerdian* | hoxe ao mediodía |
| *gaur arratsaldean* | esta tarde |
| *gaur gauean* | esta noite |
| *atzo goizean* | onte pola mañá |
| *atzo arratsaldean* | onte pola tarde |
| *atzo gauean* | onte á noite |
| *bihar goizean* | mañá pola mañá |

> **Atención:** *gaur gauean* é "esta noite" (a noite que tes por diante ou a que estás a vivir); *atzo gauean* é "onte á noite". O euskera distingue moi claramente.

## Frecuencia e veces

| Euskera | Galego |
|---|---|
| *behin* | unha vez |
| *bi aldiz* | dúas veces |
| *hiru aldiz* | tres veces |
| *askotan* | a miúdo, moitas veces |
| *batzuetan* | ás veces |
| *inoiz ez* | nunca |
| *gaur goizetik* | desde esta mañá |
| *aste honetan* | esta semana |
| *hilabete honetan* | este mes |

## Frases con pasado recente e marcadores

> *Gaur goizean lanera joan naiz autoz.* — Esta mañá fun ao traballo en coche.
>
> *Gaur arratsaldean Mikelekin egon naiz.* — Esta tarde estiven con Mikel.
>
> *Gaur oso berandu esnatu naiz.* — Hoxe espertei moi tarde.
>
> *Gaur eguerdian etxean bazkaldu dut.* — Hoxe ao mediodía comín na casa.
>
> *Aste honetan bi aldiz egon naiz zinean.* — Esta semana estiven dúas veces no cine.

## A orde da frase

En euskera a orde típica dunha frase con marcador temporal é:

> **CANDO + ONDE + QUE + VERBO**

> **Gaur goizean** + **euskaltegian** + **euskara** + **ikasi dut**. — Esta mañá estudei euskera no euskaltegi.

Non é a única posibilidade — o euskera permite mover os elementos para enfatizar — pero se segues esta orde non te equivocas.

## Mini-conversación

> *— Eta gaur zer egin duzu, Maialen?*
>
> *— **Gaur goizean** lanera joan naiz oinez. **Eguerdian** lagunekin bazkaldu dut tabernan. **Arratsaldean** etxera bueltatu naiz eta liburua irakurri dut. **Gauean** ez dut afaldu, asko jan baitut bazkarian.*

> **Nota:** *gaur* abrangue todo o día actual. Se queres precisar, engade o momento (*gaur goizean*, *gaur arratsaldean*). En conversas correntes, *Zer egin duzu gaur?* enténdese como "que fixeches hoxe en xeral?".
