Esta lección integra todo o do tema 9: as habitacións, os mobles, as tarefas e con quen vives. O obxectivo é que poidas **describir a túa casa** en frases sinxelas, coma se cho estivesen preguntando na clase ou nunha conversa informal.

## As preguntas básicas

| Pregunta | Galego |
|---|---|
| *Non dago zure etxea?* | Onde está a túa casa? |
| *Nolakoa da?* | Como é? |
| *Zenbat gela ditu?* | Cantas habitacións ten? |
| *Norekin bizi zara?* | Con quen vives? |
| *Zenbatgarren solairuan bizi zara?* | En que andar vives? |
| *Etxeak badu igogailua / balkoia / terraza?* | A casa ten ascensor / balcón / terraza? |

## Adxectivos útiles para describir a casa

| Euskera | Galego |
|---|---|
| *handia* | grande |
| *txikia* | pequena |
| *argitsua* | luminosa |
| *iluna* | escura |
| *berria* | nova |
| *zaharra* | vella |
| *lasaia* | tranquila |
| *zaratatsua* | ruidosa |
| *erosoa* | cómoda |
| *garestia* | cara |
| *merkea* | barata |

> Lembra o patrón **NOLAKOA da?** xa visto noutras unidades: a resposta leva o adxectivo + `-a` final + verbo *izan*. Ex.: *Nire etxea **handia da**.* — A miña casa é grande.

## Un parágrafo modelo

Aquí tes un texto sinxelo que describe unha casa. Leo en voz alta e fíxate nos patróns.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## A túa quenda: describe a túa casa

Constrúe o teu propio parágrafo respondendo nesta orde:

1. **Non bizi zara?** (Onde vives?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (Como é?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (Cantas habitacións?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (Que hai?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

## Vocabulario de peche

| Euskera | Galego |
|---|---|
| *etxebizitza* | vivenda |
| *etxejabea* | propietario |
| *alokairua* | alugueiro |
| *errenta* | renda / alugueiro mensual |
| *auzoa* | barrio |
| *erdigunea* | centro (da cidade) |
| *aldiriak* | extrarradio / arredores |

> **Consello:** cando fales da túa casa na clase, non intentes traducir cada detalle. Queda con frases curtas e ben feitas. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* é mellor descrición ca unha frase longa chea de erros.
