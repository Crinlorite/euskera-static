Esta lección integra todo o do tema 10. Aquí practicas algo concreto e útil: **describir tres veciños teus** con frases simples pero conectadas. É a proba de lume do A1 — e é exactamente o que se traballa nas primeiras presentacións de aula.

## Estrutura tipo (orde recomendada)

Para cada veciño, segue sempre estas pezas. Se as tes, a descrición constrúese soa:

1. **Presentación** — *Hau nire bizilaguna da. Bera X-koa da.* (de onde é)
2. **Idade / estado civil** — *X urte ditu. Ezkonduta / Bakarrik dago.*
3. **Profesión** — *Bera Y da.* ou *Y-an lan egiten du.*
4. **Carácter** — *Oso jatorra da.* ou *Lasaia / Berritsua da.*
5. **Detalle propio** — algo que ti observas: cando o ves, que fai.

> Non precisas dicir as cinco cousas. Con tres xa hai descrición.

## Vocabulario para o carácter do veciño

| Euskera | Galego |
|---|---|
| *jatorra* | simpático/a, próximo |
| *atsegina* | agradable |
| *alaia* | alegre |
| *lasaia* | tranquilo/a |
| *isila* | calado/a |
| *berritsua* | faladoir/a |
| *zaratatsua* | ruidoso/a |
| *zakarra* | borde / brusco |
| *jakintsua* | sabio/a |

## Texto modelo

> *Nire atarian hiru bizilagun nagusi ditugu — bakoitza solairu batean.*
>
> ***Lehenengoa Marina da.*** *Bera 70 urteko jubilatua da, lehenengo solairuan bizi da. Bakarrik bizi da, baina txakur batekin. Oso atsegina eta berritsua da: igogailuan beti hitz egiten du.*
>
> ***Bigarrena Egoitz da.*** *Bera kazetaria da eta egunkari batean lan egiten du. Bere bikotekidearekin bizi da bigarren solairuan. Lasaia da, ez dugu ia entzuten.*
>
> ***Hirugarrena Hodei eta Maialen dira.*** *Hauek nire ikaskideak ere badira: euskaltegira elkarrekin joaten gara. Hirugarren solairuan bizi dira eta ikasleak dira. Alaiak eta zaratatsuak dira: asteburuetan musika ozen entzuten dute.*

## Agora ti: describe tres veciños teus

Segue este modelo e completa cos teus datos reais (ou inventados, se prefires):

> *Nire bizilagunetatik hiru aukeratuko ditut.*
>
> *Lehenengoa _________ da. Bera _________-koa da, _________ urte ditu eta _________ da. Etxean _________ bizi da. Oso _________ da.*
>
> *Bigarrena _________ da. Bera _________ da eta _________(e)an lan egiten du. _________ solairuan bizi da. _________ da: _________ .*
>
> *Hirugarrena _________ da. Hau nire _________ ere bada. _________ ditu, eta _________(e)an lan egiten du. _________-rekin bizi da.*

## Frases conectoras útiles

Se queres encadear mellor as descricións, estes conectores sinxelos axúdanche:

| Euskera | Galego |
|---|---|
| *eta* | e |
| *baina* | pero |
| *baita ere* | tamén |
| *gainera* | ademais |
| *adibidez* | por exemplo |
| *batzuetan* | ás veces |

> Ex.: *Bera lasaia da, **baina** asteburuetan zaratatsua bihurtzen da. **Adibidez**, ostiraletan musika entzuten du gauean.*

> **Consello final:** as primeiras descricións de veciños non van ser perfectas. O importante é que quen escoita entenda **quen é**, **que fai** e **como o ves ti**. Con esa base, as correccións de gramática veñen soas coa práctica.
