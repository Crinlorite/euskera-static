Ein guter Teil des gesellschaftlichen Lebens in Euskal Herria spielt sich in der **taberna** (der Bar) ab. Wenn du die grundlegenden Getränke und Speisen auf Baskisch kennst, öffnen sich dir sofort Türen: eine Clique von Freunden, ein Besuch in einem neuen Dorf, ein *poteo*. Wir beginnen mit dem Wortschatz.

## Der Kaffee und seine Varianten

Kaffee ist wahrscheinlich das, was du am häufigsten bestellen wirst. Es gibt verschiedene Formen:

| Baskisch | Deutsch |
|---|---|
| *kafea* | der Kaffee (allgemein) |
| *kafe hutsa* | schwarzer Kaffee (wörtlich: "purer/leerer Kaffee") |
| *kafesnea* | Milchkaffee (aus *kafe* + *esne*) |
| *kafe ebakia* | Espresso mit etwas Milch (*ebaki* = "geschnitten") |
| *kafe deskafeinatua* | entkoffeinierter Kaffee |
| *infusioa* | Kräutertee |
| *tea* | Tee |

> *Kafesne* bedeutet wörtlich "Kaffee-Milch". Das Baskische bildet viele zusammengesetzte Wörter auf diese Weise, ohne Bindestrich und mit dem Hauptwort am Ende.

## Andere Getränke

| Baskisch | Deutsch |
|---|---|
| *ura* | das Wasser |
| *zumoa* | Saft |
| *esnea* | die Milch |
| *ardoa* | Wein |
| *ardo gorria* | Rotwein |
| *ardo zuria* | Weißwein |
| *garagardoa* | Bier |
| *zuritoa* | kleines Bier (typisches kleines Glas) |
| *sagardoa* | Apfelwein |
| *txakolina* | Txakoli (baskischer Weißwein) |
| *patxarana* | Pacharán |

> Drei interessante Wörter: **sagardo** (Apfelwein) = *sagar* (Apfel) + *ardo* (Wein); **garagardo** (Bier) = *garagar* (Gerste) + *ardo*; **txakolina** ist ein junger, leichter Weißwein, das Markenzeichen des Baskenlandes.

## Grundlegende Speisen in der Bar

Um das, was du trinkst, zu begleiten:

| Baskisch | Deutsch |
|---|---|
| *ogia* | das Brot |
| *gazta* | Käse |
| *urdaiazpikoa* | Schinken |
| *tortilla* | Tortilla (aus Kartoffeln) |
| *patata frijituak* | Pommes frites |
| *gozokia* | Süßigkeit (allgemein) |
| *txokolatea* | Schokolade |

## Der Artikel `-a`

Fast alle diese Wörter tragen am Ende ein **`-a`**, wenn sie "neutral" in einem Satz erscheinen: das ist der Artikel "der / die / das".

- *Ardoa gustatzen zait.* — Ich mag Wein.
- *Ura nahi dut.* — Ich möchte Wasser.

Wenn du "einen" bestellst, verschwindet das `-a` und es kommt **bat** ("ein/eine") hinzu:

- *Kafe **bat**, mesedez.* — Einen Kaffee, bitte.
- *Garagardo **bat**.* — Ein Bier.

> In Lektion 3 wirst du das im Detail sehen. Merke dir vorerst die Formen mit `-a`, um sie auf der Karte der Bar wiederzuerkennen.
