**الأفعال المتعدية** هي تلك التي تحمل مفعولًا: شخصٌ يأكل *شيئًا*، يرى *شيئًا*، يقرأ *شيئًا*. في اللغة الباسكية تستخدم هذه الأفعال الفعل المساعد **ukan** (dut، duzu، du…). في الماضي القريب تكون الصيغة:

> **partizipioa + dut/duzu/du/dugu/duzue/dute**

## الأفعال المتعدية الأساسية

| Partizipioa | أنا (لقد…) | العربية |
|---|---|---|
| *jan* | **jan dut** | لقد أكلتُ |
| *edan* | **edan dut** | لقد شربتُ |
| *ikusi* | **ikusi dut** | لقد رأيتُ |
| *irakurri* | **irakurri dut** | لقد قرأتُ |
| *entzun* | **entzun dut** | لقد استمعتُ |
| *erosi* | **erosi dut** | لقد اشتريتُ |
| *egin* | **egin dut** | لقد فعلتُ |
| *hartu* | **hartu dut** | لقد أخذتُ / تناولتُ |
| *ekarri* | **ekarri dut** | لقد أحضرتُ |
| *eraman* | **eraman dut** | لقد حملتُ |
| *ikasi* | **ikasi dut** | لقد تعلّمتُ / لقد درستُ |
| *eskatu* | **eskatu dut** | لقد طلبتُ |
| *ordaindu* | **ordaindu dut** | لقد دفعتُ |
| *idatzi* | **idatzi dut** | لقد كتبتُ |
| *prestatu* | **prestatu dut** | لقد حضّرتُ |
| *garbitu* | **garbitu dut** | لقد نظّفتُ |
| *gosaldu* | **gosaldu dut** | لقد تناولتُ الفطور |
| *bazkaldu* | **bazkaldu dut** | لقد تناولتُ الغداء |
| *afaldu* | **afaldu dut** | لقد تناولتُ العشاء |
| *lan egin* | **lan egin dut** | لقد عملتُ |

## التصريف الكامل بفعلٍ واحد

نأخذ *ikusi* (يرى) مثالًا:

| الضمير + الفاعل | الصيغة | العربية |
|---|---|---|
| Nik | **ikusi dut** | لقد رأيتُ |
| Zuk | **ikusi duzu** | لقد رأيتَ |
| Hark | **ikusi du** | لقد رأى (هو/هي) |
| Guk | **ikusi dugu** | لقد رأينا |
| Zuek | **ikusi duzue** | لقد رأيتم |
| Haiek | **ikusi dute** | لقد رأوا |

> فاعل الأفعال المتعدية يحمل علامة **NORK** (-k): *nik, zuk, hark, guk, zuek, haiek*. ستراها كثيرًا، خاصةً عندما يكون الفاعل اسمًا: *Mikelek* (Mikel قد…)، *Maialenek* (Maialen قد…). إن كان ذلك يربكك، ففي المستوى A1 يمكنك حذف الضمير والاكتفاء بالفعل: السياق يكفي.

## جُمَل نمطيّة

- *Gaur goizean egunkaria irakurri dut.* — هذا الصباح قرأتُ الجريدة.
- *Bazkaltzeko arraina jan dut.* — في الغداء أكلتُ سمكًا.
- *Lagunekin kafea hartu dut.* — تناولتُ القهوة مع الأصدقاء.
- *Mikelek ardoa erosi du.* — Mikel قد اشترى نبيذًا.
- *Guk pelikula bat ikusi dugu.* — لقد شاهدنا فيلمًا.
- *Haiek euskara ikasi dute gaur.* — لقد درسوا اللغة الباسكية اليوم.

## النفي

لقول "لم أفعل X"، تأتي **ez** قبل الفعل المساعد:

> *Ez dut bazkaldu.* — لم أتناول الغداء.
>
> *Ez du egunkaria irakurri.* — لم يقرأ الجريدة.
>
> *Ez dugu ardorik edan.* — لم نشرب نبيذًا. (لاحظ *ardorik*: مع النفي، يحمل المفعول عادةً اللاحقة التبعيضية **-rik**.)

## حوار مصغّر

> *— Zer afaldu duzu gaur, Maialen?*
>
> *— Entsalada eta arraina jan dut, eta postrerako jogurta hartu dut. Eta zuk?*
>
> *— Nik tortilla egin dut, baina ez dut postrerik jan.*

> **ملخّص بصري:** إذا كان الفعل ذا مفعولٍ صريح أو ضمني (يأكل شيئًا، يرى شيئًا، يفعل شيئًا)، فإن الفعل المساعد هو **dut** (ومشتقّاته). احفظ هذه الأزواج من الفعل والفعل المساعد، والباقي يأتي من تلقاء نفسه.
