للتعبير عمّا تفعله كل يوم، تستخدم لغة الإيوسكيرا **الصيغة الاعتيادية**: تأخذ الفعل (partizipio) وتضيف إليه **-tzen** أو **-ten**، ثم تجمعه مع الفعل المساعد **izan** (naiz، zara، da…) أو **ukan** (dut، duzu، du…).

## القاعدة

| نهاية الفعل | لاحقة الصيغة الاعتيادية | مثال |
|---|---|---|
| حرف ساكن أو `-tu` / `-du` | **-tzen** | *esnatu* ← *esnatzen* |
| `-n` | **-ten** | *jan* ← *jaten*؛ *egin* ← *egiten* |
| `-i` (شاذ) | **-tzen** أو **-ten** | *etorri* ← *etortzen*؛ *ikusi* ← *ikusten* |

> الفعل *jan* (الأكل) **لا** يتحوّل إلى *janten* بل إلى *jaten* — الحرف "n" في النهاية يختفي. والأمر نفسه مع *edan* ← *edaten*، *egon* ← *egoten*.

## أفعال الحياة اليومية

### NOR (الفعل المساعد **izan**: naiz، zara، da…)

هذه تأتي **مع izan** لأنها أفعال لازمة — الفاعل يختبر الفعل دون وجود مفعول به:

| Partizipioa | الصيغة الاعتيادية | المعنى |
|---|---|---|
| *esnatu* | *esnatzen naiz* | أستيقظ |
| *jaiki* | *jaikitzen naiz* | أنهض |
| *altxatu* | *altxatzen naiz* | أنهض (صيغة مرادفة لـ *jaiki*) |
| *dutxatu* | *dutxatzen naiz* | أستحمّ |
| *jantzi* | *janzten naiz* | أرتدي ملابسي |
| *joan* | *joaten naiz* | أذهب |
| *etorri* | *etortzen naiz* | آتي |
| *oheratu* | *oheratzen naiz* | أذهب إلى الفراش |
| *lokartu* | *lokartzen naiz* | أغفو |

### NOR-NORK (الفعل المساعد **ukan**: dut، duzu، du…)

هذه تأتي **مع ukan** لأنها تملك فاعلًا ومفعولًا به:

| Partizipioa | الصيغة الاعتيادية | المعنى |
|---|---|---|
| *gosaldu* | *gosaltzen dut* | أتناول الفطور |
| *bazkaldu* | *bazkaltzen dut* | أتناول الغداء |
| *afaldu* | *afaltzen dut* | أتناول العشاء |
| *jan* | *jaten dut* | آكل |
| *edan* | *edaten dut* | أشرب |
| *lan egin* | *lan egiten dut* | أعمل |
| *lo egin* | *lo egiten dut* | أنام |
| *ikusi* | *ikusten dut* | أرى |
| *irakurri* | *irakurtzen dut* | أقرأ |
| *erosi* | *erosten dut* | أشتري |

> **لماذا تأتي بعض الأفعال مع NAIZ وأخرى مع DUT؟** هذا هو التمييز الأساسي في لغة الإيوسكيرا. *Esnatzen naiz* (أستيقظ) شيء يحدث لي أنا: لا يوجد مفعول به. أمّا *Jaten dut* (آكل) فلها مفعول به: *ogia jaten dut* (آكل خبزًا). يُشرح هذا بالتفصيل في الوحدة القادمة — في الوقت الحالي احفظ أزواج الفعل + الفعل المساعد.

## أمثلة مترابطة

> *Egunero zazpietan esnatzen naiz eta dutxatzen naiz.* — كل يوم أستيقظ في الساعة السابعة وأستحمّ.
>
> *Goizean kafea hartzen dut.* — في الصباح أتناول القهوة.
>
> *Askotan lagunekin afaltzen dut.* — غالبًا أتناول العشاء مع الأصدقاء.
>
> *Batzuetan lo-kuluxka egiten dut bazkalostean.* — أحيانًا آخذ قيلولة بعد الغداء.

> **ملاحظة:** *bazkalostean* = "بعد الغداء" (حرفيًا: "عقب الوجبة"). تملك ثقافة الإيوسكيرا كلمات مكثّفة لمفاهيم يومية مثل هذه القيلولة القصيرة — *lo-kuluxka* هي بالضبط ذلك.
