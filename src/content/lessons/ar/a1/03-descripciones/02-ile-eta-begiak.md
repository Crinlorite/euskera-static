لوصف الشعر والعيون في الإيوسكيرا لا يُستخدم **izan** ("يكون") كما في الإسبانية ("أنا أسمر")، بل **ukan** ("يملك"): *لديّ شعر أسود*، *لديّ عينان زرقاوان*.

## الشعر: ile

الكلمة الدالة على "الشعر" هي **ile** (مع أداة التعريف: *ilea*). تأتي الصفات بعده، والأخيرة فقط هي التي تحمل العلامة `-a`:

| الإيوسكيرا | الإسبانية |
|---|---|
| *ile beltza* | شعر أسود |
| *ile horia* | شعر أشقر |
| *ile zuria* | شعر أبيض / أشيب |
| *ile gorria* | شعر أحمر |
| *ile urdina* | شعر رمادي |
| *ile motza* | شعر قصير |
| *ile luzea* | شعر طويل |
| *ile lisoa* | شعر أملس |
| *ile kizkurra* | شعر مجعّد |

> يُعامَل الشعر في الإيوسكيرا بوصفه **مفرداً**، تماماً كما في الإسبانية. لهذا يُستخدم *dut* / *du* (وليس *ditut* / *ditu*).

- *Ile beltza eta luzea **dut**.* — لديّ شعر أسود وطويل.
- *Maialenek ile kizkurra **du**.* — لدى Maialen شعر مجعّد.

## العيون: begiak

خلافاً للشعر، "العيون" تكون في صيغة **الجمع** (*begiak*). لذلك تأتي الصفات أيضاً في صيغة الجمع (`-ak`) ويُستخدم **ditut / ditu**:

| الإيوسكيرا | الإسبانية |
|---|---|
| *begi urdinak* | عيون زرقاء |
| *begi marroiak* | عيون بنيّة |
| *begi berdeak* | عيون خضراء |
| *begi beltzak* | عيون سوداء |
| *begi argiak* | عيون فاتحة |
| *begi ilunak* | عيون داكنة |

- *Begi urdinak **ditut**.* — لديّ عينان زرقاوان.
- *Mikelek begi marroiak **ditu**.* — لدى Mikel عينان بنيّتان.

## ملامح أخرى: اللاحقة -dun

للقول "مَن يرتدي X" أو "ذو X"، تملك الإيوسكيرا لاحقة منتجة جداً: **-duna** (مفرد) / **-dunak** (جمع). تُضاف إلى الاسم فيصبح الشخص "صاحب النظارة"، "صاحب اللحية"...

| الإيوسكيرا | الإسبانية |
|---|---|
| *betaurrekoduna* | مع نظارة / صاحب النظارة |
| *bizarduna* | ذو لحية |
| *biboteduna* | ذو شارب |
| *burusoila* | أصلع (حرفياً "رأس عارٍ") |

أمّا هذه فتأتي مع **izan**، لأنها تصف صفة مثل الصفة العادية:

- *Hura biboteduna da.* — هو ذو شارب.
- *Nire aitona burusoila da.* — جدّي أصلع.

## الجمع بين كل شيء

البورتريه النموذجي يمزج بين عدة تراكيب:

> *Maialen txikia eta argala da. Ile gorria eta kizkurra du. Begi berdeak ditu. Betaurrekoduna da.*
>
> Maialen قصيرة ونحيلة. لديها شعر أحمر ومجعّد. لديها عينان خضراوان. ترتدي نظارة.

لاحظ كيف يتوزّع الأمر: **izan** للصفات العامة (*txikia da*)، و**ukan** للشعر/العيون (*ile … du*, *begi … ditu*)، و*-dun*-a مرة أخرى مع **izan** للإكسسوارات.
