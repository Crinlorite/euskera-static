La 1ª idazlana (180 hitz) suele ofrecer dos opciones: una **carta formal** (gutun formala) y una **carta personal** (gutun pertsonala). La diferencia entre ellas no está en el contenido sino en el **registro** y en las **fórmulas fijas** de apertura y cierre.

## Carta formal — gutun formala

Se usa para queja, reclamación, petición o solicitud a una administración, empresa o institución.

### Estructura tipo

```
[Lugar, fecha]

[Destinatario]:

[Cuerpo: planteamiento del asunto, exposición de hechos, petición]

[Despedida formal]

[Firma]
```

### Saludos formales

- *Agurgarria,* — neutro formal (cuando el destinatario es individual y específico).
- *Jaun-andre agurgarriak,* — para destinatarios institucionales (comité, departamento).
- *[Cargo] agurgarria,* — *Zinegotzi agurgarria,* / *Zuzendari agurgarria,*.

### Aperturas tipo

- *Honen bidez jakinarazi nahi nizueke …*
- *Eskutitz hau …-ri buruzko kexa adieraztera dator.*
- *Helburu honekin idazten dizut: …*
- *Honako gutun honen bidez, eskaera bat egin nahi dizuet.*
- *Lerro hauen bidez nire desadostasuna agertu nahi dut.*

### Verbos y locuciones útiles

| Función | Fórmulas |
|---|---|
| Plantear el asunto | *jakinarazi nahi dizuet*, *adierazi behar dut*, *azaldu nahi dizuet* |
| Pedir | *eskatzen dizuet*, *eskaera hau egiten dizuet*, *mesedez …* |
| Quejarse | *kexa agertu*, *ez nago ados …-rekin*, *salatu nahi dut* |
| Justificar | *izan ere, …*, *honako arrazoi hau dela medio*, *aurreko egunetan zera gertatu zen: …* |
| Pedir respuesta | *erantzunaren zain geratzen naiz*, *ahalik eta lasterren erantzungo didazuela espero dut* |
| Agradecer | *aldez aurretik eskerrak emanez*, *eskerrik asko zuen arretagatik* |

### Despedidas formales

- *Adeitasunez,*
- *Begirunez,*
- *Agur bero bat,* (semi-formal)
- *Eskerrik asko, agur t'erdi.* (NO en formal)

### Lo que se PENALIZA

- Tutear sin transición (*hi, motel, lagun*).
- Usar muletillas coloquiales (*ba*, *bueno*, *ezta?*).
- Diminutivos afectivos (*-txo, -txu* expresivos).
- Errores de tratamiento (*zuek* / *zu* mezclados sin razón).

## Carta personal — gutun pertsonala

Se usa para situaciones informales: contar algo a un amigo, agradecer, invitar, dar una noticia personal.

### Saludos informales

- *Kaixo, [izena]!*
- *Aupa, [izena]!*
- *Lagun maitea,* (algo más formal)
- *[Izena] maitea,*

### Aperturas tipo

- *Aspaldi ez dugula elkarren berririk! …*
- *Gutun hau idazten dizut zera kontatzeko: …*
- *Ez dakit zergatik, baina hilabete honetan ez dut zugaz harremanetan jartzeko adorerik izan, …*
- *Zer moduz? Aspaldikoa naiz!*

### Tono y léxico

- Tutea libremente.
- Diminutivos afectivos OK: *txikitxo*, *etxetxoa*.
- Anécdotas personales y humor son válidos.
- Lenguaje vivo, expresivo, no académico.

### Despedidas informales

- *Muxu handi bat!*
- *Besarkada bat!*
- *Agur, eta laster arte!*
- *Maite zaitut!*
- *Idatzi laster!*

### Lo que se PENALIZA

- Saltar al registro formal en mitad de la carta (incoherencia).
- Falta de contenido emocional o personal (queda fría).
- Fórmulas administrativas (*honen bidez jakinarazi nahi nizueke*).
- Discurso argumentativo en vez de narrativo o relacional.

## Mini-plantilla: queja formal

```
Donostia, 2026ko maiatzaren 8a.

Udaleko Zerbitzuak agurgarriak,

Eskutitz honen bidez azken hilabetean nire auzoan
gertatzen ari den arazo bati buruzko kexa adierazi
nahi dizuet. Izan ere, …

[Hechos: 2-3 frases concretas con datos]

[Petición: qué quieres que se haga]

Aldez aurretik eskerrak emanez, ahalik eta lasterren
erantzunaren zain geratzen naiz.

Adeitasunez,

[Izen-abizenak]
```

## Mini-plantilla: carta personal

```
Bilbo, 2026ko maiatzak 8.

Maite!

Aspaldi ez dugula elkarren berririk eta gaur
gauean hartu dut, azkenean, idazteko gogoa.
…

[Cuerpo: noticias personales, anécdotas, preguntas a la otra persona]

Idatzi laster, eta jakinarazi nola zauden!

Muxu handi bat,

Egoitz
```

> **Resumen:** la carta formal pide registro elevado, fórmulas fijas y precisión; la personal pide cercanía, voz propia y tono natural. Decide en el primer minuto cuál escribes y mantén el registro hasta la firma.
