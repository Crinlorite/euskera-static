El paso de B2 a C1 se nota, sobre todo, en el **léxico abstracto**. Quien sigue narrando con palabras concretas se queda atrás; quien sabe nominalizar y manejar abstracciones, sube de nivel.

## Familias léxicas C1 imprescindibles

### Causa-efecto

- **eragin** — impacto, influencia. *Eraginkorra, eraginpean, eragiteko gaitasuna.*
- **ondorio** — consecuencia. *Ondorio sakonak, ondoriozkoa, ondorioztatu.*
- **kausa** — causa. *Sustrai, oinarri, abiapuntu* — sinónimos según matiz.

### Postura, opinión

- **jarrera** — actitud, postura. *Jarrera kritikoa, jarrera irekia, jarrera aldatu.*
- **iritzi** — opinión. *Iritzi orokorra, iritzi zorrotza.*
- **ikuspegi** — perspectiva. *Ikuspegi zabala, ikuspegi historikoa.*

### Reflexión, debate

- **gogoeta** — reflexión. *Gogoeta sakona, gogoeta egin.*
- **hausnarketa** — meditación, sopesar. *Hausnarketa-prozesua.*
- **eztabaida** — debate.
- **auzia** — el asunto, la cuestión.

### Sociedad y estructura

- **gizarte-eredua** — modelo de sociedad.
- **pentsamoldea** — mentalidad.
- **aukera-berdintasuna** — igualdad de oportunidades.
- **iraunkortasuna** — sostenibilidad.
- **gardentasuna** — transparencia.
- **berdintasuna** — igualdad.

## Nominalización: cómo convertir verbos en sustantivos

El euskera tiene una **familia rica de sufijos** para crear sustantivos abstractos. Conocerlos amplía rápidamente el léxico.

### Sufijo -tasun (cualidad)

Adjetivo + tasun = sustantivo abstracto de cualidad.

| Adjetivo | Nominal |
|---|---|
| aske | askatasun (libertad) |
| malgu | malgutasun (flexibilidad) |
| ahul | ahultasun (debilidad) |
| zintzo | zintzotasun (honradez) |
| iraunkorra | iraunkortasun (sostenibilidad) |

### Sufijo -keta (acción / resultado)

Verbo / raíz + keta = acción o su resultado.

| Raíz | Nominal |
|---|---|
| ikertu | ikerketa (investigación) |
| eztabaidatu | eztabaida (debate, sin -keta también) |
| aldatu | aldaketa (cambio) |
| neurtu | neurketa (medición) |

### Sufijo -kuntza (proceso, oficio, sistema)

Indica proceso o sistema completo.

| Raíz | Nominal |
|---|---|
| eraiki | eraikuntza (construcción) |
| heziketa-base | heziketa / hezkuntza (educación) |
| antolatu | antolakuntza (organización) |
| aldatu | aldakuntza (proceso de cambio) |

### Sufijo -men (facultad, capacidad)

Indica capacidad o facultad de.

| Raíz | Nominal |
|---|---|
| erabaki | erabakimen (capacidad de decisión) |
| ahalmena (poder, capacidad) |
| sentimena (sentimiento) |
| iritzi → iritzi-emaile *(otro patrón)* |

### Sufijo -pen (acción / efecto, frecuente con verbos)

| Raíz | Nominal |
|---|---|
| asebete | asebetetzea / asebetepena (satisfacción) |
| onartu | onarpen (aceptación, aprobación) |
| ulertu | ulermen (comprensión) |
| sortu | sorkuntza/sorpen (creación) |

### Sufijo -tza (estado, oficio, abstracto)

| Raíz | Nominal |
|---|---|
| nagusi → nagusitza (predominio) |
| irakasle → irakaskuntza (enseñanza) |
| izan → izaera (carácter) |

## Cómo usar nominalizaciones para densificar

Frase B2:

> *Erabakitzeko aukera daukate eta gainera negoziatzen jakin behar dute.*

Versión C1 con nominalización:

> *Erabakimena eta negoziazio-gaitasuna dituzte ezinbestekoak.*

Más densa, menos verbal, más abstracta.

## Cuándo NO nominalizar

La nominalización en exceso vuelve el texto burocrático. Si todo son sustantivos abstractos sin verbos vivos, el texto se ahoga.

> *"Telelana-eredu hau ezarpena gauzatu zenetik, langileen-egokitzapen-prozesua aurrera eraman izana ez da inolako-zalantzarik."*

Eso no es C1, es Boletín Oficial. Alterna nominal y verbal.

> **Pista:** una nominalización por frase suele bastar. Dos en frases consecutivas, también. Tres o más seguidas, ya es indigestión.

## Sustitutos sinonímicos para palabras frecuentes

| Palabra común | Alternativas formales |
|---|---|
| arazo | erronka, oztopo, zailtasun, korapilo, auzia |
| onura | abantaila, etekin, ekarpena, mesedea |
| txarra | kaltegarria, eskasa, desegokia, problematikoa |
| ona | aproposa, baliagarria, eraginkorra, oparoa |
| handia | nabarmena, esanguratsua, sakona, hedatua |
| pentsatu | hausnartu, gogoeta egin, irizten, ohartu |
| aldatu | eraldatu, moldatu, biratu, doitu |

Saber alternar tres o cuatro versiones de cada palabra-comodín distingue el C1 del B2.
