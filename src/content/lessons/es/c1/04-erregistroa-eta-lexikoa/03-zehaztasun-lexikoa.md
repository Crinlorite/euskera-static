C1 es **precisión**. La diferencia entre "*aldaketa handia*" y "*aldaketa sakona*" parece pequeña; en una rúbrica IVAP/HABE, distingue.

## Adjetivos para "intenso, fuerte, importante"

Cinco palabras que NO son intercambiables del todo:

| Adjetivo | Matiz |
|---|---|
| **handi** | Genérico: grande. Vale, pero no destaca. |
| **nabarmen** | Visible, llamativo, evidente. |
| **sakon** | Profundo: tiene calado interno. |
| **esanguratsu** | Significativo: tiene sentido o trascendencia. |
| **funtsezko** | Fundamental, esencial: sin él, el conjunto no se sostiene. |
| **erabakigarri** | Decisivo: inclina la balanza. |

Aplícalos a *aldaketa* (cambio):

- *aldaketa handia* — neutro, B1.
- *aldaketa nabarmena* — observable, mejor.
- *aldaketa sakona* — afecta a estructuras, C1.
- *aldaketa esanguratsua* — con peso simbólico, C1.
- *aldaketa funtsezkoa* — clave para todo lo demás.
- *aldaketa erabakigarria* — el que decide el resultado.

## Adjetivos para "negativo"

| Adjetivo | Cuándo |
|---|---|
| **txarra** | Genérico, B1. |
| **eskasa** | Insuficiente, de baja calidad. |
| **desegokia** | Inadecuado, fuera de lugar. |
| **kaltegarria** | Dañino, perjudicial. |
| **arriskutsua** | Peligroso. |
| **kezkagarria** | Preocupante. |
| **larria** | Grave. |

> Una situación *kezkagarria* invita a tomar nota. Una situación *larria* exige acción. *Arriskutsua* añade dimensión de peligro futuro.

## Verbos de cambio y movimiento argumentativo

| Verbo | Uso |
|---|---|
| **areagotu** | Aumentar, intensificarse. *Tirabirak areagotu dira.* |
| **eskuratu** | Lograr, obtener. *Eskuragarri dauden datuak.* |
| **eragin** | Producir, causar. *Erabakiak ondorioak eragin ditu.* |
| **bideratu** | Encauzar, dirigir. *Konponbidera bideratu nahi dugu auzia.* |
| **azpimarratu** | Subrayar, destacar. *Azpimarratu beharrekoa da...* |
| **gainditu** | Superar. *Krisia gainditzeko aukerak.* |
| **ezarri** | Implantar, establecer. *Lege berria ezarri zen.* |
| **eraldatu** | Transformar. *Ohitura sozialak eraldatu dira.* |

## Verbos de pensamiento y enunciación

| Verbo | Matiz |
|---|---|
| **uste izan** | Creer, opinar (general). |
| **iritzi** | Opinar (más formal). |
| **ohartu** | Darse cuenta, advertir. |
| **iruditu** | Parecer (subjetivo). |
| **azpimarratu** | Subrayar (énfasis). |
| **adierazi** | Manifestar, expresar. |
| **defendatu** | Defender (postura). |
| **frogatu** | Demostrar. |
| **proposatu** | Proponer. |

> *"Adituek azpimarratu dute..."* es más fuerte que *"Adituek esan dute..."*. La elección verbal pesa.

## Colocaciones (kolokazioak)

Una **colocación** es una combinación habitual de palabras. C1 evalúa que sepas combinar bien.

| Sustantivo | Verbo / adjetivo habitual |
|---|---|
| *aldaketa* | nabarmena, sakona, eraldatzailea |
| *erabakia* | hartu, atzeratu, justifikatu |
| *erronka* | aurre egin, gainditu, planteatu |
| *eztabaida* | piztu, mantendu, itxi |
| *konfiantza* | irabazi, galdu, eman |
| *bidea* | jorratu, ireki, hautatu |

Las colocaciones agudizadas son señal de competencia léxica avanzada. *"Eztabaida piztu"* (encender el debate) es C1; *"eztabaida egin"* (hacer debate) es genérico.

## Falsos sinónimos: trampas frecuentes

- **handi vs zabal vs hedatu** — *handi* (grande), *zabal* (amplio, ancho), *hedatu* (extendido). No son intercambiables.
- **arazoa vs auzi vs gatazka** — *arazoa* (problema), *auzi* (cuestión, asunto), *gatazka* (conflicto, choque).
- **zaila vs konplexu vs korapilatsua** — *zail* (difícil), *konplexu* (con muchos elementos), *korapilatsua* (enredado, lioso).
- **iritzia vs uste vs aburua** — *iritzi* (opinión, formal), *uste* (creencia), *aburu* (parecer, formal-libresco).

> **Pista C1 final:** cuando dudes entre dos sinónimos, elige el más **específico**, no el más rebuscado. C1 premia precisión, no exotismo. *"Aldaketa sakona"* gana a *"aldakuntza intzidente bat"*.
