Cuando alguien te pregunta cómo llegar a un sitio — o cuando te pierdes y preguntas tú — necesitas un puñado de verbos imperativos y de adverbios de dirección. En esta lección reunimos lo esencial.

## Imperativos básicos

En euskera, el imperativo afirmativo de los verbos de movimiento es muy directo:

| Forma | Verbo base | Sentido |
|---|---|---|
| **Jo** | jo | golpea / dirige (jo ezkerrera = gira a la izquierda) |
| **Segi** / **Jarraitu** | segitu / jarraitu | sigue, continúa |
| **Hartu** | hartu | toma (jartu kale hau = toma esta calle) |
| **Igo** | igo | sube |
| **Jaitsi** | jaitsi | baja |
| **Sartu** | sartu | entra |
| **Irten** | irten | sal |
| **Zeharkatu** | zeharkatu | cruza |
| **Iritsi** / **heldu** | iritsi / heldu | llega (… arte = "hasta…") |

> *Jo* literalmente significa "golpear", pero se usa metafóricamente como "dirigirse hacia": **jo ezkerrera** = "tira hacia la izquierda".

## Adverbios y referencias direccionales

| Euskera | Castellano |
|---|---|
| *ezkerra* | la izquierda |
| *eskuina* | la derecha |
| *aurrera* | hacia delante |
| *atzera* | hacia atrás |
| *zuzen* | recto, derecho |
| *gora* | hacia arriba |
| *behera* | hacia abajo |

> Fíjate: para indicar **dirección hacia** se usa el sufijo `-ra` (que verás a fondo en la lección 03): *ezkerre-ra* (a la izquierda), *aurre-ra* (hacia delante), *atze-ra* (hacia atrás). Lo mismo para *gora*, *behera*.

## Lugares de referencia típicos

| Euskera | Castellano |
|---|---|
| *semaforoa* | semáforo |
| *bidegurutzea* | cruce, intersección |
| *zebrabidea* | paso de cebra |
| *biribilgunea* / *errotonda* | rotonda |
| *aldapa* | cuesta |
| *eskailerak* | escaleras |
| *zubia* | puente |
| *errepidea* | carretera |
| *kalea* | calle |

## Frases tipo

- **Jo ezkerrera bidegurutzean.** — Gira a la izquierda en el cruce.
- **Segi aurrera semaforora arte.** — Sigue recto hasta el semáforo.
- **Zeharkatu zebrabidea.** — Cruza el paso de cebra.
- **Igo eskailerak eta sartu eraikinean.** — Sube las escaleras y entra en el edificio.
- **Jaitsi aldapa, eta gero jo eskuinera.** — Baja la cuesta, y luego gira a la derecha.
- **Egin atzera; gehiegi joan zara.** — Vuelve atrás; te has pasado.

## Pedir y dar indicaciones

> — *Barkatu, **non dago** udaletxea?* — Perdona, ¿dónde está el ayuntamiento?
>
> — ***Segi aurrera** semaforora arte. Han **jo eskuinera** eta **zeharkatu** plaza. Udaletxea elizaren ondoan dago.* — Sigue recto hasta el semáforo. Allí gira a la derecha y cruza la plaza. El ayuntamiento está al lado de la iglesia.
>
> — *Eta gertu **dago**?* — ¿Y está cerca?
>
> — *Bai, oso gertu. Bost minutu oinez.* — Sí, muy cerca. Cinco minutos a pie.

> **Truco:** muchas indicaciones siguen un patrón fijo: imperativo (*jo*, *segi*, *zeharkatu*) + lugar con sufijo (*ezkerrera*, *semaforora*, *zebrabidea*). Si memorizas el patrón, puedes encadenar instrucciones largas combinando piezas.
