Para describir el pelo y los ojos en euskera no se usa **izan** ("ser") como en castellano ("soy moreno"), sino **ukan** ("tener"): *tengo el pelo negro*, *tengo los ojos azules*.

## Pelo: ile

La palabra para "pelo" es **ile** (con artículo: *ilea*). Los adjetivos van detrás, y solo el último lleva la marca `-a`:

| Euskera | Castellano |
|---|---|
| *ile beltza* | pelo negro |
| *ile horia* | pelo rubio |
| *ile zuria* | pelo blanco / canoso |
| *ile gorria* | pelo pelirrojo |
| *ile urdina* | pelo gris |
| *ile motza* | pelo corto |
| *ile luzea* | pelo largo |
| *ile lisoa* | pelo liso |
| *ile kizkurra* | pelo rizado |

> El pelo en euskera se trata como **singular**, igual que en castellano. Por eso se usa *dut* / *du* (no *ditut* / *ditu*).

- *Ile beltza eta luzea **dut**.* — Tengo el pelo negro y largo.
- *Maialenek ile kizkurra **du**.* — Maialen tiene el pelo rizado.

## Ojos: begiak

A diferencia del pelo, "los ojos" es **plural** (*begiak*). Por tanto los adjetivos van también en plural (`-ak`) y se usa **ditut / ditu**:

| Euskera | Castellano |
|---|---|
| *begi urdinak* | ojos azules |
| *begi marroiak* | ojos marrones |
| *begi berdeak* | ojos verdes |
| *begi beltzak* | ojos negros |
| *begi argiak* | ojos claros |
| *begi ilunak* | ojos oscuros |

- *Begi urdinak **ditut**.* — Tengo los ojos azules.
- *Mikelek begi marroiak **ditu**.* — Mikel tiene los ojos marrones.

## Otros rasgos: el sufijo -dun

Para decir "que lleva X" o "con X", el euskera tiene un sufijo muy productivo: **-duna** (singular) / **-dunak** (plural). Se añade al sustantivo y la persona pasa a ser "el de las gafas", "el de la barba"...

| Euskera | Castellano |
|---|---|
| *betaurrekoduna* | con gafas / el de las gafas |
| *bizarduna* | con barba |
| *biboteduna* | con bigote |
| *burusoila* | calvo (literalmente "cabeza pelada") |

Estos sí van con **izan**, porque describen una cualidad como un adjetivo:

- *Hura biboteduna da.* — Él lleva bigote.
- *Nire aitona burusoila da.* — Mi abuelo es calvo.

## Combinando todo

Un retrato típico mezcla varias estructuras:

> *Maialen txikia eta argala da. Ile gorria eta kizkurra du. Begi berdeak ditu. Betaurrekoduna da.*
>
> Maialen es bajita y delgada. Tiene el pelo pelirrojo y rizado. Tiene los ojos verdes. Lleva gafas.

Fíjate cómo se reparte: **izan** para cualidades generales (*txikia da*), **ukan** para pelo/ojos (*ile … du*, *begi … ditu*), y *-dun*-a otra vez con **izan** para los accesorios.
