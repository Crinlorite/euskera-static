Esta lección integra todo lo del tema 10. Aquí practicas algo concreto y útil: **describir a tres vecinos tuyos** con frases simples pero conectadas. Es la prueba de fuego de A1 — y es exactamente lo que se trabaja en las primeras presentaciones de aula.

## Estructura tipo (orden recomendado)

Para cada vecino, sigue siempre estas piezas. Si las tienes, la descripción se construye sola:

1. **Presentación** — *Hau nire bizilaguna da. Bera X-koa da.* (de dónde es)
2. **Edad / estado civil** — *X urte ditu. Ezkonduta / Bakarrik dago.*
3. **Profesión** — *Bera Y da.* o *Y-an lan egiten du.*
4. **Carácter** — *Oso jatorra da.* o *Lasaia / Berritsua da.*
5. **Detalle propio** — algo que tú observas: cuándo le ves, qué hace.

> No necesitas decir las cinco cosas. Con tres ya hay descripción.

## Vocabulario para el carácter del vecino

| Euskera | Castellano |
|---|---|
| *jatorra* | simpático/a, cercano |
| *atsegina* | agradable |
| *alaia* | alegre |
| *lasaia* | tranquilo/a |
| *isila* | callado/a |
| *berritsua* | charlatán/a |
| *zaratatsua* | ruidoso/a |
| *zakarra* | borde / brusco |
| *jakintsua* | sabio/a |

## Texto modelo

> *Nire atarian hiru bizilagun nagusi ditugu — bakoitza solairu batean.*
>
> ***Lehenengoa Marina da.*** *Bera 70 urteko jubilatua da, lehenengo solairuan bizi da. Bakarrik bizi da, baina txakur batekin. Oso atsegina eta berritsua da: igogailuan beti hitz egiten du.*
>
> ***Bigarrena Mikel da.*** *Bera kazetaria da eta egunkari batean lan egiten du. Bere bikotekidearekin bizi da bigarren solairuan. Lasaia da, ez dugu ia entzuten.*
>
> ***Hirugarrena Aitor eta Maialen dira.*** *Hauek nire ikaskideak ere badira: euskaltegira biok joaten gara. Hirugarren solairuan bizi dira eta ikasleak dira. Alaiak eta zaratatsuak dira: asteburuetan musika ozen entzuten dute.*

## Ahora tú: describe tres vecinos tuyos

Sigue esta plantilla y completa con tus datos reales (o inventados, si prefieres):

> *Nire bizilagunetatik hiru aukeratuko ditut.*
>
> *Lehenengoa _________ da. Bera _________-koa da, _________ urte ditu eta _________ da. Etxean _________ bizi da. Oso _________ da.*
>
> *Bigarrena _________ da. Bera _________ da eta _________(e)an lan egiten du. _________ solairuan bizi da. _________ da: _________ .*
>
> *Hirugarrena _________ da. Hau nire _________ ere bada. _________ ditu, eta _________(e)an lan egiten du. _________-rekin bizi da.*

## Frases conectoras útiles

Si quieres encadenar mejor las descripciones, estas conectores sencillos te ayudan:

| Euskera | Castellano |
|---|---|
| *eta* | y |
| *baina* | pero |
| *baita ere* | también |
| *gainera* | además |
| *adibidez* | por ejemplo |
| *batzuetan* | a veces |

> Ej.: *Bera lasaia da, **baina** asteburuetan zaratatsua bihurtzen da. **Adibidez**, ostiraletan musika entzuten du gauean.*

> **Consejo final:** las primeras descripciones a vecinos no van a ser perfectas. Lo importante es que el oyente entienda **quién es**, **qué hace** y **cómo lo ves tú**. Con esa base, las correcciones de gramática vienen solas con la práctica.
