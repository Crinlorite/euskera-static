Para hablar de planes hay que dominar los marcadores temporales. Aquí tienes los del eje pasado-presente-futuro y los más usados para el futuro próximo.

## El eje básico

| Euskera | Castellano |
|---|---|
| *atzo* | ayer |
| *gaur* | hoy |
| *bihar* | mañana |
| *etzi* | pasado mañana |
| *etziluma / etzidamu* | "tras-pasado mañana" (poco usado) |

Tres palabras cortas (*atzo, gaur, bihar*) y *etzi* — con eso resuelves casi cualquier conversación cotidiana sobre fechas.

## El sufijo `datorren` (que viene)

Para hablar de "la próxima semana", "el próximo mes" se usa **datorren** + nombre:

| Euskera | Castellano |
|---|---|
| *datorren astean* | la semana que viene |
| *datorren astelehenean* | el lunes que viene |
| *datorren hilean* | el mes que viene |
| *datorren urtean* | el año que viene |

`Datorren` significa "que viene" (es la forma de relativo del verbo *etorri*, venir). Lo importante es aprenderlo como un bloque.

## "Este/esta" — `hau` / `honetan`

Para "esta semana", "este fin de semana":

- *aste honetan* — esta semana
- *asteburu honetan* — este fin de semana
- *hilabete honetan* — este mes

## Otros marcadores clave

| Euskera | Castellano |
|---|---|
| *orain* | ahora |
| *gero* | luego, después |
| *lehen* | antes |
| *gaur goizean* | esta mañana |
| *gaur arratsaldean* | esta tarde |
| *gaur gauean* | esta noche |
| *bihar goizean* | mañana por la mañana |

> *Goiza* = mañana (parte del día). *Bihar* = mañana (día siguiente). Son dos palabras distintas, ¡no las confundas!

## Hacer preguntas sobre tiempo

- **Noiz?** — ¿Cuándo?
- **Noiz etorriko zara?** — ¿Cuándo vendrás?
- **Noiz egingo dugu afaria?** — ¿Cuándo haremos la cena?

## Ejemplos en contexto

- *Bihar mendira joango naiz.* — Mañana iré al monte.
- *Etzi euskara klasea dut.* — Pasado mañana tengo clase de euskera.
- *Datorren astean lan asko izango dut.* — La semana que viene tendré mucho trabajo.
- *Asteburu honetan ez dut planik.* — Este fin de semana no tengo plan.

> **Nota cultural:** en euskera "mañana" como momento del día es *goiza* y "mañana" como día siguiente es *bihar*. La frase *bihar goizean* ("mañana por la mañana") junta los dos. En las primeras conversaciones es muy frecuente fallar este matiz, así que conviene fijarlo desde el principio.
