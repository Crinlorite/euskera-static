Esta lección integra todo lo del tema 9: las habitaciones, los muebles, las tareas y con quién vives. El objetivo es que puedas **describir tu casa** en frases sencillas, como si te estuvieran preguntando en clase o en una conversación informal.

## Las preguntas básicas

| Pregunta | Castellano |
|---|---|
| *Non dago zure etxea?* | ¿Dónde está tu casa? |
| *Nolakoa da?* | ¿Cómo es? |
| *Zenbat gela ditu?* | ¿Cuántas habitaciones tiene? |
| *Norekin bizi zara?* | ¿Con quién vives? |
| *Zenbatgarren solairuan bizi zara?* | ¿En qué planta vives? |
| *Etxeak badu igogailua / balkoia / terraza?* | ¿La casa tiene ascensor / balcón / terraza? |

## Adjetivos útiles para describir la casa

| Euskera | Castellano |
|---|---|
| *handia* | grande |
| *txikia* | pequeña |
| *argitsua* | luminosa |
| *iluna* | oscura |
| *berria* | nueva |
| *zaharra* | vieja |
| *lasaia* | tranquila |
| *zaratatsua* | ruidosa |
| *erosoa* | cómoda |
| *garestia* | cara |
| *merkea* | barata |

> Recuerda el patrón **NOLAKOA da?** ya visto en otras unidades: la respuesta lleva el adjetivo + `-a` final + verbo *izan*. Ej.: *Nire etxea **handia da**.* — Mi casa es grande.

## Un párrafo modelo

Aquí tienes un texto sencillo describiendo una casa. Léelo en voz alta y fíjate en los patrones.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## Tu turno: describe tu casa

Construye tu propio párrafo respondiendo en este orden:

1. **Non bizi zara?** (¿Dónde vives?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (¿Cómo es?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (¿Cuántas habitaciones?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (¿Qué hay?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

## Vocabulario de cierre

| Euskera | Castellano |
|---|---|
| *etxebizitza* | vivienda |
| *etxejabea* | propietario |
| *alokairua* | alquiler |
| *errenta* | renta / alquiler mensual |
| *auzoa* | barrio |
| *erdigunea* | centro (de la ciudad) |
| *hiriaurrea* | extrarradio / afueras |

> **Consejo:** cuando hables de tu casa en clase, no intentes traducir cada detalle. Quédate con frases cortas y bien hechas. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* es mejor descripción que una frase larga llena de errores.
