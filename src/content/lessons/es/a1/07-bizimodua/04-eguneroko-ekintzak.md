Para decir lo que haces todos los días, el euskera usa el **aspecto habitual**: tomas el verbo (partizipio) y le añades **-tzen** o **-ten**, después combinas con el verbo auxiliar **izan** (naiz, zara, da…) o **ukan** (dut, duzu, du…).

## La regla

| Final del verbo | Sufijo habitual | Ejemplo |
|---|---|---|
| consonante o `-tu` / `-du` | **-tzen** | *esnatu* → *esnatzen* |
| `-n` | **-ten** | *jan* → *jaten*; *egin* → *egiten* |
| `-i` (irregular) | **-tzen** o **-ten** | *etorri* → *etortzen*; *ikusi* → *ikusten* |

> El verbo *jan* (comer) **no** se convierte en *janten* sino en *jaten* — la "n" final desaparece. Lo mismo con *edan* → *edaten*, *egon* → *egoten*.

## Verbos del día a día

### NOR (auxiliar **izan**: naiz, zara, da…)

Estos van **con izan** porque son intransitivos — el sujeto experimenta la acción sin un objeto:

| Partizipioa | Aspecto habitual | Significado |
|---|---|---|
| *esnatu* | *esnatzen naiz* | me despierto |
| *jaiki* | *jaikitzen naiz* | me levanto |
| *altxatu* | *altxatzen naiz* | me levanto (variante de *jaiki*) |
| *dutxatu* | *dutxatzen naiz* | me ducho |
| *jantzi* | *janzten naiz* | me visto |
| *joan* | *joaten naiz* | voy |
| *etorri* | *etortzen naiz* | vengo |
| *oheratu* | *oheratzen naiz* | me acuesto |
| *lokartu* | *lokartzen naiz* | me duermo |

### NOR-NORK (auxiliar **ukan**: dut, duzu, du…)

Estos van **con ukan** porque tienen sujeto y objeto:

| Partizipioa | Aspecto habitual | Significado |
|---|---|---|
| *gosaldu* | *gosaltzen dut* | desayuno |
| *bazkaldu* | *bazkaltzen dut* | como (almuerzo) |
| *afaldu* | *afaltzen dut* | ceno |
| *jan* | *jaten dut* | como |
| *edan* | *edaten dut* | bebo |
| *lan egin* | *lan egiten dut* | trabajo |
| *lo egin* | *lo egiten dut* | duermo |
| *ikusi* | *ikusten dut* | veo |
| *irakurri* | *irakurtzen dut* | leo |
| *erosi* | *erosten dut* | compro |

> **¿Por qué algunos van con NAIZ y otros con DUT?** Es la distinción clave del euskera. *Esnatzen naiz* (me despierto) es algo que me pasa a mí: no hay un objeto. *Jaten dut* (como) tiene objeto: *ogia jaten dut* (como pan). Esto se ve a fondo en la próxima unidad — de momento memoriza las parejas verbo + auxiliar.

## Marcadores de frecuencia

| Euskera | Castellano |
|---|---|
| *egunero* | cada día / todos los días |
| *beti* | siempre |
| *askotan* | a menudo |
| *batzuetan* | a veces |
| *normalean* | normalmente |
| *gehienetan* | la mayoría de las veces |
| *gutxitan* | pocas veces |
| *inoiz ez* | nunca |
| *astero* | cada semana |
| *hilero* | cada mes |

## Ejemplos enlazados

> *Egunero zazpietan esnatzen naiz eta dutxatzen naiz.* — Cada día me despierto a las siete y me ducho.
>
> *Goizean kafea hartzen dut.* — Por la mañana tomo café.
>
> *Askotan lagunekin afaltzen dut.* — A menudo ceno con amigos.
>
> *Batzuetan lo-kuluxka egiten dut bazkalostean.* — A veces echo la siesta después de comer.

> **Nota:** *bazkalostean* = "después de comer" (literal: "tras la comida"). La cultura del euskera tiene palabras compactas para conceptos cotidianos como esa siesta corta — *lo-kuluxka* es justo eso.
