El **imperativo** (*agintera*) sirve para dar instrucciones, hacer peticiones y dar consejos. En euskera es **mucho más sencillo de lo que parece**: la forma más usada coincide casi siempre con el **participio** del verbo.

## La regla básica: usa el participio

Para muchas órdenes cotidianas, el verbo aparece en su forma de participio sin auxiliar:

| Castellano | Euskera |
|---|---|
| Toma | *Hartu* |
| Siéntate | *Eseri* |
| Levántate | *Jaiki* |
| Escribe | *Idatzi* |
| Lee | *Irakurri* |
| Escucha | *Entzun* |
| Repite | *Errepikatu* |
| Espera | *Itxoin* |
| Cierra la puerta | *Itxi atea* |
| Abre la ventana | *Ireki leihoa* |

> Esta es la **forma más usada** en aula, en la calle y en instrucciones escritas. Sirve tanto para *zu* como para *zuek*.

## Formas sintéticas (trinkoak): *zatoz*, *zoaz*

Algunos verbos muy frecuentes tienen formas sintéticas específicas para el imperativo. Estas son las que conviene memorizar:

| Verbo | Tú (zu) | Vosotros (zuek) |
|---|---|---|
| *etorri* (venir) | **Zatoz** | **Zatozte** |
| *joan* (ir) | **Zoaz** | **Zoazte** |
| *ekarri* (traer) | **Ekarri** / **Ekartzazu** | **Ekartzazue** |
| *eraman* (llevar) | **Eraman** / **Eramazu** | **Eramazue** |
| *egon* (estar) | **Egon** / **Lasai** (¡tranqui!) | **Egon** |

> *— **Zatoz** hona, mesedez.*
> *— **Zoazte** lehenbailehen.*

## Imperativo NOR-NORI-NORK: *emadazu*, *esadazu*

Cuando das una orden con doble objeto ("dame esto", "dile aquello"), el euskera tiene formas compactas muy útiles:

| Castellano | Euskera |
|---|---|
| Dame | *Emadazu* / *Eman iezadazu* |
| Dime | *Esadazu* / *Esan iezadazu* |
| Dale | *Emaiozu* / *Eman iezaiozu* |
| Dile | *Esaiozu* / *Esan iezaiozu* |
| Danos | *Emaguzu* |
| Dadme | *Emadazue* |

Estas formas son largas de aprender; en A2 basta con **emadazu / esadazu / emaiozu / esaiozu** y entender que vienen del verbo *eman* / *esan*.

## "Por favor" y suavizadores

El imperativo en euskera **no suena brusco** si añades alguna palabra de cortesía:

- *Mesedez* — por favor
- *Ahal duzu(la)?* — ¿puedes?
- *Faborez* (variante coloquial)
- *Eskerrik asko aldez aurretik* — gracias por adelantado

> *Emadazu ogia, **mesedez**.*
> *Itxoin pixka bat, **ahal duzula**.*

## Instrucciones (recetas, anuncios, aula)

El imperativo es la forma estándar en:

- **Recetas:** *Moztu tipula. Bota olioa zartaginera. Frijitu bost minutuz.*
- **Carteles:** *Itxi atea. Ez ukitu. Erabili maskara.*
- **Aula:** *Ireki liburua. Idatzi koadernoan. Errepikatu nirekin.*
- **Médico:** *Hartu pilula bat. Edan ur asko. Hartu atseden.*

## Mini-texto

> *Sukaldari eskolan, irakasleak esaten du:*
>
> *— **Hartu** patata bat. **Garbitu** ondo. **Zuritu** azala. **Moztu** zatitan eta **bota** ur irakinera. **Itxoin** hogei minutu. Eta **probatu** ondoren.*

> **Nota cultural:** en euskera el tono **mandón** del imperativo se atenúa con la entonación y el contexto, no tantas veces con perífrasis ("podrías..."). Acompañar la orden con *mesedez* o con una sonrisa suele ser suficiente. Si quieres algo más cortés, di *eman ahal didazu...?* (¿me puedes dar...?).
