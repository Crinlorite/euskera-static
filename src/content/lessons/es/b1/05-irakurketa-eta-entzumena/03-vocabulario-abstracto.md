## Los grandes sufijos productivos

### -**tasun** : crea cualidades abstractas

Equivale al castellano "-dad / -idad / -tud".

| Adjetivo / nombre | + -tasun |
|---|---|
| zorion (felicidad) | zorion**tasun** (felicidad) |
| zail (difícil) | zail**tasun** (dificultad) |
| erraz (fácil) | erraz**tasun** (facilidad) |
| eder (bonito) | eder**tasun** (belleza) |
| handi (grande) | handi**tasun** (grandeza) |
| gai (apto) | gai**tasun** (capacidad) |
| beharrezko (necesario) | beharrezko**tasun** (necesidad) |

### -**keta** : acción o proceso

| Verbo | + -keta |
|---|---|
| ikasi | ikas**keta** (estudio) |
| aldatu | alda**keta** (cambio) |
| erosi | eros**keta** (compra) |
| aztertu | azter**keta** (examen) |
| ikertu | iker**keta** (investigación) |
| azaldu | azal**keta** (explicación) |

### -**kuntza** / -**pen** : acción más abstracta

| Verbo | Sufijo | Resultado |
|---|---|---|
| irabazi | -pen | irabaz**pen** (ganancia) |
| arrakasta | -tsu | arrakasta**tsu** (exitoso) |
| jakin | -kuntza | jakin**duria** (sabiduría) |
| eraiki | -kuntza | eraiki**kuntza** (construcción) |
| laguntza | --- | lagun**tza** (ayuda) |

### -**garri** : "digno de" o "que provoca"

Crea adjetivos.

| Raíz | + -garri |
|---|---|
| interes | interes**garri** (interesante) |
| harri (asombro) | harri**garri** (asombroso) |
| poz | poz**garri** (alegre) |
| min (dolor) | min**garri** (doloroso) |
| ulertu | uler**garri** (comprensible) |
| irakurri | irakur**garri** (legible) |
| kezka | kezka**garri** (preocupante) |

### -**tegi** : lugar

| Raíz | + -tegi |
|---|---|
| liburu | liburu**tegi** (biblioteca) |
| ardao (vino) | arda**tegi** (bodega) |
| agiri (documento) | agiri**tegi** (archivo) |
| oroi (recuerdo) | oroi**tegi** (memorial) |
| gauza (cosa) | gauz**ategi** (almacén) |

### -**le** / -**gile** : agente

| Verbo | + -le |
|---|---|
| irakurri | irakur**le** (lector) |
| idatzi | idaz**le** (escritor) |
| ikusi | ikus**le** (espectador) |
| egin | egi**le** (autor) |
| sortu | sortzai**le** (creador) |
| ikertu | ikertzai**le** (investigador) |

### -**dun** : "que tiene"

| Nombre | + -dun |
|---|---|
| diru | diru**dun** (rico, con dinero) |
| baso | baso**dun** (con bosque) |
| etxe | etxe**dun** (con casa, propietario) |
| hizkuntza | hizkuntza**dun** (hablante de un idioma) |

> *Euskaldun* = "el que tiene euskera", la auto-denominación de los vascohablantes.

## Familia léxica: ejemplo en cadena

Mira cómo una raíz genera todo un campo léxico:

| Raíz: *iker* (investigar) | Resultado |
|---|---|
| Verbo | iker**tu** (investigar) |
| Sustantivo acción | iker**keta** (investigación) |
| Agente | iker**tzaile** (investigador) |
| Adjetivo | iker**garri** (investigable) |
| Lugar | iker**tegi** (centro de investigación) |
| Cualidad | iker**garritasun** (carácter investigable) |

Sabiendo *iker*, dominas seis palabras a la vez.

## Truco para el examen

Cuando veas una palabra desconocida, descomponla mentalmente:

> *jakingarri*  → *jakin* (saber) + -**garri** (digno de) = digno de saber → "interesante / curioso".
>
> *erantzunbide* → *erantzun* (responder) + -**bide** (camino, vía) = vía de respuesta → "manera de responder, solución".

Con 8 sufijos (-tasun, -keta, -kuntza, -pen, -garri, -tegi, -le, -dun) y unas 50 raíces básicas, lees el 70% del vocabulario abstracto sin diccionario.
