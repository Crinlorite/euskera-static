En B1 trabajas con **textos auténticos** de varios géneros. Reconocer el tipo te ayuda en dos cosas: predecir su estructura y adaptar tu producción cuando te toca redactar uno parecido.

## Los seis tipos textuales más frecuentes

| Tipo | Función | Tono |
|---|---|---|
| **albistea** | informar de un hecho | objetivo, neutro |
| **iritzi-artikulua** | defender una postura | subjetivo |
| **erreportajea** | profundizar en un tema | mixto |
| **gutuna** / **eskutitza** | comunicar a un destinatario | varía según formalidad |
| **txostena** | analizar un tema técnico | formal |
| **iragarkia** / **kartela** | promocionar / informar breve | directo |

## 1. Albistea (noticia)

Estructura típica:

1. **Izenburua** (titular): resume el hecho central.
2. **Sarrera** (entradilla): los 5 W's (zer/nor/non/noiz/zergatik).
3. **Garapena**: detalles, declaraciones, datos.
4. **Amaiera**: contexto o consecuencias.

### Lenguaje típico

- **Atzo gertatu zen…** — ayer ocurrió…
- **Iturri ofizialen arabera…** — según fuentes oficiales…
- **Gaur jakinarazi denez…** — según se ha sabido hoy…
- **X-(e)k jakinarazi du…** — X ha informado…
- **Pasa den astean ezagutu zen…** — la pasada semana se conoció…

### Ejemplo (titular + entradilla, ficticio)

> **Lekittoko hondartzan plastiko-bilketa egin dute boluntarioek**
>
> Joan den larunbatean berrogeita hamar bat lagunek parte hartu zuten Lekittoko Isuntza hondartzako garbiketan. Ekintza udalak antolatu zuen, Itsasargia elkartearekin elkarlanean, eta hiru ordutan 200 kilo plastiko jaso zituzten.

## 2. Iritzi-artikulua (artículo de opinión)

Estructura:

1. **Tesia o postura inicial**.
2. **Argumentos** con ejemplos.
3. **Contraargumentos** y refutación.
4. **Conclusión** que reafirma o llama a la acción.

### Lenguaje típico

- *Nire ustez…*
- *Argi dago…*
- *Egia esan…*
- *Zalantzarik gabe…*
- *Hala ere, batzuek diote…*
- *Laburbilduz, …*

> En B1 esta es **la** tipología que se te puede pedir escribir. Domina la plantilla.

## 3. Erreportajea (reportaje)

Mezcla noticia y artículo. Más extenso, con varias voces, datos y observaciones del autor. Lenguaje narrativo y descriptivo, con alguna entrevista intercalada.

### Marcas frecuentes

- *X-(e)k esan zigun "...".* — X nos dijo "...".
- *Han geunden, eta…* — estábamos allí, y…
- *Atentzio handia eman zigun…* — nos llamó mucho la atención…

## 4. Gutuna / eskutitza (carta)

### Carta informal

Para amigo o familiar.

| Apertura | Cierre |
|---|---|
| *Kaixo, Olatz!* | *Agur, Anboto* |
| *Aupa, Lander!* | *Musu handi bat* |
| *Maite Garazi:* | *Laster arte* |
| *Adiskide:* | *Besarkada bat* |

### Carta formal

Para administración, empresa, autoridad.

| Apertura | Cierre |
|---|---|
| *Agurgarri X jauna,* | *Adeitasunez,* |
| *Andre/Jaun agurgarria,* | *Begirunez,* |
| *X enpresako zuzendari jaunari,* | *Eskerrik asko zure denboragatik,* |

### Estructura básica de carta formal

1. **Lekua eta data**: *Donostian, 2025eko maiatzaren 5ean,*.
2. **Hartzailea**: *Agurgarri X jauna,*.
3. **Sarrera**: *Eskutitz hau idazten dizut …*.
4. **Cuerpo**: motivo, exposición, petición.
5. **Cierre**: *Adeitasunez,*.
6. **Firma**: *Sinatua: Garazi*.

### Lenguaje formal vs informal

| Informal | Formal |
|---|---|
| Kaixo | Agurgarri X-a |
| Egunero idazten dizut | Eskutitz hau idazten dizut … aurkezteko |
| Eskerrik asko! | Eskerrik asko aldez aurretik |
| Laster arte | Adeitasunez |
| Hi, lasai egon | Mesedez, kontuan har dezala |

## 5. Posta elektronikoa (correo electrónico)

Funciona como una carta, pero más breve. Estructura:

1. **Gaia / asuntoa**: claro y específico.
2. **Agurra**: *Kaixo X*, *Agurgarri X-a*…
3. **Cuerpo**: 1-2 párrafos máximo.
4. **Cierre**: *Eskerrik asko, agur, adeitasunez*.

### Fórmulas de email

- *Mezu hau idazten dizut … galdetzeko.*
- *Aurreko mezuan aipatu zenuen bezala, …*
- *Erantzungo didazu mesedez, ahalik eta lasterren?*
- *Edozein zalantzarik baduzu, jakinarazi.*
- *Mila esker aurrez. Adeitasunez,*

## 6. Iragarkia / kartela

Texto breve, directo, con datos clave. Suele aparecer en las pruebas de mediación oral o en preguntas de comprensión rápida.

### Lenguaje típico

- **Noiz?** Maiatzaren 12an, larunbatean, 18:00etan.
- **Non?** Udal Plazan.
- **Zer?** Euskal Dantza Tradizionalen Erakusketa.
- **Sarrera doan / 5 € / izen-emateak…**

> Las cuatro w's básicas: *Zer, Non, Noiz, Nork* y, si procede, *Zenbat balio du* / *Zenbat sarrera*.

## Relación texto / tarea

| Tarea de examen | Tipo textual probable |
|---|---|
| Mediar un anuncio | iragarkia, kartela |
| Resumir información | albistea, txostena |
| Argumentar | iritzi-artikulua |
| Escribir una solicitud | gutun formala, posta elektronikoa |
| Contar una experiencia | gutun informala, blog-sarrera |

Identificar el tipo te dice qué fórmulas y qué tono debes usar. Esa es media batalla del B1 escrito.
