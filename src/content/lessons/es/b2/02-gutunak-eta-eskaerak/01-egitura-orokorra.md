Una **carta formal** (*gutun formala*) sigue siempre la misma estructura. Saberla te permite escribir o leer cualquier carta administrativa, comercial o académica con seguridad.

## Las siete partes de una carta formal

```
1. DATA ETA TOKIA          → Donostian, 2026ko otsailaren 12an
2. IGORLEA                 → tus datos
3. HARTZAILEA              → datos del destinatario
4. GAIA                    → "Asunto: ..."
5. AGURRA                  → "Agurgarria,"
6. GORPUTZA                → cuerpo de la carta
7. AGUR-FORMULA + SINADURA → "Adeitasunez, Garazi Etxeberria"
```

## 1. Data eta tokia

> *Donostian, 2026ko otsailaren 12an*

- *Donostian* — caso inesivo (-an): "en Donostia".
- *2026ko* — adjetivo (-ko): "de 2026".
- *otsailaren 12an* — *-aren* (genitivo del mes) + *12an* (-an inesivo): "el día 12 de febrero".

> En estilo más administrativo: *Donostia, 2026/02/12*.

## 2. Igorlea (remitente)

Va arriba a la izquierda. Si es persona física:

```
Olatz Mendizabal Lasa
Atotxa kalea 12, 2.B
20013 Donostia
olatz.mendizabal@adibidea.eus
```

Si es persona jurídica, primero el nombre de la empresa, luego la dirección.

## 3. Hartzailea (destinatario)

Debajo del remitente, alineado a la derecha o a la izquierda según estilo:

```
Eusko Jaurlaritzaren Hezkuntza Saila
Donostia kalea 1
01010 Vitoria-Gasteiz
```

Puedes también añadir el cargo: *Zuzendari jauna* (Sr. director).

## 4. Gaia (asunto)

Una línea, dos como mucho:

> *Gaia: aparkalekurako baimen-txartelaren eskaera.*

Es lo primero que lee el destinatario para saber de qué va — sé concreto.

## 5. Agurra (saludo)

Las fórmulas más habituales:

| Fórmula | Cuándo se usa |
|---|---|
| *Agurgarria,* | Estándar, neutra |
| *Agurgarri jauna / andrea,* | Cuando conoces el género |
| *Jaun/Andre agurgarriak,* | Plural, varios destinatarios |
| *Zuzendari jauna,* | Si conoces el cargo |
| *Kaixo,* | Solo informal — **NO** en cartas administrativas |

> El saludo va **seguido de coma**, no de dos puntos. La frase siguiente empieza con mayúscula y aparte.

## 6. Gorputza (cuerpo)

Tres bloques internos:

1. **Sarrera** (introducción) — quién soy y por qué escribo.
2. **Garapena** (desarrollo) — los hechos, las razones, la petición o queja.
3. **Itxiera** (cierre del cuerpo) — qué espero de la otra parte.

Cada bloque, su propio párrafo. No mezcles.

## 7. Agur-formula eta sinadura

Las fórmulas finales más frecuentes, **de menos a más formal**:

| Fórmula | Registro |
|---|---|
| *Agurrak,* | Neutra-cordial |
| *Adeitasunez,* | Estándar formal |
| *Begirunez,* | Formal con respeto |
| *Zinez agurtzen zaituen X-(e)k* | Muy formal, antigua |

Después, **firma** (escrita o digital) y **nombre y apellidos** debajo. Si firmas en nombre de una entidad, añade tu cargo:

> *Adeitasunez,*
>
> Olatz Mendizabal Lasa
> Idazkaria

## Una plantilla mínima

```
Donostian, 2026ko martxoaren 8an

[Igorlea: nombre, dirección, contacto]

[Hartzailea: nombre, cargo, dirección]

Gaia: ...

Agurgarria,

[Sarrera: nor naizen eta zergatik idazten dudan]

[Garapena: gertakariak, arrazoiak, eskaera]

[Itxiera: zer espero dudan]

Adeitasunez,

[Sinadura]
[Izen-abizenak]
```

> **Nota:** la mayoría de instituciones públicas vascas tienen formularios oficiales en euskera. Lo que aprendes aquí te sirve también para **rellenar** esos formularios con la fórmula correcta.
