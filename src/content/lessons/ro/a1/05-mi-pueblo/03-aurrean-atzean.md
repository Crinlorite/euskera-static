Odată ce știi să spui „este în X”, pasul următor este să spui **unde exact**: deasupra, dedesubt, alături, în spate… În euskera aceste poziții se construiesc întotdeauna cu tiparul **loc de referință + poziție**.

## Pozițiile de bază

| Euskera | Română |
|---|---|
| *aurrean* | în față |
| *atzean* | în spate |
| *ondoan* | alături |
| *gainean* | deasupra, peste |
| *azpian* | dedesubt |
| *artean* | între |
| *ezkerrean* | la stânga |
| *eskuinean* | la dreapta |
| *barruan* | înăuntru |
| *kanpoan* | afară |
| *erdian* | la mijloc, în centru |

## Cum se construiesc: -(r)en + poziție

Formula de bază este **(referință)-(r)en + poziție**. Referința ia genitivul `-(r)en` (echivalentul lui „al/a” din română):

| Euskera | Română (literal) |
|---|---|
| **mahaiaren gainean** | deasupra mesei (lit. „a mesei, deasupra”) |
| **etxearen aurrean** | în fața casei |
| **elizaren atzean** | în spatele bisericii |
| **parkearen ondoan** | alături de parc |
| **kutxaren barruan** | înăuntrul cutiei |
| **bi mendien artean** | între doi munți |

Tipar: referința vine prima, **apoi** poziția. Este exact invers față de română.

## Exemple complete

- **Liburua mahaiaren gainean dago.** — Cartea este deasupra mesei.
- **Katua aulkiaren azpian dago.** — Pisica este sub scaun.
- **Eliza plazaren erdian dago.** — Biserica este în centrul pieței.
- **Botika eskolaren ondoan dago.** — Farmacia este alături de școală.
- **Ni Maiteren eta Anaren artean nago.** — Eu sunt între Maite și Ana.

## Adverbe de loc: hemen / hor / han

Pentru a indica rapid, există trei adverbe — și, atenție, sunt **trei distanțe**, nu două ca în română:

| Euskera | Română | Când |
|---|---|---|
| *hemen* | aici | aproape de vorbitor |
| *hor* | acolo (la tine) | aproape de ascultător |
| *han* | acolo (departe) | departe de amândoi |

> Aceste trei adverbe corespund demonstrativelor *hau / hori / hura* și formelor lor locative lungi *honetan / horretan / hartan* („în acesta / în acela / în acela de acolo”).

Exemple rapide:

- **Hemen** taberna bat dago. — Aici este un bar.
- **Hor** parkea dago. — Acolo este parcul.
- **Han** dago euskaltegia. — Euskaltegi-ul este acolo.

## Mini-conversație: de-a v-ați ascunselea

> — *Pilota mahaiaren azpian dago?* — Mingea este sub masă?
>
> — *Ez, pilota ez dago mahaiaren azpian.* — Nu, mingea nu este sub masă.
>
> — *Aulkiaren atzean dago?* — Este în spatele scaunului?
>
> — *Bai, hor dago!* — Da, acolo este!

> **Truc:** toate pozițiile au deja sufixul `-an` încorporat (*aurr-ean*, *atz-ean*, *azp-ian*…). Nu trebuie să adaugi nimic în plus.
