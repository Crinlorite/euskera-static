Cunoști deja sufixul **`-(e)an`** din lecția „Unde locuiești” (*Bilbon bizi naiz*). Aici îl aplicăm la cartier: unde se află fiecare lucru.

## Întrebarea: Non dago…?

Pentru a întreba **unde este…?** folosim:

- **Non dago …?** — Unde este + (subiect singular)?
- **Non daude …?** — Unde sunt + (subiect plural)?
- **Non zaude?** — Unde ești (tu)?
- **Non nago?** — Unde sunt eu?

Verbul este **egon** („a fi/a se afla”), pe care l-ai văzut deja la saluturi. Reține formele:

| Persoană | Formă |
|---|---|
| ni | **nago** |
| zu | **zaude** |
| hura | **dago** |
| gu | **gaude** |
| zuek | **zaudete** |
| haiek | **daude** |

## Răspunsul: loc + `-(e)an` + dago

Pentru a spune **unde se află** ceva, adaugă sufixul **`-(e)an`** la loc:

| Terminația locului | Tipar | Exemplu |
|---|---|---|
| -a (organic, ca *Donostia*) | + n | Donostia → **Donostian** |
| Vocală fără -a final (*mendi*, *kale-*) | + an | mendi → **mendian** |
| Consoană (*Madril*, *lur*) | + en (nume proprii) / + ean (nume comune) | Madril → **Madrilen**; lur → **lurrean** |

> Atenție la cuvinte precum *kale* (stradă): când poartă articolul *kalea* („strada”), se elimină `-a` din articol înainte de a adăuga sufixul: *kalea* → *kale-an* = **kalean** („pe stradă”).

## Exemple complete

- **Botika kalean dago.** — Farmacia este pe stradă.
- **Liburuak mahaian daude.** — Cărțile sunt pe masă.
- **Ikaslea ikasgelan dago.** — Elevul este în clasă.
- **Aita tabernan dago.** — Tata este la bar.
- **Ni euskaltegian nago.** — Eu sunt la euskaltegi.
- **Bilbo Bizkaian dago.** — Bilbao se află în Bizkaia.

## Mini-dialog

> — *Aupa! **Non dago** liburutegia?* — Bună! Unde este biblioteca?
>
> — *Liburutegia plazan dago, eskolaren ondoan.* — Biblioteca este în piață, lângă școală.
>
> — *Eskerrik asko!*

> **Truc:** *non* = „unde” în euskera. Orice răspuns la *non…?* conține un substantiv cu sufixul `-(e)an` sau un adverb de loc (*hemen*, *hor*, *han* — le vei vedea în lecția următoare).
