Această lecție integrează tot ce ține de tema 9: camerele, mobilierul, treburile casnice și cu cine locuiești. Scopul este să poți **să-ți descrii casa** în propoziții simple, ca și cum ai fi întrebat la clasă sau într-o conversație informală.

## Întrebările de bază

| Întrebare | Română |
|---|---|
| *Non dago zure etxea?* | Unde este casa ta? |
| *Nolakoa da?* | Cum este? |
| *Zenbat gela ditu?* | Câte camere are? |
| *Norekin bizi zara?* | Cu cine locuiești? |
| *Zenbatgarren solairuan bizi zara?* | La ce etaj locuiești? |
| *Etxeak badu igogailua / balkoia / terraza?* | Casa are lift / balcon / terasă? |

## Adjective utile pentru a descrie casa

| Euskera | Română |
|---|---|
| *handia* | mare |
| *txikia* | mică |
| *argitsua* | luminoasă |
| *iluna* | întunecată |
| *berria* | nouă |
| *zaharra* | veche |
| *lasaia* | liniștită |
| *zaratatsua* | zgomotoasă |
| *erosoa* | confortabilă |
| *garestia* | scumpă |
| *merkea* | ieftină |

> Amintește-ți tiparul **NOLAKOA da?** deja văzut în alte unități: răspunsul include adjectivul + `-a` final + verbul *izan*. Ex.: *Nire etxea **handia da**.* — Casa mea este mare.

## Un paragraf model

Iată un text simplu care descrie o casă. Citește-l cu voce tare și observă tiparele.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## E rândul tău: descrie-ți casa

Construiește-ți propriul paragraf răspunzând în această ordine:

1. **Non bizi zara?** (Unde locuiești?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (Cum este?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (Câte camere?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (Ce se află?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

## Vocabular de încheiere

| Euskera | Română |
|---|---|
| *etxebizitza* | locuință |
| *etxejabea* | proprietar |
| *alokairua* | închiriere |
| *errenta* | chirie / chirie lunară |
| *auzoa* | cartier |
| *erdigunea* | centru (al orașului) |
| *aldiriak* | periferie / suburbii |

> **Sfat:** când vorbești despre casa ta la clasă, nu încerca să traduci fiecare detaliu. Rămâi la propoziții scurte și bine făcute. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* este o descriere mai bună decât o propoziție lungă plină de greșeli.
