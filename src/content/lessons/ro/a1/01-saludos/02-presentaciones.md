Ca să te prezinți în euskera este suficientă o formulă foarte simplă: **Ni** (eu) + numele tău + **naiz** (sunt).

## Tiparul

| Euskera | Română |
|---|---|
| *Ni Egoitz naiz.* | Sunt Egoitz. |
| *Ni Maialen naiz.* | Sunt Maialen. |
| *Ni Hodei naiz.* | Sunt Hodei. |

Verbul **izan** („a fi”) se schimbă în funcție de persoană. Deocamdată memorează doar trei forme:

- *Ni* + **naiz** → eu sunt
- *Zu* + **zara** → tu ești
- *Hura* + **da** → el/ea este

> În euskera nu există gen gramatical la verb: *Hura da* este la fel de valabil pentru „el este” sau „ea este”. Contextul sau numele lămuresc lucrurile.

## „Ce mai faci?”

Odată ce te-ai prezentat, este firesc să întrebi cum se simte cealaltă persoană:

- *Zer moduz?* → Ce mai faci?
- *Ondo, eskerrik asko.* → Bine, mulțumesc.
- *Ni ere ondo.* → Și eu bine.

Cu aceste trei propoziții poți deja purta un salut politicos complet.
