Odată ce știi să prezinți pe cineva (*Hau X da*), următorul pas natural este să **povestești ce face persoana respectivă**: unde locuiește, unde lucrează, câți ani are, cum este viața ei de zi cu zi. Toată conjugarea este la **persoana a treia**, așa că aici exersăm exact acest lucru.

## Întrebările

| Întrebare | Română |
|---|---|
| *Non bizi da?* | Unde locuiește? |
| *Norekin bizi da?* | Cu cine locuiește? |
| *Zenbat urte ditu?* | Câți ani are? |
| *Zer da bera?* / *Zer egiten du?* | Ce este? / Ce face? |
| *Non egiten du lan?* | Unde lucrează? |
| *Ezkonduta dago?* | Este căsătorit/ă? |

## Tipare de răspuns

Combină ceea ce ai învățat. Structura este: **subiect + complemente + verb**.

| Temă | Frază tip |
|---|---|
| Reședință | *Bera **Iruñean bizi da**.* — Locuiește în Pamplona. |
| Companie | *Bera **bikotekidearekin bizi da**.* — Locuiește cu partenerul/partenera. |
| Vârstă | *Bera**k hogeita hamar urte ditu**.* — Are 30 de ani. |
| Profesie | *Bera **medikua da**.* — Este medic. |
| Muncă | *Bera**k ospitalean lan egiten du**.* — Lucrează la spital. |

## Locativul `-(e)an` pentru locul de muncă

Sufixul **`-(e)an`** („în/la”) îl cunoști deja din unitatea despre cartier și despre reședință. Aici îl folosești pentru a spune unde lucrează cineva.

| Loc | Frază |
|---|---|
| ospitala (spital) | *ospitale**an** lan egiten du* |
| eskola (școală) | *eskola**n** lan egiten du* |
| taberna (bar) | *taberna**n** lan egiten du* |
| bulegoa (birou) | *bulego**an** lan egiten du* |
| denda (magazin) | *denda**n** lan egiten du* |
| euskaltegia (academie de euskera) | *euskaltegi**an** lan egiten du* |
| etxean (acasă) | *etxean lan egiten du* |

> Reține: dacă cuvântul se termină în `-a` organic (tabern**a**), se păstrează → *tabernan*. Dacă se termină în consoană (Madril), se intercalează `-e-` → *Madrilen*.

## Profesie + loc = portret complet

Combinarea profesiei cu locul este ceea ce dă coerență prezentării unui vecin:

> *Bera **erizaina** da, eta **ospitalean lan egiten du**. **Iruñean bizi da, bikotekidearekin**, eta **bi seme-alaba ditu**.*

> *Hau Maialen da. Bera **kazetaria** da, **egunkari batean lan egiten du**. **Bilbokoa da, baina orain Donostian bizi da**.*

## Starea civilă (vocabular util)

| Euskera | Română |
|---|---|
| *ezkonduta dago* | este căsătorit/ă |
| *ezkongabea da* | este necăsătorit/ă |
| *banatuta dago* | este separat/ă |
| *dibortziatuta dago* | este divorțat/ă |
| *alargun da* | este văduv/ă |

## O rutină simplă

Pentru a descrie ce face cineva în mod obișnuit, recapitulează tiparul **`-tzen` + du** pe care l-ai văzut deja la treburile casnice:

- *Goizean **lana hasten du** zortzietan.* — Începe lucrul la opt.
- *Eguerdian **etxean bazkaltzen du**.* — La prânz mănâncă acasă.
- *Arratsaldean **kirola egiten du**.* — După-amiaza face sport.

> Pentru A1 îți sunt suficiente fraze izolate; nu este nevoie să înlănțui toată ziua. Dacă poți spune trei lucruri clare despre un vecin, deja îl prezinți bine.
