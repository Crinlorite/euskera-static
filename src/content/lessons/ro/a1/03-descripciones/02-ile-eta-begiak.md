Pentru a descrie părul și ochii în euskera nu se folosește **izan** ("a fi") ca în spaniolă ("sunt brunet"), ci **ukan** ("a avea"): *am părul negru*, *am ochii albaștri*.

## Părul: ile

Cuvântul pentru "păr" este **ile** (cu articol: *ilea*). Adjectivele vin după el, și doar ultimul poartă marca `-a`:

| Euskera | Română |
|---|---|
| *ile beltza* | păr negru |
| *ile horia* | păr blond |
| *ile zuria* | păr alb / cărunt |
| *ile gorria* | păr roșcat |
| *ile urdina* | păr gri |
| *ile motza* | păr scurt |
| *ile luzea* | păr lung |
| *ile lisoa* | păr drept |
| *ile kizkurra* | păr creț |

> Părul în euskera este tratat ca **singular**, la fel ca în spaniolă. De aceea se folosește *dut* / *du* (nu *ditut* / *ditu*).

- *Ile beltza eta luzea **dut**.* — Am părul negru și lung.
- *Maialenek ile kizkurra **du**.* — Maialen are părul creț.

## Ochii: begiak

Spre deosebire de păr, "ochii" este la **plural** (*begiak*). Prin urmare adjectivele sunt și ele la plural (`-ak`) și se folosește **ditut / ditu**:

| Euskera | Română |
|---|---|
| *begi urdinak* | ochi albaștri |
| *begi marroiak* | ochi căprui |
| *begi berdeak* | ochi verzi |
| *begi beltzak* | ochi negri |
| *begi argiak* | ochi deschiși la culoare |
| *begi ilunak* | ochi închiși la culoare |

- *Begi urdinak **ditut**.* — Am ochii albaștri.
- *Mikelek begi marroiak **ditu**.* — Mikel are ochii căprui.

## Alte trăsături: sufixul -dun

Pentru a spune "care poartă X" sau "cu X", euskera are un sufix foarte productiv: **-duna** (singular) / **-dunak** (plural). Se adaugă la substantiv, iar persoana devine "cel cu ochelari", "cel cu barbă"...

| Euskera | Română |
|---|---|
| *betaurrekoduna* | cu ochelari / cel cu ochelari |
| *bizarduna* | cu barbă |
| *biboteduna* | cu mustață |
| *burusoila* | chel (literal "cap pleșuv") |

Acestea da se folosesc cu **izan**, pentru că descriu o calitate ca un adjectiv:

- *Hura biboteduna da.* — El poartă mustață.
- *Nire aitona burusoila da.* — Bunicul meu este chel.

## Combinând totul

Un portret tipic îmbină mai multe structuri:

> *Maialen txikia eta argala da. Ile gorria eta kizkurra du. Begi berdeak ditu. Betaurrekoduna da.*
>
> Maialen este scundă și slabă. Are părul roșcat și creț. Are ochii verzi. Poartă ochelari.

Observă cum se distribuie: **izan** pentru calități generale (*txikia da*), **ukan** pentru păr/ochi (*ile … du*, *begi … ditu*), și *-dun*-a din nou cu **izan** pentru accesorii.
