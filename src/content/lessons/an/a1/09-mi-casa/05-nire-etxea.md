Esta lección integra tot lo d'o tema 9: as cambras, os mobles, as fainas y con quí bibes. L'obchectibo ye que puedas **describir a tuya casa** en frases sencillas, como si te'n estasen preguntando en clase u en una charrada informal.

## As preguntas basicas

| Pregunta | Aragonés |
|---|---|
| *Non dago zure etxea?* | ¿Án ye a tuya casa? |
| *Nolakoa da?* | ¿Cómo ye? |
| *Zenbat gela ditu?* | ¿Quántas cambras tien? |
| *Norekin bizi zara?* | ¿Con quí bibes? |
| *Zenbatgarren solairuan bizi zara?* | ¿En qué planta bibes? |
| *Etxeak badu igogailua / balkoia / terraza?* | ¿A casa tien ascensor / balcón / terraza? |

## Achectibos utils ta describir a casa

| Euskera | Aragonés |
|---|---|
| *handia* | gran |
| *txikia* | chicota |
| *argitsua* | luminosa |
| *iluna* | foscura |
| *berria* | nueba |
| *zaharra* | biella |
| *lasaia* | tranquila |
| *zaratatsua* | estridén |
| *erosoa* | comoda |
| *garestia* | cara |
| *merkea* | barata |

> Remera lo patrón **NOLAKOA da?** ya bisto en atras unidaz: a respuesta leba l'achectibo + `-a` final + berbo *izan*. Ex.: *Nire etxea **handia da**.* — A mía casa ye gran.

## Un paragrafo modelo

Aquí tiens un texto sencillo describindo una casa. Leye-lo en boz alta y fixa-te en os patrons.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## A tuya quenda: describe a tuya casa

Construye o tuyo propio paragrafo respondendo en iste orden:

1. **Non bizi zara?** (¿Án bibes?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (¿Cómo ye?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (¿Quántas cambras?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (¿Qué bi ha?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

| Euskera | Aragonés |
|---|---|
| *etxebizitza* | bibienda |
| *etxejabea* | propietario |
| *alokairua* | alquilé |
| *errenta* | renta / alquilé mensual |
| *auzoa* | bico |
| *erdigunea* | centro (d'a ciudat) |
| *aldiriak* | afueras / arredols |

> **Consello:** quan charres d'a tuya casa en clase, no procures traducir cada detalle. Queda-te con frases curtas y bien feitas. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* ye millor descripción que una frase larga plena de fallos.
