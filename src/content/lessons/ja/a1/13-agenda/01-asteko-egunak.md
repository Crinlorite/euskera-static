予定やスケジュールについて話す前に、曜日を復習し、それらを日常の活動と結びつける方法を学びましょう。

## 7つの曜日

| バスク語 | 日本語 | ヒント |
|---|---|---|
| *astelehena* | 月曜日 | aste + lehen =「週の最初の日」 |
| *asteartea* | 火曜日 | aste + arte =「週の間」 |
| *asteazkena* | 水曜日 | aste + azken =「労働週の最後の日」（歴史的に） |
| *osteguna* | 木曜日 | egun = 日 |
| *ostirala* | 金曜日 | *osteguna* と語根 *ortzi/osti-*（空）を共有する |
| *larunbata* | 土曜日 | larun + bata |
| *igandea* | 日曜日 | 歴史的な祝日 |

> **記憶のヒント：** **aste**（週）という語は平日の始まりに現れます。*Astelehen-, astear-, asteazken-* はほとんど声に出して、週の始め・半ば・終わりを数えているようなものです。

## 「月曜日に」と言う — 接尾辞 `-(e)an`

出来事に日付を付けるには、曜日に `-(e)an` を加えます：

| 曜日 | 「～に」 |
|---|---|
| astelehena | astelehen**ean** |
| asteartea | astearte**an** |
| asteazkena | asteazken**ean** |
| osteguna | ostegun**ean** |
| ostirala | ostirale**an** |
| larunbata | larunbat**ean** |
| igandea | igande**an** |

- *Astelehenean lan egiten dut.* — 月曜日に働きます。
- *Larunbatean afaltzera joango gara.* — 土曜日に夕食を食べに行きます。

## 「毎～」と言う — 接尾辞 `-ero`

規則的な頻度を表すには：

- *astelehenero* — 毎週月曜日
- *larunbatero* — 毎週土曜日
- *egunero* — 毎日
- *astero* — 毎週
- *astebururo* — 毎週末

## 曜日＋活動：例

| バスク語 | 日本語 |
|---|---|
| *Astelehenean kiroldegira noa.* | 月曜日にジムへ行きます。 |
| *Asteartean euskara klasea dut.* | 火曜日にバスク語の授業があります。 |
| *Ostiralean parranda egiten dugu.* | 金曜日に飲みに繰り出します。 |
| *Larunbatean familiaren etxera joango naiz.* | 土曜日に家族の家へ行きます。 |
| *Igandean lasai gelditzen naiz etxean.* | 日曜日は家でゆっくり過ごします。 |

## Asteburua — 週末

`Asteburua`（週末）はとても便利な語です：

- *Asteburuan zer egingo duzu?* — 週末は何をしますか？
- *Asteburu honetan mendira joango naiz.* — 今週末は山へ行きます。
- *Asteburu pasa egitea gustatzen zait.* — 週末の小旅行をするのが好きです。

> **文化メモ：** *igandea* と並んで *jaieguna*（祝日）をよく目にします。*igandeak eta jaiegunak* という表現は、バスの時刻表、商店、市役所の掲示でよく使われます。
