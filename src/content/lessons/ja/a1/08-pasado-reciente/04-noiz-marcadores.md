起きたことをきちんと話すには、**いつ**かを言う必要があります。このレッスンでは、近い過去と組み合わせて動作を時間の中に位置づけるための時を表す言葉を紹介します。

## 基本となる時の言葉

| エウスケラ | 日本語 |
|---|---|
| *gaur* | 今日 |
| *atzo* | 昨日 |
| *bihar* | 明日 |
| *etzi* | あさって |
| *orain* | 今 |
| *oraintxe* | たった今 |
| *lehen* | 以前 |
| *gero* | あとで、その後 |

> 気づいたかもしれませんが、*atzo*（昨日）と *bihar*（明日）はとても短い二語なので、一気に覚えられます。*Etzi*（あさって）は、スペイン語には一語で存在しない言葉です。エウスケラからのちょっとした贈り物ですね。

## 一日の中の時間帯と組み合わせる

これらの言葉は、前のユニットで学んだ *goizean*、*arratsaldean*、*gauean* などと組み合わせます。公式は：**時を表す言葉 + 一日の時間帯**。

| エウスケラ | 日本語 |
|---|---|
| *gaur goizean* | 今朝／今日の朝 |
| *gaur eguerdian* | 今日の正午 |
| *gaur arratsaldean* | 今日の午後 |
| *gaur gauean* | 今夜 |
| *atzo goizean* | 昨日の朝 |
| *atzo arratsaldean* | 昨日の午後 |
| *atzo gauean* | 昨夜 |
| *bihar goizean* | 明日の朝 |

> **注意：** *gaur gauean* は「今夜」（これから来る夜、または今過ごしている夜）、*atzo gauean* は「昨夜」です。エウスケラはこれをとてもはっきり区別します。

## 頻度と回数

| エウスケラ | 日本語 |
|---|---|
| *behin* | 一度 |
| *bi aldiz* | 二度 |
| *hiru aldiz* | 三度 |
| *askotan* | よく、何度も |
| *batzuetan* | ときどき |
| *inoiz ez* | 決して〜ない |
| *gaur goizetik* | 今朝から |
| *aste honetan* | 今週 |
| *hilabete honetan* | 今月 |

## 近い過去と時を表す言葉を使った文

> *Gaur goizean lanera joan naiz autoz.* — 今朝、車で仕事に行った。
>
> *Gaur arratsaldean Mikelekin egon naiz.* — 今日の午後、Mikel と一緒にいた。
>
> *Gaur oso berandu esnatu naiz.* — 今日はとても遅く起きた。
>
> *Gaur eguerdian etxean bazkaldu dut.* — 今日の正午、家で昼食をとった。
>
> *Aste honetan bi aldiz egon naiz zinean.* — 今週は二度映画館に行った。

## 文の語順

エウスケラでは、時を表す言葉を含む文の典型的な語順はこうです：

> **いつ + どこで + 何を + 動詞**

> **Gaur goizean** + **euskaltegian** + **euskara** + **ikasi dut**. — 今朝、エウスカルテギ（語学学校）でエウスケラを勉強した。

これが唯一の可能性ではありません。エウスケラは強調のために要素を移動できます。でもこの語順に従えば間違えません。

## ミニ会話

> *— Eta gaur zer egin duzu, Maialen?*
>
> *— **Gaur goizean** lanera joan naiz oinez. **Eguerdian** lagunekin bazkaldu dut tabernan. **Arratsaldean** etxera bueltatu naiz eta liburua irakurri dut. **Gauean** ez dut afaldu, asko jan baitut bazkarian.*

> **メモ：** *gaur* は今日一日全体を含みます。もっと正確に言いたいときは、時間帯を加えます（*gaur goizean*、*gaur arratsaldean*）。普通の会話では、*Zer egin duzu gaur?* は「今日は全体として何をした？」という意味で理解されます。
