エウスカラ（バスク語）では、最近したこと — とくに今日のこと — を語るには **orainaldi burutua**（文字どおり「現在完了」）を使います。これは日本語の「食べました／行きました」に当たります。これがいわゆる **近い過去** です。

## 公式はとても簡単

> **Partizipioa ＋ 助動詞（izan または ukan）**

前のユニットで見た習慣相と比べると、変わるのは動詞が **-tzen / -ten なし** になることだけです：

| 習慣 | 近い過去 | 日本語 |
|---|---|---|
| *gosaltzen dut* | **gosaldu dut** | 朝食をとる → 朝食をとった |
| *jaten dut* | **jan dut** | 食べる → 食べた |
| *joaten naiz* | **joan naiz** | 行く → 行った |
| *etortzen naiz* | **etorri naiz** | 来る → 来た |
| *ikusten dut* | **ikusi dut** | 見る → 見た |

> **これは日本語の** 習慣的な現在と完了相の違いと同じです：
>
> - *Egunero euskaltegira **joaten naiz**.* — 毎日 euskaltegi に行きます。
> - *Gaur goizean euskaltegira **joan naiz**.* — 今朝 euskaltegi に行きました。

## いちばん自然な質問

> *— **Zer egin duzu gaur?*** — 今日は何をしましたか？
>
> *— Lan egin dut, eta gero zinera joan naiz.* — 働いて、そのあと映画館に行きました。

2つの形に注目してください：**lan egin dut**（エウスカラでは「働く」が目的語をとるので *dut*）と **joan naiz**（「行く」は目的語をとらないので *naiz*）です。

## 2つの助動詞 — さっとおさらい

### izan を使う（NOR）— naiz / zara / da / gara / zarete / dira

| 代名詞 | 助動詞 | 例 |
|---|---|---|
| ni（私） | naiz | *joan **naiz*** （行きました） |
| zu（あなた） | zara | *etorri **zara*** （来ました） |
| hura（彼／彼女） | da | *esnatu **da*** （目が覚めました） |
| gu（私たち） | gara | *joan **gara*** （行きました） |
| zuek（あなたたち） | zarete | *etorri **zarete*** （来ました） |
| haiek（彼ら） | dira | *joan **dira*** （行きました） |

### ukan を使う（NOR-NORK）— dut / duzu / du / dugu / duzue / dute

| 代名詞 | 助動詞 | 例 |
|---|---|---|
| nik（私） | dut | *jan **dut*** （食べました） |
| zuk（あなた） | duzu | *ikusi **duzu*** （見ました） |
| hark（彼／彼女） | du | *erosi **du*** （買いました） |
| guk（私たち） | dugu | *egin **dugu*** （しました） |
| zuek（あなたたち） | duzue | *jan **duzue*** （食べました） |
| haiek（彼ら） | dute | *ikusi **dute*** （見ました） |

## ミニ会話

> *— Kaixo Maialen! Zer moduz?*
>
> *— Ondo. Eta zu? **Zer egin duzu gaur?***
>
> *— Lan egin dut goizean, eta arratsaldean lagun batekin kafea hartu dut.*
>
> *— Polita. Ni gaur ez naiz lanera joan, gaixo egon naiz.*

> **まとめ：** コツはすべて、どの動詞がどの助動詞を使うかを知ること（次の2つのレッスンで見ていきます）と、partizipio を習慣の標識 *-tzen/-ten* なしのままにしておくことにあります。
