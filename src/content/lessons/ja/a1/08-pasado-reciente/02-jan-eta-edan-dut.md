**他動詞**とは目的語を伴う動詞です。誰かが*何か*を食べる、*何か*を見る、*何か*を読む、といった動詞です。バスク語ではこれらの動詞は助動詞 **ukan** (dut, duzu, du…) を使います。近い過去では、形は次のようになります:

> **partizipioa + dut/duzu/du/dugu/duzue/dute**

## 基本的な他動詞

| Partizipioa | 私 (…した) | 日本語 |
|---|---|---|
| *jan* | **jan dut** | 食べた |
| *edan* | **edan dut** | 飲んだ |
| *ikusi* | **ikusi dut** | 見た |
| *irakurri* | **irakurri dut** | 読んだ |
| *entzun* | **entzun dut** | 聞いた |
| *erosi* | **erosi dut** | 買った |
| *egin* | **egin dut** | した |
| *hartu* | **hartu dut** | 手に取った / とった |
| *ekarri* | **ekarri dut** | 持ってきた |
| *eraman* | **eraman dut** | 持っていった |
| *ikasi* | **ikasi dut** | 学んだ / 勉強した |
| *eskatu* | **eskatu dut** | 頼んだ |
| *ordaindu* | **ordaindu dut** | 支払った |
| *idatzi* | **idatzi dut** | 書いた |
| *prestatu* | **prestatu dut** | 準備した |
| *garbitu* | **garbitu dut** | 掃除した |
| *gosaldu* | **gosaldu dut** | 朝食をとった |
| *bazkaldu* | **bazkaldu dut** | 昼食をとった |
| *afaldu* | **afaldu dut** | 夕食をとった |
| *lan egin* | **lan egin dut** | 働いた |

## ひとつの動詞による完全な活用

例として *ikusi* (見る) をとりあげます:

| 代名詞 + 主語 | 形 | 日本語 |
|---|---|---|
| Nik | **ikusi dut** | 私は見た |
| Zuk | **ikusi duzu** | あなたは見た |
| Hark | **ikusi du** | 彼/彼女は見た |
| Guk | **ikusi dugu** | 私たちは見た |
| Zuek | **ikusi duzue** | あなたたちは見た |
| Haiek | **ikusi dute** | 彼らは見た |

> 他動詞の主語は **NORK** (-k) の標識を付けます: *nik, zuk, hark, guk, zuek, haiek*。とりわけ主語が名前のときによく見かけます: *Mikelek* (Mikel は…した)、*Maialenek* (Maialen は…した)。混乱する場合は、A1 では代名詞を省いて動詞だけを残しても構いません。文脈で十分に通じます。

## よく使う文

- *Gaur goizean egunkaria irakurri dut.* — 今朝、新聞を読んだ。
- *Bazkaltzeko arraina jan dut.* — 昼食に魚を食べた。
- *Lagunekin kafea hartu dut.* — 友達とコーヒーを飲んだ。
- *Mikelek ardoa erosi du.* — Mikel はワインを買った。
- *Guk pelikula bat ikusi dugu.* — 私たちは映画を一本見た。
- *Haiek euskara ikasi dute gaur.* — 彼らは今日バスク語を勉強した。

## 否定

「Xをしなかった」と言うには、**ez** を助動詞の前に置きます:

> *Ez dut bazkaldu.* — 昼食をとらなかった。
>
> *Ez du egunkaria irakurri.* — 新聞を読まなかった。
>
> *Ez dugu ardorik edan.* — ワインを飲まなかった。 (*ardorik* に注目: 否定では目的語にはたいてい部分格の **-rik** が付きます。)

## ミニ対話

> *— Zer afaldu duzu gaur, Maialen?*
>
> *— Entsalada eta arraina jan dut, eta postrerako jogurta hartu dut. Eta zuk?*
>
> *— Nik tortilla egin dut, baina ez dut postrerik jan.*

> **視覚的なまとめ:** 動作に明示的または暗黙の目的語があるなら (何かを食べる、何かを見る、何かをする)、助動詞は **dut** (とその派生形) です。これらの動詞と助動詞のペアを覚えれば、あとは自然に出てきます。
