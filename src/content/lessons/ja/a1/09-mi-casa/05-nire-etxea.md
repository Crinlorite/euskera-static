このレッスンでは、第9課のすべて——部屋、家具、家事、そして誰と住んでいるか——を統合します。目標は、授業中やくだけた会話で尋ねられたかのように、簡単な文で**自分の家を描写できる**ようになることです。

## 基本的な質問

| 質問 | 日本語 |
|---|---|
| *Non dago zure etxea?* | あなたの家はどこにありますか? |
| *Nolakoa da?* | どんな感じですか? |
| *Zenbat gela ditu?* | 部屋はいくつありますか? |
| *Norekin bizi zara?* | 誰と一緒に住んでいますか? |
| *Zenbatgarren solairuan bizi zara?* | 何階に住んでいますか? |
| *Etxeak badu igogailua / balkoia / terraza?* | 家にはエレベーター/バルコニー/テラスがありますか? |

## 家を描写するのに役立つ形容詞

| Euskera | 日本語 |
|---|---|
| *handia* | 大きい |
| *txikia* | 小さい |
| *argitsua* | 明るい |
| *iluna* | 暗い |
| *berria* | 新しい |
| *zaharra* | 古い |
| *lasaia* | 静かな |
| *zaratatsua* | うるさい |
| *erosoa* | 快適な |
| *garestia* | 高い(値段が) |
| *merkea* | 安い |

> 他のユニットですでに見た **NOLAKOA da?** のパターンを思い出してください。答えには形容詞 + 語末の `-a` + 動詞 *izan* が付きます。例: *Nire etxea **handia da**.* — 私の家は大きい。

## モデル段落

ここに、ある家を描写する簡単な文章があります。声に出して読み、パターンに注目しましょう。

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## あなたの番:自分の家を描写しよう

次の順序で答えながら、自分自身の段落を組み立ててみましょう。

1. **Non bizi zara?** (どこに住んでいますか?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (どんな感じですか?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (部屋はいくつありますか?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (何がありますか?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

## 締めくくりの語彙

| Euskera | 日本語 |
|---|---|
| *etxebizitza* | 住居 |
| *etxejabea* | 家主 |
| *alokairua* | 賃貸 |
| *errenta* | 家賃/月々の賃料 |
| *auzoa* | 地区 |
| *erdigunea* | (街の)中心部 |
| *aldiriak* | 郊外/町外れ |

> **アドバイス:** 授業で自分の家について話すときは、細部を一つ一つ訳そうとしないでください。短くてきちんとした文にとどめましょう。*"Bi logela ditut, sukalde txiki bat eta egongela handia"* は、間違いだらけの長い文よりも良い描写です。
