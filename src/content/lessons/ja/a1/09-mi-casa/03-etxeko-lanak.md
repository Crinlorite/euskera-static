このレッスンでは、**etxeko lanak**（家事）について話すための基本的な動詞を学びます。バスク語では、動詞はふつう**分詞**（`-tu`、`-i`、`-n`）の形で示され、これが「辞書形」にあたります：*garbitu*、*prestatu*、*jaso*…

## 主要な動詞

| Euskera | 日本語 |
|---|---|
| *garbitu* | 掃除する |
| *prestatu* | 準備する |
| *jaso* | 片付ける |
| *jantzi* | 着せる／整える |
| *atera* | 出す／取り出す |
| *zabaldu* | 干す／開く |
| *lisatu* | アイロンをかける |
| *paseatu* | 散歩させる |
| *erosi* | 買う |

## 典型的な家事

| 家事 | 言い方 |
|---|---|
| ベッドを整える | *ohea jantzi* |
| 皿を洗う | *ontziak garbitu* / *harrikoa egin* |
| 掃除機をかける | *xurgagailua pasa* |
| 洗濯物を干す | *arropa zabaldu* |
| 服にアイロンをかける | *arropa lisatu* |
| トイレを掃除する | *komuna garbitu* |
| ゴミを出す | *zaborra atera* |
| 食事を作る | *bazkaria prestatu* |
| 買い物をする | *erosketak egin* |
| 犬を散歩させる | *txakurra paseatu* |

## 辞書形から習慣的現在へ

動作が習慣的に繰り返されるとき（毎日、毎週など）、動詞は `-tzen` ＋ 助動詞 **dut/duzu/du…**（*ukan* ですでに知っている形）に変わります：

| 人称 | 例 | 日本語 |
|---|---|---|
| Nik | *Nik etxea **garbitzen dut**.* | 私は家を掃除する。 |
| Zuk | *Zuk arropa **lisatzen duzu**.* | あなたは服にアイロンをかける。 |
| Berak | *Aitak bazkaria **prestatzen du**.* | 父が食事を作る。 |
| Guk | *Guk erosketak **egiten ditugu**.* | 私たちは買い物をする。 |

> このレッスンでは活用を完璧に身につける必要はありません。日課について話すための**`-tzen` ＋ dut/du** というパターンを覚えておきましょう。

## 頻度

ある家事をどのくらいの頻度でするかを言うには：

- **egunero** — 毎日
- **astean behin** — 週に一度
- **astean bitan / hirutan** — 週に二度／三度
- **noizean behin** — ときどき
- **inoiz ez** — けっして〜ない

> *Nire etxean nik **astean behin** lisatzen dut, baina **egunero** garbitzen dut sukaldea.* — うちでは私は週に一度アイロンをかけますが、台所は毎日掃除します。

## 誰が何をするかをたずねる

- **Nork garbitzen ditu ontziak?** — 誰が皿を洗うの？
- **Nor arduratzen da arropaz?** — 誰が洗濯物を担当しているの？
- **Zuri gustatzen zaizu sukaldatzea?** — 料理は好き？

> **文化メモ：** 家庭での家事分担に関するアンケートは euskaltegi（バスク語学校）の教材の定番です——それには理由があります。友人や家族との実際の会話でとても役立つ語彙だからです。
