このレッスンでは、このユニットで学んだすべて——時刻、一日の時間帯、日課の動詞、頻度——をまとめます。目標は、自分の一日を最初から最後までエウスケラで話せるようになることです。

## 鍵となる質問："Zer ordutan…?"

| 質問 | 日本語 |
|---|---|
| *Zer ordutan esnatzen zara?* | 何時に起きますか？ |
| *Zer ordutan jaikitzen zara?* | 何時に起き上がりますか？ |
| *Zer ordutan gosaltzen duzu?* | 何時に朝食を食べますか？ |
| *Zer ordutan bazkaltzen duzu?* | 何時に昼食を食べますか？ |
| *Zer ordutan ateratzen zara etxetik?* | 何時に家を出ますか？ |
| *Zer ordutan afaltzen duzu?* | 何時に夕食を食べますか？ |
| *Zer ordutan oheratzen zara?* | 何時に寝ますか？ |

> 助動詞の変化に注意：*esnatzen **zara*** （NOR）ですが、*gosaltzen **duzu*** （NOR-NORK）です。動詞＋助動詞のペアを少しずつ意識していきましょう。

## 「○時に」——どう言うか

「○時に」と答えるには、複数形の時刻を取って、*-ak* を **-etan** に変えます（すでに母音で終わるもの以外は `-e-` を挟みます）：

| 時刻 | 「○時に…」 |
|---|---|
| zazpiak（7時） | **zazpietan** |
| zortziak | **zortzietan** |
| hamarrak | **hamarretan** |
| hamabiak | **hamabietan** |
| ordu bata（1時） | **ordu batean**（単数、その形を保つ） |
| ordu biak | **ordu bietan** |
| bost eta erdiak | **bost eta erdietan**（5時半に） |
| zortziak laurden gutxi | **zortziak laurden gutxitan**（8時15分前に） |

## まる一日——モデル

この例文を読んで、すべての要素がどのようにつながるかに注目してください：

> *Ni goizeko zazpietan esnatzen naiz. Poliki-poliki altxatzen naiz eta komunera joaten naiz. Dutxa bat hartzen dut, arropa janzten dut eta sukaldera joaten naiz. Sukaldean gosaria prestatzen dut eta lasai-lasai gosaltzen dut. Ondoren, etxetik ateratzen naiz eta lanera joaten naiz. Eguerdiko ordu bietan bazkaltzen dut. Arratsaldean batzuetan zinemara joaten naiz, beste batzuetan lagunekin egoten naiz. Gauean, etxean afaltzen dut, telebista ikusten dut eta hamaiketan oheratzen naiz.*

おおまかな訳：
> *わたしは朝7時に起きます。ゆっくりと起き上がり、トイレに行きます。シャワーを浴び、服を着て、台所に行きます。台所で朝食を用意し、ゆったりと朝食を食べます。その後、家を出て仕事に行きます。昼の2時に昼食を食べます。午後は、ときどき映画館に行き、ときどき友達と過ごします。夜は、家で夕食を食べ、テレビを見て、11時に寝ます。*

## 自分のものを組み立てよう

自分の実際の情報を使って、このテンプレートに沿ってみましょう。学んだことを定着させる一番の方法です：

1. **何時に起きるか：** *Goizeko ___etan esnatzen naiz.*
2. **起きてから何をするか：** *... dutxatzen naiz / gosaltzen dut / ...*
3. **仕事や学校へどう行くか：** *... oinez / autoz / autobusez joaten naiz.*
4. **午後に何をするか：** *Arratsaldean ...*
5. **何時に夕食を食べ、何時に寝るか：** *Gaueko ___etan afaltzen dut eta ___etan oheratzen naiz.*

## つなぐのに役立つ表現

- *Lehenengo* — 最初に
- *Gero / Ondoren* — それから、その後
- *Eta gero* — そして次に
- *Azkenik* — 最後に
- *Egunero* — 毎日
- *Batzuetan / Beste batzuetan* — ときどき／別のときには
- *Beti* — いつも
- *Inoiz ez* — 決して〜ない

> **文化メモ：** Euskal Herria では多くの人が14:00から15:00の間に昼食をとり（*bazkaldu*）、かなり遅く、21:00〜22:00ごろに夕食をとります（*afaldu*）。*Hamaiketakoa*——朝11時の軽食で、たいていピンチョやサンドイッチ——は一種の習慣です。仲間うち（cuadrilla）にいて11時に誰かが *hamaiketakoa egitera goaz* と言ったら、何をするときなのかもうわかりますね。
