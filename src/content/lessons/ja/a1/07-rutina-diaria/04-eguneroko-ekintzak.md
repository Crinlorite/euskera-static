毎日行っていることを言うとき、euskera は**習慣相**を使います。動詞（partizipio）を取り、それに**-tzen** または **-ten** を加え、それから助動詞 **izan**（naiz, zara, da…）または **ukan**（dut, duzu, du…）と組み合わせます。

## 規則

| 動詞の語末 | 習慣形の接尾辞 | 例 |
|---|---|---|
| 子音または `-tu` / `-du` | **-tzen** | *esnatu* → *esnatzen* |
| `-n` | **-ten** | *jan* → *jaten*；*egin* → *egiten* |
| `-i`（不規則） | **-tzen** または **-ten** | *etorri* → *etortzen*；*ikusi* → *ikusten* |

> 動詞 *jan*（食べる）は *janten* には**ならず** *jaten* になります — 語末の「n」が消えます。*edan* → *edaten*、*egon* → *egoten* も同じです。

## 日常の動詞

### NOR（助動詞 **izan**：naiz, zara, da…）

これらは自動詞なので **izan と一緒に**使います — 主語が目的語なしで動作を経験します。

| Partizipioa | 習慣相 | 意味 |
|---|---|---|
| *esnatu* | *esnatzen naiz* | 目が覚める |
| *jaiki* | *jaikitzen naiz* | 起き上がる |
| *altxatu* | *altxatzen naiz* | 起き上がる（*jaiki* の変種） |
| *dutxatu* | *dutxatzen naiz* | シャワーを浴びる |
| *jantzi* | *janzten naiz* | 服を着る |
| *joan* | *joaten naiz* | 行く |
| *etorri* | *etortzen naiz* | 来る |
| *oheratu* | *oheratzen naiz* | 寝床につく |
| *lokartu* | *lokartzen naiz* | 眠りにつく |

### NOR-NORK（助動詞 **ukan**：dut, duzu, du…）

これらは主語と目的語を持つので **ukan と一緒に**使います。

| Partizipioa | 習慣相 | 意味 |
|---|---|---|
| *gosaldu* | *gosaltzen dut* | 朝食を食べる |
| *bazkaldu* | *bazkaltzen dut* | 昼食を食べる |
| *afaldu* | *afaltzen dut* | 夕食を食べる |
| *jan* | *jaten dut* | 食べる |
| *edan* | *edaten dut* | 飲む |
| *lan egin* | *lan egiten dut* | 働く |
| *lo egin* | *lo egiten dut* | 眠る |
| *ikusi* | *ikusten dut* | 見る |
| *irakurri* | *irakurtzen dut* | 読む |
| *erosi* | *erosten dut* | 買う |

> **なぜ NAIZ を使うものと DUT を使うものがあるのか。** これは euskera の核心となる区別です。*Esnatzen naiz*（目が覚める）は自分に起こることで、目的語がありません。*Jaten dut*（食べる）には目的語があります。*ogia jaten dut*（パンを食べる）。これは次のユニットで詳しく見ます — 今のところは「動詞 + 助動詞」の組み合わせを覚えましょう。

## つなげた例文

> *Egunero zazpietan esnatzen naiz eta dutxatzen naiz.* — 毎日七時に目が覚めて、シャワーを浴びる。
>
> *Goizean kafea hartzen dut.* — 朝はコーヒーを飲む。
>
> *Askotan lagunekin afaltzen dut.* — よく友達と夕食を食べる。
>
> *Batzuetan lo-kuluxka egiten dut bazkalostean.* — ときどき昼食のあとに昼寝をする。

> **メモ：** *bazkalostean* =「昼食のあとに」（直訳：「食事のあとで」）。euskera の文化には、その短い昼寝のような日常の概念を表すコンパクトな言葉があります — *lo-kuluxka* がまさにそれです。
