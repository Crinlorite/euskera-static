この課を締めくくるには、もう一つのピースが必要です。それは**どうやって移動するか**です。日本語では「〜で」（「車で」「バスで」）を使います。euskeraでは接尾辞 **`-z`**（*具格*）を使い、交通手段に付けます。そして驚くことに、「足」を表す語にも付けます。

## 質問

| 質問 | 日本語 |
|---|---|
| ***Nola* zoaz?** | どうやって行く？ |
| ***Zerez* zoaz?** | 何の（手段）で行く？ |
| ***Nola* doa Iruñera?** | どうやってPamplonaへ行く？ |

二つの形（*Nola* と *Zerez*）はどちらも正しく、よく使われます。*Nola* はより一般的で、*zerez* は「何の手段で」を直接表します。

## 交通手段

| 手段 | -z 付きの形 | 日本語 |
|---|---|---|
| oin (足) | **oinez** | 徒歩で |
| kotxe / auto | **kotxez** / **autoz** | 車で |
| autobus | **autobusez** | バスで |
| tren | **trenez** | 電車で |
| metro | **metroz** | 地下鉄で |
| taxi | **taxiz** | タクシーで |
| bizikleta | **bizikletaz** | 自転車で |
| motor | **motorrez** | バイクで |
| hegazkin | **hegazkinez** | 飛行機で |
| itsasontzi | **itsasontziz** | 船で |
| furgoneta | **furgonetaz** | バンで |
| zaldi (馬) | **zaldiz** | 馬で |

> 接尾辞は母音の後では **`-z`**、子音の後では **`-ez`** です（*autobus → autobusez*）。実際には、交通手段の名詞はほとんどが母音で終わるので、基本の公式は **手段 + z** です。

## 完全な例文

- **Oinez noa lanera.** — 徒歩で仕事に行く。
- **Iruñera trenez goaz.** — 電車でPamplonaへ行く。
- **Bilbora hegazkinez doaz.** — 飛行機でBilboへ行く。
- **Eskolara bizikletaz dator alaba.** — 娘は自転車で学校へ来る。
- **Mendira oinez igotzen gara.** — 徒歩で山に登る。

## NONDIK + NORA + NOLA を組み合わせる

典型的な旅行の文には三つのピースがあります。**どこから + どこへ + どうやって**。

- **Iruñetik Donostiara autobusez doa.** — バスでPamplonaからDonostiaへ行く。
- **Bilbotik Madrilera trenez gatoz.** — 電車でBilboからMadridへ来る。
- **Etxetik euskaltegira oinez noa.** — 徒歩で家からeuskaltegiへ行く。

## ミニ会話

> — *Aupa! Bihar Bilbora **noa**.* — やあ！明日Bilboへ行くんだ。
>
> — *Eta **nola** zoaz?* — で、どうやって行くの？
>
> — *Trenez. Oso erosoa da.* — 電車で。とても快適だよ。
>
> — *Ni autoz noa beti, baina garestia da gasolina.* — 私はいつも車で行くけど、ガソリンが高いんだ。

## 「oinez」について

一見すると *oinez* は特別なケース（「足でもって」）のように見えますが、実は他と全く同じパターンです。*oin*（足）+ *ez*（具格の接尾辞）。euskeraは「徒歩で行く」を他のどんな移動手段とも同じように扱います。自分を運ぶ手段を使うのです。

## NORA / NONDIK / NOLA ブロックの最終復習

| 質問 | 接尾辞 | 質問の型 | 答えの型 |
|---|---|---|---|
| **Nora?** | -ra | どこへ？ | *Bilbo**ra** noa.* |
| **Nondik?** | -tik | どこから？ | *Bilbo**tik** nator.* |
| **Nola? / Zerez?** | -z | どうやって？ / 何で？ | *Tren**ez** noa.* |

> **コツ：** この三つの接尾辞を合わせると、ほとんどどんな旅行の文でも作れます。例えば：*Bilbotik Donostiara autobusez goaz, lagun bat bisitatzera*. —「友達に会いに、バスでBilboからDonostiaへ行きます。」この三つの接尾辞と *joan* および *etorri* の総合形をマスターすれば、日常のどんな移動についても話せるようになります。
