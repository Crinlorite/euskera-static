基本となる3つの動詞が、これからずっと出てきます。**joan** (行く)、**etorri** (来る)、そして **ibili** (歩く、「あちこち動き回る」) です。この3つにはすべて、現在形で非常によく使われる**総合形** (助動詞なし) があります。これらはひとまとまりとして覚えておくとよいでしょう。

## Joan — 行く

| 人称 | 形 | 日本語 |
|---|---|---|
| ni | **noa** | 私は行く |
| zu | **zoaz** | あなたは行く |
| hura | **doa** | 彼／彼女は行く |
| gu | **goaz** | 私たちは行く |
| zuek | **zoazte** | あなたたちは行く |
| haiek | **doaz** | 彼らは行く |

> 重要な質問: ***Nora zoaz?*** — どこへ行くの? これはレッスン03で扱います。

## Etorri — 来る

| 人称 | 形 | 日本語 |
|---|---|---|
| ni | **nator** | 私は来る |
| zu | **zatoz** | あなたは来る |
| hura | **dator** | 彼／彼女は来る |
| gu | **gatoz** | 私たちは来る |
| zuek | **zatozte** | あなたたちは来る |
| haiek | **datoz** | 彼らは来る |

> 重要な質問: ***Nondik zatoz?*** — どこから来るの? これはレッスン04で扱います。

## Ibili — 歩く、動き回る

| 人称 | 形 | 日本語 |
|---|---|---|
| ni | **nabil** | 私は歩く、動き回っている |
| zu | **zabiltza** | あなたは歩く |
| hura | **dabil** | 彼／彼女は歩く |
| gu | **gabiltza** | 私たちは歩く |
| zuek | **zabiltzate** | あなたたちは歩く |
| haiek | **dabiltza** | 彼らは歩く |

> *Ibili* は「何をしているの?」と尋ねるのにも使えます: **Zer zabiltza?** — 元気? / 最近どんなことをしてるの? 互いに知っている人どうしのあいさつとして非常によく使われます。

## その他の役立つ移動の動詞

| Euskera | 日本語 | 例文 |
|---|---|---|
| *sartu* | 入る | *Tabernan sartu naiz.* — バルに入った。 |
| *irten* | 出る | *Etxetik irten naiz.* — 家を出た。 |
| *igo* | 上がる | *Eskailerak igo ditut.* — 階段を上がった。 |
| *jaitsi* | 下りる | *Aldapa jaitsi dut.* — 坂を下りた。 |
| *iritsi / heldu* | 着く | *Iruñera iritsi naiz.* — Iruña に着いた。 |

## 総合形が大切な理由

euskera では、ほとんどの動詞は**助動詞**を使って活用します: *jan dut* (食べた)、*ikusi dut* (見た)。しかし、ごく一部の非常によく使われる動詞 — その中には **joan**、**etorri**、**egon**、**ibili**、**eduki**、**jakin** などがあります — には、現在形で使われる独自の「コンパクトな」(*総合的な*) 活用があります。これらは非常に頻繁に出てくるので、遠回しな言い方をいくつも覚えるより、6つの形を暗記するほうが簡単です。

## ミニ会話

> — *Aupa, nora zoaz?* — やあ! どこへ行くの?
>
> — *Banketxera **noa**. Eta zu, nondik **zatoz**?* — 銀行へ行くんだ。で、君は、どこから来たの?
>
> — *Lanetik **nator**.* — 仕事から来たんだ。
>
> — *Eta Lide? Zer **dabil**?* — で、Lide は? 何してるの?
>
> — *Enkargutan **dabil**, batetik bestera.* — あちこち用事をして回ってるよ。

> **メモ:** 総合形は A1 の中心となる土台です。2つ (ni/zu) ずつ、または3つ (gu/zuek/haiek) ずつのまとまりで、歌ったり、書いたり、繰り返したりして覚えましょう。練習を重ねれば、自然と身についてきます。
