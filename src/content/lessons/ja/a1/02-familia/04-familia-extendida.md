両親、兄弟姉妹、子どもの先には、おじ・おば、いとこ、甥・姪、そして姻族へと家族は広がります。ここでは最もよく使う語彙を紹介します。

## おじ・おばといとこ

| バスク語 | 日本語 |
|---|---|
| *osaba* | おじ |
| *izeba* | おば |
| *lehengusu* | いとこ（男） |
| *lehengusina* | いとこ（女） |
| *iloba* | 甥・姪（方言によっては「孫」の意味にもなり、文脈で判断する） |

## 姻族

| バスク語 | 日本語 |
|---|---|
| *aitaginarreba* | 義父 |
| *amaginarreba* | 義母 |
| *koinatu* | 義兄弟 |
| *koinata* | 義姉妹 |
| *errain* | 嫁 |
| *suhi* | 婿 |

## 使い方のパターン

- *Nire osaba Bilbon bizi da.* — 私のおじは Bilbo に住んでいます。
- *Bere lehengusinak hamabi urte ditu.* — 彼（女）のいとこは12歳です。
- *Zure koinata euskalduna da?* — あなたの義姉妹はバスク人ですか？

> 姻族を表す言葉は、既存の語を組み合わせて作られることが多いです（*aita-* ＋ *ginarreba*）。内部の論理を覚える必要はなく、使っているうちに自然と身についていきます。
