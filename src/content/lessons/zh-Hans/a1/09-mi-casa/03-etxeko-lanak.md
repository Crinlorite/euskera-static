在本课中，你将学习谈论 **etxeko lanak**（家务）的基础动词。在巴斯克语中，动词通常以**分词**形式给出（`-tu`、`-i`、`-n`），这就是“词典里”的形式：*garbitu*、*prestatu*、*jaso*……

## 关键动词

| Euskera | 中文 |
|---|---|
| *garbitu* | 打扫 |
| *prestatu* | 准备 |
| *jaso* | 收拾 |
| *jantzi* | 穿 / 铺设 |
| *atera* | 拿出 |
| *zabaldu* | 晾 / 打开 |
| *lisatu* | 熨烫 |
| *paseatu* | 散步 / 遛 |
| *erosi* | 买 |

## 常见家务

| 家务 | 怎么说 |
|---|---|
| 铺床 | *ohea jantzi* |
| 洗碗 | *ontziak garbitu* / *harrikoa egin* |
| 用吸尘器吸地 | *xurgagailua pasa* |
| 晾衣服 | *arropa zabaldu* |
| 熨衣服 | *arropa lisatu* |
| 打扫浴室 | *komuna garbitu* |
| 倒垃圾 | *zaborra atera* |
| 做饭 | *bazkaria prestatu* |
| 购物 | *erosketak egin* |
| 遛狗 | *txakurra paseatu* |

## 从词典形式到现在时习惯式

当一个动作习惯性地重复发生时（每天、每周……），动词会变为 `-tzen` ＋助动词 **dut/duzu/du……** 的形式（你已经在 *ukan* 中认识过它）：

| 人称 | 例句 | 中文 |
|---|---|---|
| Nik | *Nik etxea **garbitzen dut**.* | 我打扫房子。 |
| Zuk | *Zuk arropa **lisatzen duzu**.* | 你熨衣服。 |
| Berak | *Aitak bazkaria **prestatzen du**.* | 我父亲做饭。 |
| Guk | *Guk erosketak **egiten ditugu**.* | 我们去购物。 |

> 在本课中你不必掌握动词变位。只要记住用 **`-tzen` ＋ dut/du** 这个模式来谈论日常即可。

## 频率

要表达你多久做一次某项家务：

- **egunero** — 每天
- **astean behin** — 每周一次
- **astean bitan / hirutan** — 每周两次 / 三次
- **noizean behin** — 偶尔
- **inoiz ez** — 从不

> *Nire etxean nik **astean behin** lisatzen dut, baina **egunero** garbitzen dut sukaldea.* — 在我家，我每周熨一次衣服，但我每天都打扫厨房。

## 询问谁做什么

- **Nork garbitzen ditu ontziak?** — 谁洗碗？
- **Nor arduratzen da arropaz?** — 谁负责洗衣服？
- **Zuri gustatzen zaizu sukaldatzea?** — 你喜欢做饭吗？

> **文化小知识：** 关于家庭内家务分配的调查问卷是巴斯克语学校（euskaltegi）教材中的经典内容——而且不无道理。这是与朋友和家人进行真实对话时非常实用的词汇。
