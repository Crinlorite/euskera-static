在 Euskal Herria，社交生活很大一部分都发生在 **taberna**（酒吧）里。用巴斯克语掌握基础的饮料和食物会立刻为你打开一扇门：一群朋友、去一个新村庄、一次 *poteo*（巡杯）。我们从词汇开始。

## 咖啡及其各种形式

咖啡大概是你点得最多的东西。它有好几种形式：

| 巴斯克语 | 中文 |
|---|---|
| *kafea* | 咖啡（泛指） |
| *kafe hutsa* | 清咖啡（字面："纯/空咖啡"） |
| *kafesnea* | 拿铁咖啡（由 *kafe* + *esne* 构成） |
| *kafe ebakia* | 玛奇朵咖啡（*ebaki* = "切开的"） |
| *kafe deskafeinatua* | 低因咖啡 |
| *infusioa* | 花草茶 |
| *tea* | 茶 |

> *Kafesne* 字面意思是"咖啡-牛奶"。巴斯克语就这样构成许多复合词，不用连字符，而把主词放在末尾。

## 其他饮料

| 巴斯克语 | 中文 |
|---|---|
| *ura* | 水 |
| *zumoa* | 果汁 |
| *esnea* | 牛奶 |
| *ardoa* | 葡萄酒 |
| *ardo gorria* | 红葡萄酒 |
| *ardo zuria* | 白葡萄酒 |
| *garagardoa* | 啤酒 |
| *zuritoa* | 小生啤（小杯啤酒，很典型） |
| *sagardoa* | 苹果酒 |
| *txakolina* | txakoli（巴斯克白葡萄酒） |
| *patxarana* | 黑刺李酒 |

> 三个有趣的词：**sagardo**（苹果酒）= *sagar*（苹果）+ *ardo*（酒）；**garagardo**（啤酒）= *garagar*（大麦）+ *ardo*；**txakolina** 是一种年轻清淡的白葡萄酒，是 Euskadi 的招牌。

## 冠词 `-a`

当这些词在句子里以"中性"形式出现时，几乎都会在末尾带上 **`-a`**：这就是冠词"el / la"。

- *Ardoa gustatzen zait.* — 我喜欢葡萄酒。
- *Ura nahi dut.* — 我想要水。

当你点"一份/一个"时，`-a` 会消失，换成 **bat**（"一个/一份"）：

- *Kafe **bat**, mesedez.* — 一杯咖啡，谢谢。
- *Garagardo **bat**.* — 一杯啤酒。

> 在第 3 课你会详细看到这一点。现在先记住带 `-a` 的形式，以便在酒吧菜单上认出它们。
