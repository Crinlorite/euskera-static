在谈论计划和日程安排之前，我们先复习一周七天，并学习如何把它们与日常活动联系起来。

## 一周七天

| 巴斯克语 | 中文 | 提示 |
|---|---|---|
| *astelehena* | 星期一 | aste + lehen = "一周的第一天" |
| *asteartea* | 星期二 | aste + arte = "一周之间" |
| *asteazkena* | 星期三 | aste + azken = "工作周的最后一天"（历史上的含义） |
| *osteguna* | 星期四 | egun = 日 |
| *ostirala* | 星期五 | 与 *osteguna* 共享词根 *ortzi/osti-*（天空） |
| *larunbata* | 星期六 | larun + bata |
| *igandea* | 星期日 | 历史上的节庆日 |

> **记忆提示：** 单词 **aste**（一周）出现在工作日的开头。*Astelehen-、astear-、asteazken-* 几乎是在大声数出一周的开头、中间和结尾。

## 说"在星期一" —— 后缀 `-(e)an`

要给某个事件标注日期，在该日子后面加上 `-(e)an`：

| 日子 | "在 + 日子" |
|---|---|
| astelehena | astelehen**ean** |
| asteartea | astearte**an** |
| asteazkena | asteazken**ean** |
| osteguna | ostegun**ean** |
| ostirala | ostirale**an** |
| larunbata | larunbat**ean** |
| igandea | igande**an** |

- *Astelehenean lan egiten dut.* —— 我星期一工作。
- *Larunbatean afaltzera joango gara.* —— 星期六我们要去吃晚饭。

## 说"每个 X" —— 后缀 `-ero`

表示规律性的频率：

- *astelehenero* —— 每个星期一
- *larunbatero* —— 每个星期六
- *egunero* —— 每天
- *astero* —— 每周
- *astebururo* —— 每个周末

## 日子 + 活动：例句

| 巴斯克语 | 中文 |
|---|---|
| *Astelehenean kiroldegira noa.* | 星期一我去健身房。 |
| *Asteartean euskara klasea dut.* | 星期二我有巴斯克语课。 |
| *Ostiralean parranda egiten dugu.* | 星期五我们出去玩。 |
| *Larunbatean familiaren etxera joango naiz.* | 星期六我要去家人家里。 |
| *Igandean lasai gelditzen naiz etxean.* | 星期日我安静地待在家里。 |

## Asteburua —— 周末

`Asteburua`（周末）是一个非常实用的词：

- *Asteburuan zer egingo duzu?* —— 你周末打算做什么？
- *Asteburu honetan mendira joango naiz.* —— 这个周末我要去山上。
- *Asteburu pasa egitea gustatzen zait.* —— 我喜欢去度周末（短途出游）。

> **文化小提示：** 在 *igandea* 旁边你常会看到 *jaieguna*（节庆日）：*igandeak eta jaiegunak* 这个说法在公交时刻表、商店和市政厅的告示牌上很常见。
