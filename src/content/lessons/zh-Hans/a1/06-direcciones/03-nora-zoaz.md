**NORA 格**（也叫*向格 adlativo*）标记一个运动的**目的地**：你朝哪里去。在中文里通常用"去……"或"到……"来表达（"我*回*家"、"去毕尔巴鄂"、"去公园"）。

## 疑问词：Nora?

| 疑问句 | 中文 |
|---|---|
| ***Nora* zoaz?** | 你去哪里？ |
| ***Nora* doa?** | 他/她去哪里？ |
| ***Nora* goaz?** | 我们去哪里？ |
| ***Nora* doaz?** | 他们去哪里？ |

## 后缀：-ra

要回答，我们在目的地后面加上 **`-ra`**。具体形式根据词尾来调整：

| 地点词尾 | 模式 | 例子 |
|---|---|---|
| -a（固有词尾，*eskola*） | + ra | eskola → **eskolara** |
| 元音但非 -a 词尾（*Bilbo*、*etxe*） | + ra | etxe → **etxera**, Bilbo → **Bilbora** |
| 辅音（*Madril*、*Gasteiz*） | + era | Madril → **Madrilera**, Gasteiz → **Gasteizera** |

## 例句表

| 地点 | -ra 形式 | 示范句 |
|---|---|---|
| etxe（家） | **etxera** | *Etxera noa.* — 我回家。 |
| eskola | **eskolara** | *Eskolara doaz.* — 他们去学校。 |
| euskaltegi | **euskaltegira** | *Euskaltegira goaz.* — 我们去巴斯克语学校。 |
| taberna | **tabernara** | *Tabernara zoaz?* — 你去酒吧吗？ |
| lan（工作） | **lanera** | *Lanera nator.* — 我来上班。（同样在辅音后用 -era） |
| Bilbo | **Bilbora** | *Bilbora doa.* — 他/她去毕尔巴鄂。 |
| Donostia | **Donostiara** | *Donostiara goaz asteburuan.* — 我们周末去 Donostia。 |
| Madril | **Madrilera** | *Madrilera doaz.* — 他们去马德里。 |
| mendi（山） | **mendira** | *Mendira goaz larunbatean.* — 我们周六去山里。 |

> **小窍门：** *non* → *nora* 的变化规律和 *kale-an* → *kale-ra* 一样。如果你会构造 *Bilbon*（在毕尔巴鄂），你就会构造 *Bilbora*（去毕尔巴鄂）：把 *-n* 换成 *-ra* 即可。如果你会构造 *Madrilen*，你就会构造 *Madrilera*。

## Zertara? — "去做什么？"

要询问你去做**什么**（带着什么目的），用 **Zertara zoaz?**，并用动词 + 后缀 **`-tzera`** 来回答：

| 疑问句 | 回答 |
|---|---|
| *Zertara zoaz tabernara?* | *Kafea hartzera.* — 去喝咖啡。 |
| *Zertara doa Kepa supermerkatura?* | *Erostera.* — 去购物。 |
| *Zertara goaz mendira?* | *Ibiltzera.* — 去散步。 |
| *Zertara doaz hondartzara?* | *Igeri egitera.* — 去游泳。 |

> **模式：** *（动词分词形式）+ -t(z)era*：*erosi → erostera*, *jan → jatera*, *ikasi → ikastera*, *ikusi → ikustera*, *bisitatu → bisitatzera*。基本上相当于中文的"去 + 动词"。

## 小对话

> — *Nora zoaz, Maialen?* — 你去哪里，Maialen？
>
> — *Euskaltegira noa.* — 我去巴斯克语学校。
>
> — *Eta zertara?* — 去做什么呢？
>
> — *Klasera, noski! Euskara ikastera.* — 去上课，当然啦。去学巴斯克语。
>
> — *Eta gero?* — 那之后呢？
>
> — *Tabernara goaz lagunekin pote bat hartzera.* — 我们和朋友去酒吧喝一杯。

## 方向副词

指示词也有它们的 NORA 形式：

| NON（位格） | NORA（方向） | 含义 |
|---|---|---|
| hemen | **hona** | 朝这里 |
| hor | **horra** | 朝那里 |
| han | **hara** | 朝那边 |

- **Hona zatoz?** — 你来这里吗？
- **Hara doaz.** — 他们朝那边去。

> **重要提醒：** 很多人一开始会搞混：*etxean*（在家）≠ *etxera*（回家）≠ *etxetik*（从家，第 04 课）。这三个词很像，因为都和"家"有关，但所表达的信息完全不同。请在脑中记牢：**NON = -an（在/位于）**，**NORA = -ra（去）**，**NONDIK = -tik（来自，第 4 课）**。
