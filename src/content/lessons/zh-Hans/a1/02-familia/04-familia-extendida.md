除了父母、兄弟姐妹和子女之外，家庭还延伸到叔叔阿姨、堂表兄弟姐妹、侄子侄女以及姻亲。这里是最常用的词汇。

## 叔叔阿姨和堂表兄弟姐妹

| 巴斯克语 | 中文 |
|---|---|
| *osaba* | 叔叔/舅舅 |
| *izeba* | 阿姨/姑姑 |
| *lehengusu* | 堂/表兄弟 |
| *lehengusina* | 堂/表姐妹 |
| *iloba* | 侄子/侄女（在某些方言中也指"孙子/孙女"——由上下文决定） |

## 姻亲

| 巴斯克语 | 中文 |
|---|---|
| *aitaginarreba* | 岳父/公公 |
| *amaginarreba* | 岳母/婆婆 |
| *koinatu* | 姐夫/妹夫 |
| *koinata* | 嫂子/弟媳 |
| *errain* | 儿媳 |
| *suhi* | 女婿 |

## 使用模式

- *Nire osaba Bilbon bizi da.* — 我的叔叔住在 Bilbao。
- *Bere lehengusinak hamabi urte ditu.* — 他/她的表姐妹十二岁。
- *Zure koinata euskalduna da?* — 你的嫂子是巴斯克人吗？

> 表示姻亲的词常常由现有的词组合而成（*aita-* + *ginarreba*）。不必死记内部逻辑；通过使用自然就会掌握。
