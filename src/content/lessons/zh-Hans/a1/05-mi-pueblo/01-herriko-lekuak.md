在学会指路之前，你需要掌握谈论街区（**auzoa**）和村镇（**herria**）的基本词汇。在本课中，我们汇集了最典型的场所和服务设施。

## 商店与商业

| 巴斯克语 | 中文 |
|---|---|
| *taberna* | 酒吧 |
| *kafetegia* | 咖啡馆 |
| *jatetxea* | 餐厅 |
| *okindegia* | 面包店 |
| *harategia* | 肉店 |
| *arrandegia* | 鱼店 |
| *fruta-denda* | 水果店 |
| *gozoki-denda* | 糖果店 |
| *merkatua* / *supermerkatua* | 市场 / 超市 |
| *jantzi-denda* | 服装店 |
| *liburu-denda* | 书店 |
| *loradenda* | 花店 |

> 许多词遵循 **产品 + denda**（"……店"）的构词模式：*fruta-denda*、*liburu-denda*、*jantzi-denda*。只要你知道产品名，就能说出对应的场所。

## 后缀 -tegi 及其同类

注意这些能产性很强的后缀：它们会反复出现。

- **-tegi** → 某物所在的地方：*euskal·tegi*（学巴斯克语的地方）、*ile-apain·degi*（整理头发的地方）、*liburu·tegi*（放书的地方）。
- **-degi / -tegi** → 表达同一含义的变体：*gasolin·degi*、*okin·degi*。
- **-etxe**（"房子"）→ *udal·etxe*（市政会之屋）、*osasun·etxe*（健康之屋）、*jatetxe*（吃饭的地方）。

> **文化小贴士：** 在许多巴斯克的街区和村镇里，**taberna**（酒吧）和**广场**是最重要的聚会地点，而 **frontoia**（回力球场）通常就在教堂旁边。*euskaltegi* 正是你正在学习这门语言的那种学校。
