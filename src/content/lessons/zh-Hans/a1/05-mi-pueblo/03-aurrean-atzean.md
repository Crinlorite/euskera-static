一旦你会说“在 X 处”，下一步就是说出**具体在哪里**：上面、下面、旁边、后面……在巴斯克语中，这些方位总是用**参照地点 ＋ 方位**的模式来构造。

## 基本方位

| 巴斯克语 | 中文 |
|---|---|
| *aurrean* | 前面 |
| *atzean* | 后面 |
| *ondoan* | 旁边 |
| *gainean* | 上面、在……之上 |
| *azpian* | 下面 |
| *artean* | 之间 |
| *ezkerrean* | 左边 |
| *eskuinean* | 右边 |
| *barruan* | 里面 |
| *kanpoan* | 外面 |
| *erdian* | 中间、正中央 |

## 如何构造：-(r)en ＋ 方位

基本公式是**（参照物）-(r)en ＋ 方位**。参照物要带属格 `-(r)en`（相当于中文里的“……的”）：

| 巴斯克语 | 中文（字面） |
|---|---|
| **mahaiaren gainean** | 在桌子上面（字面：“桌子的，上面”） |
| **etxearen aurrean** | 在房子前面 |
| **elizaren atzean** | 在教堂后面 |
| **parkearen ondoan** | 在公园旁边 |
| **kutxaren barruan** | 在盒子里面 |
| **bi mendien artean** | 在两座山之间 |

规律：参照物在前，**然后**才是方位。这正好和中文相反。

## 完整例句

- **Liburua mahaiaren gainean dago.** — 书在桌子上面。
- **Katua aulkiaren azpian dago.** — 猫在椅子下面。
- **Eliza plazaren erdian dago.** — 教堂在广场正中央。
- **Botika eskolaren ondoan dago.** — 药店在学校旁边。
- **Ni Maiteren eta Anaren artean nago.** — 我在 Maite 和 Ana 之间。

## 地点副词：hemen / hor / han

要快速指示位置有三个副词——而且要注意，它们是**三个距离**，不像中文那样只有两个：

| 巴斯克语 | 中文 | 何时使用 |
|---|---|---|
| *hemen* | 这里 | 离说话者近 |
| *hor* | 那儿 | 离听者近 |
| *han* | 那里 | 离两者都远 |

> 这三个副词与指示词 *hau / hori / hura* 相对应，也与它们的长式方位形 *honetan / horretan / hartan*（“在这个 / 在那个 / 在那一个里”）相对应。

简短例句：

- **Hemen** taberna bat dago. — 这里有一家酒吧。
- **Hor** parkea dago. — 那儿有公园。
- **Han** dago euskaltegia. — 巴斯克语学校在那里。

## 小对话：玩捉迷藏

> — *Pilota mahaiaren azpian dago?* — 球在桌子下面吗？
>
> — *Ez, pilota ez dago mahaiaren azpian.* — 不，球不在桌子下面。
>
> — *Aulkiaren atzean dago?* — 在椅子后面吗？
>
> — *Bai, hor dago!* — 对，就在那儿！

> **窍门：** 所有方位词都已经把后缀 `-an` 内含其中（*aurr-ean*、*atz-ean*、*azp-ian*……）。你不需要再加任何东西。
