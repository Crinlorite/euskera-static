在巴斯克语中，时间用动词 **izan**（是）来构造，而且除了"一点"以外，钟点总是用**复数**。字面意思是：*"是五点们"*、*"是八点们"*。一开始听起来很奇怪，但这个规律非常稳定。

## 提问

要问时间，使用 **Zer ordu da?**（"几点了？"）。

> *— Zer ordu da?*
>
> *— Zazpiak dira.* （七点了。）

## 整点的钟点

| 时间 | 巴斯克语 |
|---|---|
| 1:00 | **Ordu bata da** |
| 2:00 | **Ordu biak dira** |
| 3:00 | **Hirurak dira** |
| 4:00 | **Laurak dira** |
| 5:00 | **Bostak dira** |
| 6:00 | **Seiak dira** |
| 7:00 | **Zazpiak dira** |
| 8:00 | **Zortziak dira** |
| 9:00 | **Bederatziak dira** |
| 10:00 | **Hamarrak dira** |
| 11:00 | **Hamaikak dira** |
| 12:00 | **Hamabiak dira** |

## 为什么一点和两点很特殊

- **Ordu bata**（一点）是**单数**，所以要用 **da**：*Ordu bata da*。
- **Ordu biak**（两点）前面带上 *ordu* 这个词，是因为单独的 *biak* 可能会引起混淆——这是唯一一个很多人会加上 "ordu" 来表达更清楚的钟点。其余的（从三点起）都单独使用：*hirurak*、*laurak*、*bostak*……

## 后缀 `-ak`

那个结尾的 **-ak** 是从哪来的呢？它是巴斯克语的复数标记（你在 *gurasoak*、*aiton-amonak* 中见过它）。这里也一样：*bost*（五）→ *bostak*（"五点们"）+ *dira*（"是"）。

> **注意：** 很多人用 12 小时制来报时，跟西班牙语一样。要区分上午还是下午，只需在前面加上 *goizeko*（上午的）或 *arratsaldeko*（下午的）：*goizeko zortziak*（上午八点）。我们会在第 3 课中讲到。

## 练习

现在就看一眼你手机上的时钟，然后大声把时间说出来。如果是十一点：*hamaikak dira*。如果是一点：*ordu bata da*。整个星期里每次看时钟都这样做——这是把它变成自动反应的最好方法。
