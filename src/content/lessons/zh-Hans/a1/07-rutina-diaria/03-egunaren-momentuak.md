巴斯克语中一天的各个时段靠两个关键后缀来运作：**`-(e)an`**（"在/于"）和 **`-(e)ko`**（"的"），具体用哪个取决于你的用途。好处是它们与所有单词的组合都很有规律。

## 基本词汇

| Euskera | 中文 |
|---|---|
| *goiza* | 早上 |
| *eguerdia* | 中午 |
| *arratsaldea* | 下午 |
| *iluntzea* | 傍晚 |
| *gaua* | 夜晚 |

## "在早上 / 在下午" — 方位格 `-(e)an`

要表达某事发生在什么时候（一天中的哪个时段），在词根上加 **-(e)an**：

| 什么时候 | Euskera |
|---|---|
| 在早上 | **goizean** |
| 在中午 | **eguerdian** |
| 在下午 | **arratsaldean** |
| 在傍晚 | **iluntzean** |
| 在晚上 | **gauean** |

> *Goizean kafea hartzen dut.* — 我早上喝咖啡。
>
> *Arratsaldean lagunekin egoten naiz.* — 我下午和朋友们在一起。
>
> *Gauean lo egiten dut.* — 我晚上睡觉。

## "早上的 / 下午的" — 后缀 `-ko`

要精确说明**在一天中的几点**，把那个词加上 **-ko** 放在时间前面：

| 准确时间 | Euskera |
|---|---|
| 早上八点 | **goizeko zortziak** |
| 中午十二点 | **eguerdiko hamabiak** |
| 下午五点 | **arratsaldeko bostak** |
| 晚上九点 | **gaueko bederatziak** |

> *Goizeko zazpietan esnatzen naiz.* — 我早上七点醒来。
>
> *Arratsaldeko hirurak dira.* — 现在是下午三点。

## 基本时间标志词

这些词你每天都会用到：

| Euskera | 中文 |
|---|---|
| *gaur* | 今天 |
| *atzo* | 昨天 |
| *bihar* | 明天（第二天） |
| *etzi* | 后天 |
| *orain* | 现在 |
| *gero* | 待会儿，之后 |
| *lehen* | 之前 |

> 注意：要小心区分 **bihar**（第二天，"明天"）和 **goiza**（早上，到中午为止）。中文里我们用不同的词，但西班牙语中这两个用同一个词，巴斯克语则不然。

## 综合运用

> *Bihar goizean lanera joango naiz.* — 明天早上我会去上班。
>
> *Atzo arratsaldean zinemara joan ginen.* — 昨天下午我们去了电影院。
>
> *Gaur gaueko hamarretan afalduko dut.* — 今天晚上十点我会吃晚饭。

> **文化小贴士：** Euskal Herria 以它的 *iluntze* 闻名——就是傍晚时分，那个三五好友结伴出门去 *poteo*（一轮接一轮地喝小杯酒）的时刻。它既不是下午也不是晚上：它是一天中拥有自己独特身份的一个时段。
