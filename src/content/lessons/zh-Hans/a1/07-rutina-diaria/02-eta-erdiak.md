在巴斯克语里，分钟的表达主要靠三种结构就能搞定绝大多数情况：**eta erdiak**（半）、**eta laurden**（一刻）和 **gutxi**（差）。窍门在于知道每个词放在哪里。

## “半点” —— 特殊情况

与其他情况不同，“半点”会让钟点也用复数表达 —— 而且“半”这个词也是复数：

| 时间 | 巴斯克语 |
|---|---|
| 1:30 | **Ordu bat eta erdiak dira** |
| 2:30 | **Ordu bi eta erdiak dira** |
| 3:30 | **Hiru eta erdiak dira** |
| 4:30 | **Lau eta erdiak dira** |
| 5:30 | **Bost eta erdiak dira** |
| 7:30 | **Zazpi eta erdiak dira** |

> 请注意：用 *eta erdiak* 时，前面的数字**不带 -ak**（是 *bost*，不是 *bostak*），因为复数已经由 *erdiak* 来体现了。

## “一刻”和已过的分钟

当时间已经超过整点几分钟时，用公式 **钟点-ak eta 分钟**：

| 时间 | 巴斯克语 |
|---|---|
| 3:05 | **Hirurak eta bost dira** |
| 4:10 | **Laurak eta hamar dira** |
| 5:15 | **Bostak eta laurden dira** |
| 6:20 | **Seiak eta hogei dira** |
| 7:25 | **Zazpiak eta hogeita bost dira** |

**Laurden** 意为“一刻”，所以 **eta laurden** = “一刻”。

## “差” —— gutxi 放在后面

从半点之后开始，我们以下一个整点为基准来说时间，用 **gutxi**（“差”）这个词。语序是：**钟点-ak + 分钟 + gutxi**。

| 时间 | 巴斯克语 |
|---|---|
| 6:35 | **Zazpiak hogeita bost gutxi dira** |
| 7:40 | **Zortziak hogei gutxi dira** |
| 8:45 | **Bederatziak laurden gutxi dira** |
| 9:50 | **Hamarrak hamar gutxi dira** |
| 10:55 | **Hamaikak bost gutxi dira** |

> 对说西班牙语的人来说，最奇怪的地方是 **gutxi** 要放在数字**后面**，而不是前面。*Bederatziak laurden gutxi* 字面意思是“九点 一刻 差”。

## 小结表

| 概念 | 关键词 | 位置 |
|---|---|---|
| 半 | *eta erdiak* | 放在钟点之后；数字不带 -ak |
| 一刻 | *eta laurden* | 放在带 -ak 的钟点之后 |
| 几分 | *eta + 数字* | 放在带 -ak 的钟点之后 |
| 差一刻 | *laurden gutxi* | 放在带 -ak 的下一个钟点之后 |
| 差几分 | *数字 + gutxi* | 放在带 -ak 的下一个钟点之后 |

## 对话示例

> *— Zer ordu da?*
>
> *— **Zazpiak eta laurden dira.*** （现在是七点一刻。）
>
> *— Eta orain?*
>
> *— **Zortziak laurden gutxi dira.*** （现在是差一刻八点。）

> **文化小贴士：** 在许多巴斯克村庄，教堂的钟声仍在报时和报刻。如果你停下来听，你的耳朵会在嘴巴还没把它变成下意识反应之前，就先习惯这种节奏。
