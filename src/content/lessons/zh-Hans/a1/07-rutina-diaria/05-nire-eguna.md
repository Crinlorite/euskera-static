这节课把本单元的所有内容串联起来：钟点、一天中的时段、日常动词和频率。目标是让你能够用巴斯克语从头到尾完整地讲述你的一天。

## 关键问句："Zer ordutan…？"

| 问句 | 中文 |
|---|---|
| *Zer ordutan esnatzen zara?* | 你几点醒来？ |
| *Zer ordutan jaikitzen zara?* | 你几点起床？ |
| *Zer ordutan gosaltzen duzu?* | 你几点吃早餐？ |
| *Zer ordutan bazkaltzen duzu?* | 你几点吃午饭？ |
| *Zer ordutan ateratzen zara etxetik?* | 你几点出门？ |
| *Zer ordutan afaltzen duzu?* | 你几点吃晚饭？ |
| *Zer ordutan oheratzen zara?* | 你几点睡觉？ |

> 注意助动词的变化：*esnatzen **zara*** (NOR)，但 *gosaltzen **duzu*** (NOR-NORK)。你要慢慢留意 "动词 + 助动词" 这一对的搭配。

## "在 X 点" —— 怎么说

要回答 "在（几点）"，把钟点取复数形式，并把 *-ak* 改为 **-etan**（中间插入 `-e-`，除了那些本身已经以元音结尾的）：

| 钟点 | "在……点" |
|---|---|
| zazpiak（七点） | **zazpietan** |
| zortziak | **zortzietan** |
| hamarrak | **hamarretan** |
| hamabiak | **hamabietan** |
| ordu bata（一点） | **ordu batean**（单数，保留其形式） |
| ordu biak | **ordu bietan** |
| bost eta erdiak | **bost eta erdietan**（五点半） |
| zortziak laurden gutxi | **zortziak laurden gutxitan**（八点差一刻） |

## 完整的一天 —— 范文

读一读这段范文，注意所有的部分是如何衔接起来的：

> *Ni goizeko zazpietan esnatzen naiz. Poliki-poliki altxatzen naiz eta komunera joaten naiz. Dutxa bat hartzen dut, arropa janzten dut eta sukaldera joaten naiz. Sukaldean gosaria prestatzen dut eta lasai-lasai gosaltzen dut. Ondoren, etxetik ateratzen naiz eta lanera joaten naiz. Eguerdiko ordu bietan bazkaltzen dut. Arratsaldean batzuetan zinemara joaten naiz, beste batzuetan lagunekin egoten naiz. Gauean, etxean afaltzen dut, telebista ikusten dut eta hamaiketan oheratzen naiz.*

大致翻译：
> *我早上七点醒来。我慢慢地起床，然后去洗手间。我洗个澡，穿好衣服，然后去厨房。在厨房里我准备早餐，悠闲地吃早餐。之后我出门去上班。中午两点我吃午饭。下午有时我去电影院，有时和朋友们待在一起。晚上我在家吃晚饭，看电视，十一点上床睡觉。*

## 构建你自己的一天

按照这个模板，填入你真实的信息。这是巩固所学的最佳方法：

1. **你几点醒来：** *Goizeko ___etan esnatzen naiz.*
2. **起床后做什么：** *... dutxatzen naiz / gosaltzen dut / ...*
3. **你怎么去上班或上学：** *... oinez / autoz / autobusez joaten naiz.*
4. **你下午做什么：** *Arratsaldean ...*
5. **你几点吃晚饭、几点睡觉：** *Gaueko ___etan afaltzen dut eta ___etan oheratzen naiz.*

## 用于衔接的实用短语

- *Lehenengo* — 首先
- *Gero / Ondoren* — 然后，之后
- *Eta gero* — 然后
- *Azkenik* — 最后
- *Egunero* — 每天
- *Batzuetan / Beste batzuetan* — 有时 / 其他时候
- *Beti* — 总是
- *Inoiz ez* — 从不

> **文化小贴士：** 在 Euskal Herria，很多人在 14:00 到 15:00 之间吃午饭（*bazkaldu*），晚饭（*afaldu*）吃得相当晚，大约在 21:00–22:00。*Hamaiketakoa* —— 上午十一点的加餐，通常是一份 pintxo 或三明治 —— 是一项传统。如果你和一群朋友（cuadrilla）在一起，11 点有人说 *hamaiketakoa egitera goaz*，你就知道该干什么了。
