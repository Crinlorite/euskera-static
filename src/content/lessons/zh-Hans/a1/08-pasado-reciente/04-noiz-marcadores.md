要把发生过的事情讲清楚，你需要说出**什么时候**。本课给你那些与近过去时搭配、用来把动作锚定在时间里的标记词。

## 基本标记词

| 巴斯克语 | 中文 |
|---|---|
| *gaur* | 今天 |
| *atzo* | 昨天 |
| *bihar* | 明天 |
| *etzi* | 后天 |
| *orain* | 现在 |
| *oraintxe* | 现在马上 |
| *lehen* | 以前 |
| *gero* | 随后，之后 |

> 如果你注意到了，*atzo*（昨天）和 *bihar*（明天）是两个非常短的词，一下子就能记住。*Etzi*（后天）是一个中文用一个词就能表达、而很多语言要用好几个词的概念——这是巴斯克语送的一份小礼物。

## 与一天中的时段搭配

这些标记词与上一单元的 *goizean*、*arratsaldean*、*gauean* 等词搭配使用。公式是：**标记词 + 一天中的时段**。

| 巴斯克语 | 中文 |
|---|---|
| *gaur goizean* | 今天早上 / 今天上午 |
| *gaur eguerdian* | 今天中午 |
| *gaur arratsaldean* | 今天下午 |
| *gaur gauean* | 今晚 |
| *atzo goizean* | 昨天上午 |
| *atzo arratsaldean* | 昨天下午 |
| *atzo gauean* | 昨晚 |
| *bihar goizean* | 明天上午 |

> **注意：** *gaur gauean* 是“今晚”（你即将面对的那个夜晚，或你正在经历的那个夜晚）；*atzo gauean* 是“昨晚”。巴斯克语区分得非常清楚。

## 频率与次数

| 巴斯克语 | 中文 |
|---|---|
| *behin* | 一次 |
| *bi aldiz* | 两次 |
| *hiru aldiz* | 三次 |
| *askotan* | 经常、很多次 |
| *batzuetan* | 有时 |
| *inoiz ez* | 从不 |
| *gaur goizetik* | 从今天早上起 |
| *aste honetan* | 这一周 |
| *hilabete honetan* | 这个月 |

## 带近过去时与标记词的句子

> *Gaur goizean lanera joan naiz autoz.* — 今天早上我开车去上班了。
>
> *Gaur arratsaldean Mikelekin egon naiz.* — 今天下午我和 Mikel 在一起。
>
> *Gaur oso berandu esnatu naiz.* — 今天我醒得很晚。
>
> *Gaur eguerdian etxean bazkaldu dut.* — 今天中午我在家吃了午饭。
>
> *Aste honetan bi aldiz egon naiz zinean.* — 这一周我去了两次电影院。

## 句子的语序

在巴斯克语中，一个带时间标记词的句子，典型语序是：

> **什么时候 + 在哪里 + 什么 + 动词**

> **Gaur goizean** + **euskaltegian** + **euskara** + **ikasi dut**. — 今天早上我在 euskaltegi 学了巴斯克语。

这并不是唯一的可能——巴斯克语允许移动各个成分来强调——但只要你遵循这个语序，就不会出错。

## 小对话

> *— Eta gaur zer egin duzu, Maialen?*
>
> *— **Gaur goizean** lanera joan naiz oinez. **Eguerdian** lagunekin bazkaldu dut tabernan. **Arratsaldean** etxera bueltatu naiz eta liburua irakurri dut. **Gauean** ez dut afaldu, asko jan baitut bazkarian.*

> **注：** *gaur* 涵盖一整天。如果你想说得更精确，就加上时段（*gaur goizean*、*gaur arratsaldean*）。在日常对话中，*Zer egin duzu gaur?* 会被理解为“你今天总体上做了什么？”。
