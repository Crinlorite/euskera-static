**及物动词** 是带宾语的动词：有人吃 *某物*、看 *某物*、读 *某物*。在巴斯克语中，这些动词使用助动词 **ukan**（dut, duzu, du……）。在近过去时中，公式是：

> **partizipioa + dut/duzu/du/dugu/duzue/dute**

## 基本及物动词

| Partizipioa | 我（已经……） | 中文 |
|---|---|---|
| *jan* | **jan dut** | 我吃过了 |
| *edan* | **edan dut** | 我喝过了 |
| *ikusi* | **ikusi dut** | 我看过了 |
| *irakurri* | **irakurri dut** | 我读过了 |
| *entzun* | **entzun dut** | 我听过了 |
| *erosi* | **erosi dut** | 我买过了 |
| *egin* | **egin dut** | 我做过了 |
| *hartu* | **hartu dut** | 我拿过了 / 取过了 |
| *ekarri* | **ekarri dut** | 我带来过了 |
| *eraman* | **eraman dut** | 我带走过了 |
| *ikasi* | **ikasi dut** | 我学过了 / 我学习过了 |
| *eskatu* | **eskatu dut** | 我点过了 / 要过了 |
| *ordaindu* | **ordaindu dut** | 我付过钱了 |
| *idatzi* | **idatzi dut** | 我写过了 |
| *prestatu* | **prestatu dut** | 我准备过了 |

## 用一个动词完整变位

我们以 *ikusi*（看）为例：

| 代词 + 主语 | 形式 | 中文 |
|---|---|---|
| Nik | **ikusi dut** | 我看过了 |
| Zuk | **ikusi duzu** | 你看过了 |
| Hark | **ikusi du** | 他/她看过了 |
| Guk | **ikusi dugu** | 我们看过了 |
| Zuek | **ikusi duzue** | 你们看过了 |
| Haiek | **ikusi dute** | 他们看过了 |

> 及物动词的主语带 **NORK** 标记（-k）：*nik, zuk, hark, guk, zuek, haiek*。你会经常见到它，尤其是当主语是名字时：*Mikelek*（Mikel 已经……）、*Maialenek*（Maialen 已经……）。如果你感到困惑，在 A1 阶段你可以省略代词，只留下动词：靠上下文就足够了。

## 常用句子

- *Gaur goizean egunkaria irakurri dut.* — 今天早上我读了报纸。
- *Bazkaltzeko arraina jan dut.* — 午饭我吃了鱼。
- *Lagunekin kafea hartu dut.* — 我和朋友们喝了咖啡。
- *Mikelek ardoa erosi du.* — Mikel 买了酒。
- *Guk pelikula bat ikusi dugu.* — 我们看了一部电影。
- *Haiek euskara ikasi dute gaur.* — 他们今天学了巴斯克语。

## 否定

要表达"我没有做 X"，**ez** 放在助动词前面：

> *Ez dut bazkaldu.* — 我没有吃午餐。
>
> *Ez du egunkaria irakurri.* — 他没有读报纸。
>
> *Ez dugu ardorik edan.* — 我们没有喝酒。（注意 *ardorik*：带否定时，宾语通常带 **-rik** 部分格。）

## 小对话

> *— Zer afaldu duzu gaur, Maialen?*
>
> *— Entsalada eta arraina jan dut, eta postrerako jogurta hartu dut. Eta zuk?*
>
> *— Nik tortilla egin dut, baina ez dut postrerik jan.*

> **图解小结：** 如果动作有明确或隐含的宾语（吃某物、看某物、做某物），助动词就是 **dut**（及其派生形式）。记住这些动词-助动词的搭配，其余的自然就会了。
