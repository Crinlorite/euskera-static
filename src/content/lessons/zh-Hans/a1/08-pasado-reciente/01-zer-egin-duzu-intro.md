在巴斯克语中，要讲述不久前做过的事——尤其是今天做的事——会使用 **orainaldi burutua**（字面意思是"现在完成时"），它相当于西班牙语的 *he comido / he ido*（我吃过了/我去过了）。这就是所谓的**近过去时**。

## 公式非常简单

> **Partizipioa（分词）+ 助动词（izan 或 ukan）**

与上一单元学过的习惯体相比，唯一改变的是动词**不带 -tzen / -ten**：

| 习惯体 | 近过去时 | 中文 |
|---|---|---|
| *gosaltzen dut* | **gosaldu dut** | 吃早餐 → 吃过早餐了 |
| *jaten dut* | **jan dut** | 吃 → 吃过了 |
| *joaten naiz* | **joan naiz** | 去 → 去过了 |
| *etortzen naiz* | **etorri naiz** | 来 → 来过了 |
| *ikusten dut* | **ikusi dut** | 看 → 看过了 |

> **这和中文里**一般习惯与已完成动作之间的区别是一样的：
>
> - *Egunero euskaltegira **joaten naiz**.* — 我每天去 euskaltegi（巴斯克语学校）。
> - *Gaur goizean euskaltegira **joan naiz**.* — 今天早上我去了 euskaltegi。

## 最自然的提问方式

> *— **Zer egin duzu gaur?*** — 你今天做了什么？
>
> *— Lan egin dut, eta gero zinera joan naiz.* — 我工作了，然后去了电影院。

注意这两种形式：**lan egin dut**（用 *dut*，因为在巴斯克语中"工作"带宾语）和 **joan naiz**（用 *naiz*，因为"去"不带宾语）。

## 两个助动词——快速复习

### 用 izan（NOR）—— naiz / zara / da / gara / zarete / dira

| 代词 | 助动词 | 例句 |
|---|---|---|
| ni（我） | naiz | *joan **naiz*** （我去过了） |
| zu（你） | zara | *etorri **zara*** （你来过了） |
| hura（他/她） | da | *esnatu **da*** （他/她醒了） |
| gu（我们） | gara | *joan **gara*** （我们去过了） |
| zuek（你们） | zarete | *etorri **zarete*** （你们来过了） |
| haiek（他们） | dira | *joan **dira*** （他们去过了） |

### 用 ukan（NOR-NORK）—— dut / duzu / du / dugu / duzue / dute

| 代词 | 助动词 | 例句 |
|---|---|---|
| nik（我） | dut | *jan **dut*** （我吃过了） |
| zuk（你） | duzu | *ikusi **duzu*** （你看过了） |
| hark（他/她） | du | *erosi **du*** （他/她买了） |
| guk（我们） | dugu | *egin **dugu*** （我们做了） |
| zuek（你们） | duzue | *jan **duzue*** （你们吃过了） |
| haiek（他们） | dute | *ikusi **dute*** （他们看过了） |

## 小对话

> *— Kaixo Maialen! Zer moduz?*
>
> *— Ondo. Eta zu? **Zer egin duzu gaur?***
>
> *— Lan egin dut goizean, eta arratsaldean lagun batekin kafea hartu dut.*
>
> *— Polita. Ni gaur ez naiz lanera joan, gaixo egon naiz.*

> **本课小结：** 全部诀窍就在于知道每个动词使用哪个助动词（接下来两课会讲到），并且让分词保持不带习惯体标记 *-tzen/-ten*。
