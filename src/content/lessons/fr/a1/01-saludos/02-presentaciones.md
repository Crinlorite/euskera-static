Pour te présenter en euskera, une formule très simple suffit : **Ni** (je) + ton prénom + **naiz** (suis).

## Le modèle

| Euskera | Français |
|---|---|
| *Ni Egoitz naiz.* | Je suis Egoitz. |
| *Ni Maialen naiz.* | Je suis Maialen. |
| *Ni Hodei naiz.* | Je suis Hodei. |

Le verbe **izan** (« être ») change selon la personne. Pour l'instant, mémorise seulement trois formes :

- *Ni* + **naiz** → je suis
- *Zu* + **zara** → tu es
- *Hura* + **da** → il/elle est

> En euskera, le verbe n'a pas de genre grammatical : *Hura da* vaut aussi bien pour « il est » que pour « elle est ». C'est le contexte ou le prénom qui le précise.

## « Comment ça va ? »

Une fois que tu t'es présenté, le plus naturel est de demander comment va l'autre personne :

- *Zer moduz?* → Comment ça va ?
- *Ondo, eskerrik asko.* → Bien, merci.
- *Ni ere ondo.* → Moi aussi, bien.

Avec ces trois phrases, tu peux déjà tenir une salutation polie complète.
