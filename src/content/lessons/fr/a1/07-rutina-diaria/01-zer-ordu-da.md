En euskera, l'heure se construit avec le verbe **izan** (être) et, sauf dans le cas de « une heure », l'heure se met toujours **au pluriel**. Littéralement : *« ce sont les cinqs »*, *« ce sont les huits »*. Cela paraît étrange au début, mais le schéma est très régulier.

## La question

Pour demander l'heure, on utilise **Zer ordu da?** (« quelle heure est-il ? »).

> *— Zer ordu da?*
>
> *— Zazpiak dira.* (Il est sept heures.)

## Les heures pile

| Heure | Euskera |
|---|---|
| 1:00 | **Ordu bata da** |
| 2:00 | **Ordu biak dira** |
| 3:00 | **Hirurak dira** |
| 4:00 | **Laurak dira** |
| 5:00 | **Bostak dira** |
| 6:00 | **Seiak dira** |
| 7:00 | **Zazpiak dira** |
| 8:00 | **Zortziak dira** |
| 9:00 | **Bederatziak dira** |
| 10:00 | **Hamarrak dira** |
| 11:00 | **Hamaikak dira** |
| 12:00 | **Hamabiak dira** |

## Pourquoi une heure et deux heures sont des cas particuliers

- **Ordu bata** (une heure) est **singulier**, on le dit donc avec **da** : *Ordu bata da*.
- **Ordu biak** (deux heures) porte le mot *ordu* devant car *biak* seul pourrait prêter à confusion — c'est la seule heure où beaucoup de gens ajoutent « ordu » pour plus de clarté. Les autres (à partir de trois heures) vont seules : *hirurak*, *laurak*, *bostak*…

## Le suffixe `-ak`

D'où vient ce **-ak** final ? C'est le pluriel basque (tu l'as vu dans *gurasoak*, *aiton-amonak*). Ici aussi : *bost* (cinq) → *bostak* (« les cinqs ») + *dira* (« sont »).

> **Note :** beaucoup de gens comptent l'heure de la journée avec l'horloge de 12 heures, comme en français. Pour distinguer le matin de l'après-midi, il suffit d'ajouter *goizeko* (du matin) ou *arratsaldeko* (de l'après-midi) devant : *goizeko zortziak* (huit heures du matin). Nous le verrons à la leçon 3.

## Entraîne-toi

Regarde l'horloge de ton portable tout de suite et dis-la à voix haute. S'il est onze heures : *hamaikak dira*. S'il est une heure : *ordu bata da*. Fais-le toute la semaine chaque fois que tu regardes l'heure — c'est la meilleure manière de l'automatiser.
