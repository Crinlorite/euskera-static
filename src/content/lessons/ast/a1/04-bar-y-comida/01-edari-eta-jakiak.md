Una bona parte de la vida social n'Euskal Herria pasa pela **taberna** (el bar). Conocer les bébores y comíes básiques n'euskera abriráte puertes nel intre: una cuadriella d'amigos, una visita a un pueblu nuevu, un *poteo*. Entamamos pol vocabulariu.

## El café y les sos variantes

El café ye dablemente lo que más vas pidir. Tien delles formes:

| Euskera | Asturianu |
|---|---|
| *kafea* | el café (xenéricu) |
| *kafe hutsa* | café solu (lliteral: "café puru/vacíu") |
| *kafesnea* | café con lleche (de *kafe* + *esne*) |
| *kafe ebakia* | café cortáu (*ebaki* = "cortáu") |
| *kafe deskafeinatua* | café descafeináu |
| *infusioa* | infusión |
| *tea* | té |

> *Kafesne* lliteralmente significa "café-lleche". L'euskera forma munches pallabres compuestes asina, ensin guión y cola pallabra principal al final.

## Otres bébores

| Euskera | Asturianu |
|---|---|
| *ura* | l'agua |
| *zumoa* | zusmiu |
| *esnea* | la lleche |
| *ardoa* | vinu |
| *ardo gorria* | vinu tintu |
| *ardo zuria* | vinu blancu |
| *garagardoa* | cerveza |
| *zuritoa* | caña (cerveza pequeña, típica) |
| *sagardoa* | sidra |
| *txakolina* | txakoli (vinu blancu vascu) |
| *patxarana* | pacharán |

> Tres pallabres curioses: **sagardo** (sidra) = *sagar* (mazana) + *ardo* (vinu); **garagardo** (cerveza) = *garagar* (cebada) + *ardo*; **txakolina** ye un vinu blancu xoven y llixeru, marca de la casa n'Euskadi.

## Comíes básiques nel bar

Pa acompañar lo que bebes:

| Euskera | Asturianu |
|---|---|
| *ogia* | el pan |
| *gazta* | quesu |
| *urdaiazpikoa* | xamón |
| *tortilla* | tortiella (de pataca) |
| *patata frijituak* | pataques frites |
| *gozokia* | dulce (xenéricu) |
| *txokolatea* | chocolate |

## L'artículu `-a`

Cuasi toes estes pallabres lleven **`-a`** al final cuando apaecen "neutres" nuna frase: ye l'artículu "el / la".

- *Ardoa gustatzen zait.* — Préstame'l vinu.
- *Ura nahi dut.* — Quiero agua.

Cuando pides "ún", desapaez la `-a` y entra **bat** ("ún/una"):

- *Kafe **bat**, mesedez.* — Un café, por favor.
- *Garagardo **bat**.* — Una cerveza.

> Na lleición 3 verás esto en detalle. Pel momentu memoriza les formes con `-a` pa reconocelos na carta del bar.
