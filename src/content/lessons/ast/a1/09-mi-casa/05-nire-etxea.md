Esta lección integra tolo del tema 9: les habitaciones, los muebles, les xeres y con quién vives. L'oxetivu ye qu'apodas **describir la to casa** en frases cenciellas, como si te tuvieren entrugando en clase o nuna conversación informal.

## Les entrugues básiques

| Entruga | Castellanu |
|---|---|
| *Non dago zure etxea?* | ¿Ónde ta la to casa? |
| *Nolakoa da?* | ¿Cómo ye? |
| *Zenbat gela ditu?* | ¿Cuántes habitaciones tien? |
| *Norekin bizi zara?* | ¿Con quién vives? |
| *Zenbatgarren solairuan bizi zara?* | ¿En qué planta vives? |
| *Etxeak badu igogailua / balkoia / terraza?* | ¿La casa tien ascensor / balcón / terraza? |

## Axetivos útiles pa describir la casa

| Euskera | Castellanu |
|---|---|
| *handia* | grande |
| *txikia* | pequeña |
| *argitsua* | lluminosa |
| *iluna* | escura |
| *berria* | nueva |
| *zaharra* | vieya |
| *lasaia* | tranquila |
| *zaratatsua* | ruidosa |
| *erosoa* | cómoda |
| *garestia* | cara |
| *merkea* | barata |

> Recuerda'l patrón **NOLAKOA da?** yá vistu n'otres unidaes: la rempuesta lleva l'axetivu + `-a` final + verbu *izan*. Exemplu: *Nire etxea **handia da**.* — La mio casa ye grande.

## Un párrafu modelu

Equí tienes un testu cenciellu describiendo una casa. Lléelu en voz alta y fíxate nos patrones.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## El to turnu: describi la to casa

Constrúi'l to propiu párrafu respondiendo nesti orde:

1. **Non bizi zara?** (¿Ónde vives?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (¿Cómo ye?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (¿Cuántes habitaciones?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (¿Qué hai?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*

> **Conseyu:** cuando faples de la to casa en clase, nun intentes traducir cada detalle. Quédate con frases curties y bien feches. *"Bi logela ditut, sukalde txiki bat eta egongela handia"* ye meyor descripción qu'una frase llarga enllena de fallos.
