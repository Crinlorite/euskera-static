이 과는 이 단원의 모든 내용을 한데 모읍니다: 시간, 하루의 때, 일과 동사, 그리고 빈도. 목표는 당신의 하루 전체를 처음부터 끝까지 바스크어로 이야기할 수 있게 되는 것입니다.

## 핵심 질문: "Zer ordutan…?"

| 질문 | 한국어 |
|---|---|
| *Zer ordutan esnatzen zara?* | 몇 시에 잠에서 깨요? |
| *Zer ordutan jaikitzen zara?* | 몇 시에 일어나요(자리에서 일어나요)? |
| *Zer ordutan gosaltzen duzu?* | 몇 시에 아침을 먹어요? |
| *Zer ordutan bazkaltzen duzu?* | 몇 시에 점심을 먹어요? |
| *Zer ordutan ateratzen zara etxetik?* | 몇 시에 집에서 나가요? |
| *Zer ordutan afaltzen duzu?* | 몇 시에 저녁을 먹어요? |
| *Zer ordutan oheratzen zara?* | 몇 시에 잠자리에 들어요? |

> 보조동사의 변화에 주의하세요: *esnatzen **zara*** (NOR)이지만, *gosaltzen **duzu*** (NOR-NORK)입니다. 동사 + 보조동사 짝을 차츰 익혀 나가야 합니다.

## "X시에" — 말하는 법

"(몇 시)에"라고 대답하려면, 복수형 시간을 가져와서 *-ak*를 **-etan**으로 바꿉니다(이미 모음으로 끝나는 경우를 제외하고는 `-e-`가 사이에 들어갑니다):

| 시간 | "…시에" |
|---|---|
| zazpiak (7시) | **zazpietan** |
| zortziak | **zortzietan** |
| hamarrak | **hamarretan** |
| hamabiak | **hamabietan** |
| ordu bata (1시) | **ordu batean** (단수, 자기 형태를 유지함) |
| ordu biak | **ordu bietan** |
| bost eta erdiak | **bost eta erdietan** (5시 반에) |
| zortziak laurden gutxi | **zortziak laurden gutxitan** (8시 15분 전에) |

## 완전한 하루 — 모범 글

이 예시 글을 읽고 모든 조각들이 어떻게 이어지는지 살펴보세요:

> *Ni goizeko zazpietan esnatzen naiz. Poliki-poliki altxatzen naiz eta komunera joaten naiz. Dutxa bat hartzen dut, arropa janzten dut eta sukaldera joaten naiz. Sukaldean gosaria prestatzen dut eta lasai-lasai gosaltzen dut. Ondoren, etxetik ateratzen naiz eta lanera joaten naiz. Eguerdiko ordu bietan bazkaltzen dut. Arratsaldean batzuetan zinemara joaten naiz, beste batzuetan lagunekin egoten naiz. Gauean, etxean afaltzen dut, telebista ikusten dut eta hamaiketan oheratzen naiz.*

대략적인 번역:
> *나는 아침 7시에 잠에서 깬다. 천천히 일어나서 화장실에 간다. 샤워를 하고, 옷을 입고, 부엌으로 간다. 부엌에서 아침을 준비하고 느긋하게 아침을 먹는다. 그다음 집에서 나가 직장에 간다. 정오 2시에 점심을 먹는다. 오후에는 어떤 때는 영화관에 가고, 어떤 때는 친구들과 함께 있는다. 밤에는 집에서 저녁을 먹고, 텔레비전을 보고, 11시에 잠자리에 든다.*

## 자신의 하루를 만들어 보세요

당신의 실제 정보로 이 틀을 따라 해 보세요. 배운 것을 굳히는 가장 좋은 방법입니다:

1. **몇 시에 잠에서 깨는지:** *Goizeko ___etan esnatzen naiz.*
2. **일어나서 무엇을 하는지:** *... dutxatzen naiz / gosaltzen dut / ...*
3. **직장이나 학교에 어떻게 가는지:** *... oinez / autoz / autobusez joaten naiz.*
4. **오후에 무엇을 하는지:** *Arratsaldean ...*
5. **몇 시에 저녁을 먹고 잠자리에 드는지:** *Gaueko ___etan afaltzen dut eta ___etan oheratzen naiz.*

## 연결할 때 유용한 표현

- *Lehenengo* — 먼저, 첫째로
- *Gero / Ondoren* — 그다음, 후에
- *Eta gero* — 그리고 나서
- *Azkenik* — 마지막으로
- *Egunero* — 매일
- *Batzuetan / Beste batzuetan* — 가끔 / 다른 때는
- *Beti* — 항상
- *Inoiz ez* — 절대로 (~하지 않는다)

> **문화 노트:** Euskal Herria에서는 많은 사람들이 14:00에서 15:00 사이에 점심(*bazkaldu*)을 먹고, 꽤 늦게 21:00~22:00쯤에 저녁(*afaldu*)을 먹습니다. *Hamaiketakoa* — 오전 11시의 간식으로, 보통 핀초나 샌드위치 — 는 하나의 전통입니다. 친구 무리(cuadrilla)와 함께 있을 때 11시에 누군가가 *hamaiketakoa egitera goaz*라고 말하면, 무슨 차례인지 이제 아실 겁니다.
