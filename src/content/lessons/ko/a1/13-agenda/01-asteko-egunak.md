계획과 일정에 대해 이야기하기 전에, 요일을 복습하고 일상 활동과 연결하는 법을 배워봅시다.

## 일곱 개의 요일

| 바스크어 | 한국어 | 힌트 |
|---|---|---|
| *astelehena* | 월요일 | aste + lehen = "한 주의 첫째 날" |
| *asteartea* | 화요일 | aste + arte = "주중" |
| *asteazkena* | 수요일 | aste + azken = "노동 주간의 마지막 날" (역사적) |
| *osteguna* | 목요일 | egun = 날 |
| *ostirala* | 금요일 | *osteguna* 와 어근 *ortzi/osti-* (하늘)을 공유함 |
| *larunbata* | 토요일 | larun + bata |
| *igandea* | 일요일 | 역사적으로 축일 |

> **기억 힌트:** 단어 **aste** (주)는 평일 요일들의 앞부분에 나타납니다. *Astelehen-, astear-, asteazken-* 은 거의 소리 내어 한 주의 처음, 중간, 끝을 세는 셈입니다.

## "월요일에"라고 말하기 — 접미사 `-(e)an`

어떤 일의 날짜를 정할 때는 요일에 `-(e)an` 을 붙입니다:

| 요일 | "~에 + 요일" |
|---|---|
| astelehena | astelehen**ean** |
| asteartea | astearte**an** |
| asteazkena | asteazken**ean** |
| osteguna | ostegun**ean** |
| ostirala | ostirale**an** |
| larunbata | larunbat**ean** |
| igandea | igande**an** |

- *Astelehenean lan egiten dut.* — 월요일에 일합니다.
- *Larunbatean afaltzera joango gara.* — 토요일에 저녁을 먹으러 갈 것입니다.

## "매 X마다"라고 말하기 — 접미사 `-ero`

규칙적인 빈도를 나타낼 때:

- *astelehenero* — 매주 월요일
- *larunbatero* — 매주 토요일
- *egunero* — 매일
- *astero* — 매주
- *astebururo* — 매 주말마다

## 요일 + 활동: 예문

| 바스크어 | 한국어 |
|---|---|
| *Astelehenean kiroldegira noa.* | 월요일에 체육관에 갑니다. |
| *Asteartean euskara klasea dut.* | 화요일에 바스크어 수업이 있습니다. |
| *Ostiralean parranda egiten dugu.* | 금요일에 놀러 나갑니다. |
| *Larunbatean familiaren etxera joango naiz.* | 토요일에 가족 집에 갈 것입니다. |
| *Igandean lasai gelditzen naiz etxean.* | 일요일에 집에서 느긋하게 지냅니다. |

## Asteburua — 주말

`Asteburua` (주말)는 매우 유용한 단어입니다:

- *Asteburuan zer egingo duzu?* — 주말에 뭐 할 거예요?
- *Asteburu honetan mendira joango naiz.* — 이번 주말에 산에 갈 것입니다.
- *Asteburu pasa egitea gustatzen zait.* — 주말 나들이를 하는 것을 좋아합니다.

> **문화 노트:** *igandea* 옆에는 종종 *jaieguna* (축일)를 보게 됩니다: *igandeak eta jaiegunak* 라는 표현은 버스 시간표, 상점, 시청 안내문에서 전형적으로 쓰입니다.
