단원을 마무리하려면 한 가지가 더 필요합니다: **어떻게 이동하는가**. 한국어에서는 "교통수단 + 으로/로"("자동차로", "버스로")를 씁니다. 에우스케라에서는 이동수단에 붙이는 접미사 **`-z`**(*도구격*)를 쓰며, 놀랍게도 "발"을 뜻하는 단어에도 붙습니다.

## 질문

| 질문 | 한국어 |
|---|---|
| ***Nola* zoaz?** | 어떻게 가니? |
| ***Zerez* zoaz?** | 무슨 (수단)으로 가니? |
| ***Nola* doa Iruñera?** | Pamplona에 어떻게 가니? |

두 형태(*Nola*와 *Zerez*) 모두 유효하고 자주 쓰입니다. *Nola*는 더 일반적이고, *zerez*는 "무슨 수단으로"를 직접적으로 함의합니다.

## 교통수단

| 수단 | -z가 붙은 형태 | 한국어 |
|---|---|---|
| oin (발) | **oinez** | 걸어서 |
| kotxe / auto | **kotxez** / **autoz** | 자동차로 |
| autobus | **autobusez** | 버스로 |
| tren | **trenez** | 기차로 |
| metro | **metroz** | 지하철로 |
| taxi | **taxiz** | 택시로 |
| bizikleta | **bizikletaz** | 자전거로 |
| motor | **motorrez** | 오토바이로 |
| hegazkin | **hegazkinez** | 비행기로 |
| itsasontzi | **itsasontziz** | 배로 |
| furgoneta | **furgonetaz** | 승합차로 |
| zaldi (말) | **zaldiz** | 말을 타고 |

> 접미사는 모음 뒤에서는 **`-z`**, 자음 뒤에서는 **`-ez`**입니다 (*autobus → autobusez*). 실제로 거의 모든 교통수단 이름이 모음으로 끝나므로, 기본 공식은 **수단 + z**입니다.

## 완전한 예문

- **Oinez noa lanera.** — 걸어서 일터에 갑니다.
- **Iruñera trenez goaz.** — 기차로 Pamplona에 갑니다.
- **Bilbora hegazkinez doaz.** — 비행기로 Bilbao에 갑니다.
- **Eskolara bizikletaz dator alaba.** — 딸이 자전거로 학교에 옵니다.
- **Mendira oinez igotzen gara.** — 걸어서 산에 오릅니다.

## NONDIK + NORA + NOLA 결합하기

전형적인 여행 문장은 보통 세 가지 요소를 가집니다: **어디에서 + 어디로 + 어떻게**.

- **Iruñetik Donostiara autobusez doa.** — 버스로 Pamplona에서 Donostia로 갑니다.
- **Bilbotik Madrilera trenez gatoz.** — 기차로 Bilbao에서 Madrid로 옵니다.
- **Etxetik euskaltegira oinez noa.** — 걸어서 집에서 euskaltegi로 갑니다.

## 미니 대화

> — *Aupa! Bihar Bilbora **noa**.* — 안녕! 내일 Bilbao에 가.
>
> — *Eta **nola** zoaz?* — 그래서 어떻게 가?
>
> — *Trenez. Oso erosoa da.* — 기차로. 아주 편해.
>
> — *Ni autoz noa beti, baina garestia da gasolina.* — 난 항상 자동차로 가는데, 휘발유가 비싸.

## "oinez"에 관하여

언뜻 보면 *oinez*는 별개의 경우("발로")처럼 보이지만, 사실 나머지와 정확히 같은 패턴입니다: *oin* (발) + *ez* (도구격 접미사). 에우스케라는 "걸어서 가다"를 다른 모든 이동 방식과 똑같이 다룹니다: 당신을 데려가는 수단을 사용해서 말이죠.

## NORA / NONDIK / NOLA 블록 최종 복습

| 질문 | 접미사 | 질문 유형 | 답변 유형 |
|---|---|---|---|
| **Nora?** | -ra | 어디로? | *Bilbo**ra** noa.* |
| **Nondik?** | -tik | 어디에서? | *Bilbo**tik** nator.* |
| **Nola? / Zerez?** | -z | 어떻게? / 무엇으로? | *Tren**ez** noa.* |

> **팁:** 이 세 접미사를 함께 쓰면 거의 모든 여행 문장을 만들 수 있습니다. 예를 들어: *Bilbotik Donostiara autobusez goaz, lagun bat bisitatzera*. — "친구를 만나러 버스로 Bilbao에서 Donostia로 갑니다." 이 세 접미사와 *joan*, *etorri*의 종합형을 익히면, 일상의 어떤 이동에 대해서도 말할 수 있습니다.
