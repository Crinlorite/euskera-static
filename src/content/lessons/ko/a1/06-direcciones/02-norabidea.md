누군가 어떤 장소에 어떻게 가는지 물을 때 — 또는 길을 잃어 당신이 물어볼 때 — 명령형 동사 몇 개와 방향 부사가 필요합니다. 이 단원에서는 꼭 필요한 것들을 모았습니다.

## 기본 명령형

바스크어에서 이동 동사의 긍정 명령형은 매우 직접적입니다.

| 형태 | 동사 원형 | 뜻 |
|---|---|---|
| **Jo** | jo | 치다 / 향하다 (jo ezkerrera = 왼쪽으로 돌다) |
| **Segi** / **Jarraitu** | segitu / jarraitu | 따라가다, 계속하다 |
| **Hartu** | hartu | 잡다, 타다 (hartu kale hau = 이 길로 가다) |
| **Igo** | igo | 올라가다 |
| **Jaitsi** | jaitsi | 내려가다 |
| **Sartu** | sartu | 들어가다 |
| **Irten** | irten | 나가다 |
| **Zeharkatu** | zeharkatu | 건너다 |
| **Iritsi** / **heldu** | iritsi / heldu | 도착하다 (… arte = "…까지") |

> *Jo*는 문자 그대로는 "치다"를 뜻하지만, 비유적으로 "~쪽으로 향하다"라는 뜻으로 쓰입니다: **jo ezkerrera** = "왼쪽으로 가다".

## 방향 부사와 기준어

| Euskera | 한국어 |
|---|---|
| *ezkerra* | 왼쪽 |
| *eskuina* | 오른쪽 |
| *aurrera* | 앞으로 |
| *atzera* | 뒤로 |
| *zuzen* | 곧장, 똑바로 |
| *gora* | 위로 |
| *behera* | 아래로 |

> 주목하세요: **~을 향한 방향**을 나타낼 때는 접미사 `-ra`를 사용합니다(03단원에서 자세히 다룹니다): *ezkerre-ra* (왼쪽으로), *aurre-ra* (앞으로), *atze-ra* (뒤로). *gora*, *behera*도 마찬가지입니다.

## 표현 모음

- **Jo ezkerrera bidegurutzean.** — 교차로에서 왼쪽으로 도세요.
- **Segi aurrera semaforora arte.** — 신호등까지 직진하세요.
- **Zeharkatu zebrabidea.** — 횡단보도를 건너세요.
- **Igo eskailerak eta sartu eraikinean.** — 계단을 올라가서 건물 안으로 들어가세요.
- **Jaitsi aldapa, eta gero jo eskuinera.** — 비탈길을 내려간 다음, 오른쪽으로 도세요.
- **Egin atzera; gehiegi joan zara.** — 뒤로 돌아가세요; 너무 멀리 가셨어요.

## 길 묻고 안내하기

> — *Barkatu, **non dago** udaletxea?* — 실례합니다, 시청이 어디에 있나요?
>
> — ***Segi aurrera** semaforora arte. Han **jo eskuinera** eta **zeharkatu** plaza. Udaletxea elizaren ondoan dago.* — 신호등까지 직진하세요. 거기서 오른쪽으로 돌아 광장을 건너세요. 시청은 교회 옆에 있습니다.
>
> — *Eta gertu **dago**?* — 그리고 가까운가요?
>
> — *Bai, oso gertu. Bost minutu oinez.* — 네, 아주 가깝습니다. 걸어서 5분이에요.

> **요령:** 많은 길 안내는 고정된 패턴을 따릅니다: 명령형(*jo*, *segi*, *zeharkatu*) + 접미사가 붙은 장소(*ezkerrera*, *semaforora*, *zebrabidea*). 이 패턴을 외워 두면, 여러 조각을 조합하여 긴 안내문도 줄줄이 이어 말할 수 있습니다.
