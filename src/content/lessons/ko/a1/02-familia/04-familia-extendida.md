부모, 형제, 자녀를 넘어서 가족은 삼촌, 사촌, 조카, 그리고 인척으로 확장됩니다. 여기 가장 많이 쓰이는 어휘를 정리했습니다.

## 삼촌과 사촌

| 에우스케라 | 한국어 |
|---|---|
| *osaba* | 삼촌 |
| *izeba* | 이모/고모 |
| *lehengusu* | 사촌(남) |
| *lehengusina* | 사촌(여) |
| *iloba* | 조카 (일부 방언에서는 "손주"를 뜻하기도 함 — 문맥에 따라 결정됨) |

## 인척

| 에우스케라 | 한국어 |
|---|---|
| *aitaginarreba* | 장인/시아버지 |
| *amaginarreba* | 장모/시어머니 |
| *koinatu* | 처남/시동생/매형 |
| *koinata* | 처제/시누이/올케 |
| *errain* | 며느리 |
| *suhi* | 사위 |

## 사용 패턴

- *Nire osaba Bilbon bizi da.* — 내 삼촌은 Bilbao에 산다.
- *Bere lehengusinak hamabi urte ditu.* — 그의 사촌(여)은 열두 살이다.
- *Zure koinata euskalduna da?* — 네 처제는 바스크 사람이니?

> 인척을 나타내는 단어는 흔히 기존 단어를 조합해 만들어집니다 (*aita-* + *ginarreba*). 내부 논리를 외울 필요는 없습니다. 쓰다 보면 자연스럽게 익혀집니다.
