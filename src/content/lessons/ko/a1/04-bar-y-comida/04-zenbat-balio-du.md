에우스케라로 가격을 묻고 계산할 때 듣는 말을 이해하려면 두 가지가 필요합니다. 바로 **Zenbat balio du?**라는 표현과 100까지의 숫자입니다. 에우스케라의 숫자 체계는 **20진법**(20 단위 묶음을 기준으로 함)이라서 익히는 데 조금 시간이 필요합니다.

## 핵심 질문

| 에우스케라 | 한국어 |
|---|---|
| *Zenbat balio du?* | 얼마예요? |
| *Zenbat da?* | 얼마예요? |
| *Zenbat dira denera?* | 전부 얼마예요? |
| *Pagatu daiteke?* | 계산해도 될까요? (계산서 요청) |

> **Zenbat**은 "얼마 / 몇 개"라는 뜻입니다 — 이미 *Zenbat urte dituzu?* ("몇 살이에요?")에서 보았죠. **Balio** = "값이 나가다 / 들다".

## 20까지의 숫자 (복습)

가족 단원에서 1부터 10까지는 이미 보았습니다. 여기서는 11부터 20까지를 봅니다.

| 숫자 | 에우스케라 |
|---|---|
| 11 | *hamaika* |
| 12 | *hamabi* (hamar + bi) |
| 13 | *hamahiru* |
| 14 | *hamalau* |
| 15 | *hamabost* |
| 16 | *hamasei* |
| 17 | *hamazazpi* |
| 18 | *hemezortzi* |
| 19 | *hemeretzi* |
| 20 | *hogei* |

> 12부터 17까지는 *hamar* ("열") + 한 자리 숫자로 이루어진다는 점에 주목하세요. *hamabi* = "열-둘" = 열둘. *Hamaika* (11)와 *hemezortzi*/*hemeretzi* (18, 19)는 불규칙입니다.

## 십의 자리: 20진법 체계

여기서 에우스케라의 놀라운 점이 나옵니다. 십의 자리는 **20 단위 묶음**으로 셉니다.

| 숫자 | 에우스케라 | 논리 |
|---|---|---|
| 20 | *hogei* | 기준 |
| 30 | *hogeita hamar* | "20과 10" |
| 40 | *berrogei* | "두 배 20" (*ber-* = 두 배) |
| 50 | *berrogeita hamar* | "두 배 20과 10" |
| 60 | *hirurogei* | "세 개의 20" (*hiru* = 셋) |
| 70 | *hirurogeita hamar* | "세 개의 20과 10" |
| 80 | *laurogei* | "네 개의 20" |
| 90 | *laurogeita hamar* | "네 개의 20과 10" |
| 100 | *ehun* | 기준 |

> 중간 숫자들도 같은 논리를 따릅니다. 25 = *hogeita bost* ("20과 5"), 67 = *hirurogeita zazpi* ("세 개의 20과 7"). 처음에는 어렵지만, 바에서 가격을 다루며 연습하다 보면 저절로 됩니다.

## 가격 말하기

| 가격 | 에우스케라 |
|---|---|
| 1 € | *euro bat* |
| 2 € | *bi euro* |
| 1,50 € | *euro bat eta berrogeita hamar* / *euro eta erdi* |
| 3 € | *hiru euro* |
| 25 € | *hogeita bost euro* |

> **euro**는 숫자 뒤에 올 때 `-a`가 붙지 않는다는 점에 주목하세요(*bi anaia*, "두 형제"와 마찬가지입니다). 센트 단위까지의 정확한 금액은 한국어처럼 말합니다. *hiru euro eta berrogeita hamar zentimo* (3,50 €).

## 여러 개 사기

하나 이상을 주문할 때는 *dut*/*dugu* 대신 **ditut** 또는 **ditugu**를 쓰는 것을 잊지 마세요.

- *Bi kafe hartuko ditut.* — 커피 두 잔을 마실게요.
- *Hiru ogitarteko hartuko ditugu.* — 보카디요 세 개를 먹을게요.

## 가격을 말할 때 쓰는 형용사

- **garesti** — 비싼 (관사가 붙으면 *garestia*)
- **merke** — 싼 (관사가 붙으면 *merkea*)
- **doan** — 무료

> *Hau oso garestia da!* — 이거 너무 비싸네요!
> *Bilbon kafea merkea da.* — Bilbo에서는 커피가 쌉니다.
