이 과는 9과의 모든 내용을 통합합니다: 방, 가구, 집안일, 그리고 누구와 함께 사는지. 목표는 수업이나 격식 없는 대화에서 질문을 받았을 때처럼 간단한 문장으로 **당신의 집을 묘사**할 수 있게 되는 것입니다.

## 기본 질문

| 질문 | 한국어 |
|---|---|
| *Non dago zure etxea?* | 당신의 집은 어디에 있나요? |
| *Nolakoa da?* | 어떤가요? |
| *Zenbat gela ditu?* | 방이 몇 개 있나요? |
| *Norekin bizi zara?* | 누구와 함께 사나요? |
| *Zenbatgarren solairuan bizi zara?* | 몇 층에 사나요? |
| *Etxeak badu igogailua / balkoia / terraza?* | 집에 엘리베이터 / 발코니 / 테라스가 있나요? |

| *merkea* | 싼 |

> 다른 단원에서 이미 본 **NOLAKOA da?** 패턴을 기억하세요: 대답에는 형용사 + 어미 `-a` + 동사 *izan*이 옵니다. 예: *Nire etxea **handia da**.* — 우리 집은 크다.

## 모범 단락

여기 집을 묘사하는 간단한 글이 있습니다. 소리 내어 읽고 패턴에 주목해 보세요.

> *Ni Iruñean bizi naiz, auzo lasai batean. Nire etxea ez da oso handia: hiru gela ditu — bi logela eta egongela bat — sukaldea eta komuna. Egongela argitsua da, eta balkoi bat dauka. Hirugarren solairuan dago, eta etxeak igogailua du.*
>
> *Bikotekidearekin bizi naiz. Berak bazkaria prestatzen du, eta nik ontziak garbitzen ditut. Asteburuetan biok elkarrekin egiten dugu erosketa.*

## 당신 차례: 당신의 집을 묘사해 보세요

다음 순서로 대답하며 자신만의 단락을 만들어 보세요:

1. **Non bizi zara?** (어디에 사나요?) → *X-(e)an bizi naiz.*
2. **Nolakoa da etxea?** (어떤가요?) → *Nire etxea handia / argitsua / lasaia da.*
3. **Zenbat gela ditu?** (방이 몇 개 있나요?) → *Bi / Hiru / Lau gela ditu.*
4. **Zer dago etxean?** (무엇이 있나요?) → *Sukaldea, egongela, bi logela eta komun bat ditu.*
5. **Norekin bizi zara?** → *Bakarrik / Gurasoekin / Lagunekin bizi naiz.*
6. **Etxeko lanak nork egiten ditu?** → *Nik egunero garbitzen dut* / *Pisukideen artean banatzen ditugu.*
