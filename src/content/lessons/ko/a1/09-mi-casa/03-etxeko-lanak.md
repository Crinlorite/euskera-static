이번 과에서는 **etxeko lanak**(집안일)에 대해 이야기하기 위한 기본 동사들을 배웁니다. 에우스케라에서 동사는 보통 **분사** 형태(`-tu`, `-i`, `-n`)로 제시되며, 이것이 "사전형"입니다: *garbitu*, *prestatu*, *jaso*…

## 핵심 동사

| Euskera | 한국어 |
|---|---|
| *garbitu* | 청소하다 |
| *prestatu* | 준비하다 |
| *jaso* | 치우다 |
| *jantzi* | 입히다 / 씌우다 |
| *atera* | 내다 / 꺼내다 |
| *zabaldu* | 널다 / 열다 |
| *lisatu* | 다림질하다 |
| *paseatu* | 산책시키다 |
| *erosi* | 사다 |

## 흔한 집안일

| 집안일 | 어떻게 말하나 |
|---|---|
| 침대를 정리하다 | *ohea jantzi* |
| 설거지하다 | *ontziak garbitu* / *harrikoa egin* |
| 청소기를 돌리다 | *xurgagailua pasa* |
| 빨래를 널다 | *arropa zabaldu* |
| 옷을 다림질하다 | *arropa lisatu* |
| 화장실을 청소하다 | *komuna garbitu* |
| 쓰레기를 내다 버리다 | *zaborra atera* |
| 식사를 준비하다 | *bazkaria prestatu* |
| 장보다 | *erosketak egin* |
| 개를 산책시키다 | *txakurra paseatu* |

## 사전형에서 습관 현재형으로

어떤 행동이 습관적으로 반복될 때(매일, 매주…), 동사는 `-tzen` + 보조동사 **dut/duzu/du…**(이미 *ukan*에서 배운) 형태로 바뀝니다:

| 인칭 | 예문 | 한국어 |
|---|---|---|
| Nik | *Nik etxea **garbitzen dut**.* | 나는 집을 청소한다. |
| Zuk | *Zuk arropa **lisatzen duzu**.* | 너는 옷을 다림질한다. |
| Berak | *Aitak bazkaria **prestatzen du**.* | 우리 아버지는 식사를 준비하신다. |
| Guk | *Guk erosketak **egiten ditugu**.* | 우리는 장을 본다. |

> 이번 과에서는 동사 활용을 완벽하게 익힐 필요는 없습니다. 일상을 이야기할 때 쓰는 **`-tzen` + dut/du** 패턴만 기억하세요.

## 빈도

어떤 집안일을 얼마나 자주 하는지 말할 때:

- **egunero** — 매일
- **astean behin** — 일주일에 한 번
- **astean bitan / hirutan** — 일주일에 두 번 / 세 번
- **noizean behin** — 가끔
- **inoiz ez** — 전혀 (안 함)

> *Nire etxean nik **astean behin** lisatzen dut, baina **egunero** garbitzen dut sukaldea.* — 우리 집에서 나는 일주일에 한 번 다림질하지만, 매일 부엌을 청소한다.

## 누가 무엇을 하는지 묻기

- **Nork garbitzen ditu ontziak?** — 누가 설거지를 하나요?
- **Nor arduratzen da arropaz?** — 누가 빨래를 담당하나요?
- **Zuri gustatzen zaizu sukaldatzea?** — 요리하는 것을 좋아하나요?

> **문화 노트:** 가정 내 집안일 분담에 관한 설문은 euskaltegi 교재의 단골 소재입니다 — 그럴 만한 이유가 있죠. 친구나 가족과의 실제 대화에 아주 유용한 어휘입니다.
